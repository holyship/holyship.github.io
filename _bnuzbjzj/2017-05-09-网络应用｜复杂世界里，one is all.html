编者按


《ONE·一个》app自2012年横空出世以来，据官方统计，他们已经积累了3000万+用户，日活跃量近百万。可能因为当今社会信息都是来得快去得也快，导致人们对这众多的信息接收产生疲劳；而《ONE·一个》里面只有各种“一”，却吸引了这么多用户。或许这就是《ONE·一个》的宗旨一样——“复杂的世界里，一个就够了”。



/ 它是千万中的“一个” /


可能你喜欢书的质感，喜欢把书捧在手心的重量；可能你享受智能的快感，享受信息在指间快速穿行。书本有千万，阅读app也有万千，你的那一个，又是哪一个呢？


韩寒曾经说过：“身边的碎片越来越多，新闻越来越杂，话题越来越爆，什么又都是来得快去得快，多睡几个小时就感觉和世界脱节了，关机一天就以为被人类抛弃了。”当下的我们，处于信息爆炸的时代，谁也不知道下一秒又有什么新鲜事发生。当然，这给我们的平淡的生活提供的不少聊资，但信息的过快传播，容易导致人们产生信息接收疲劳。


这时的你，或许需要一个简单的、懂你的app——


“于是就有了你所看见的’一个’。每天都只有一张图片，一篇文字，一个问题和他的答案。”



/ “一个”转变:

从文艺阅读到文艺生活 /


最初的《ONE·一个》，每天只有“三个一”，但作为主创的韩寒说：“要延续每天一文、一图、一问答的模式，微信公众号就能满足，根本不需要一个app，而《ONE·一个》本身也需要更复杂的内容呈现形式。”


于是，2016年1月，《ONE·一个》3.0版本正式上线，将文章和问答结合成全新的阅读板块，并添加了音乐和影视模块。


2017年1月，《ONE·一个》4.0正式上线，加入了聚焦社会热点、挖掘事件背后真相的“one实验室”，和与你每晚相约，分享新奇有趣故事的“one深夜电台”。


这一系列的改动，也印证了《ONE·一个》3.0改版的宗旨——“我们原来是一个基于阅读的app，宗旨是陪伴文艺生活。”



/ 它是“一个”应用

却身拥多个特点 /


互联网是一种双向互动的媒体，你的所有问题都能在互联网上得到其他用户的回应，这即为“多点对多点”的传播模式。


《ONE·一个》就是践行这一传播模式的优秀代表——《ONE·一个》页面中的所有内容都来自《ONE·一个》官方论坛。论坛是向全社会开放的，鼓励所有感兴趣的网友投稿并进行讨论。这样，《ONE·一个》上所呈现的内容，都是从大家的投稿和讨论中得到的。它不仅仅是一个电子杂志软件，更是一个沟通平台。


在《ONE·一个》的运营模式下，用户看到的都是经过编辑碎片化加工后所呈现出来的内容，都是满足读者对于短小文学作品的简读，是浅阅读的表现。


但在分析《ONE·一个》中的阅读内容后不难发现，《ONE·一个》中所呈现的阅读内容价值较高，已超出了浅阅读中要求的信息提取功能。这样的阅读方式是符合碎片化阅读的本质要求的。在这样的阅读方式下，读者利用更小的篇幅、更短的时间，完成了在传统阅读中对于知识学习的本质诉求，可以说《ONE·一个》式的阅读方式是当前生活条件下极为合理的短而精的阅读方式。



/ 复杂世界里

“一个”就够了 /


《ONE·一个》创立初期，就凭借韩寒在微博上的一句话，便获得了广泛的关注，使得《ONE·一个》上了appstore总榜第一。简而言之，《ONE·一个》实则就是明星效应的产品——韩寒这个名字，就是一个响亮的品牌。然而，仅凭韩寒的名声，《ONE·一个》是不可能走到现在的。“一个”的作者群，“内容为王”的定位等，这都是《ONE·一个》可以牢牢抓住粉丝的心的原因。


《ONE·一个》的内容遵循简约的风格，不像那些融合大量信息的阅读类APP，《ONE·一个》让人接收的信息量很小。就像韩寒说的，“它们是原创的、独家的、专门为你及这个世界而存在的。”


《ONE·一个》能够给你的，只是每天一位作者写给你的一篇好文章，或者一组好图片。也许关于爱情，也许关于楼下卖西瓜的大婶，也许关于世界末日。每天你只需花很短的时间就能读完它，或许只要5分钟，或者长一些，但绝对不会超过一小时。读完后，你或许能为之一下，或许会看看窗外，静静地思考三五分钟，当然也可可能只是发个呆。


我们总是看得太多，却又懂得太少。我们自以为知道了一切，但其实什么都没有留在心里。《ONE·一个》不会让你知道那么多，但是每天都会留一点东西在你心里。这就是，“one is all”。
