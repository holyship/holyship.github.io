导读


微博，自推出仅仅两年时间里，用户数量就已经突破了两亿。但从2012年起，受微信的推出、各门户网站和Instagram、Facebook、Twitter等社交软件的冲击和其他各要素的影响，微博的网民使用率严重下滑。直至2015年，微博的网民使用率终于止跌回升。那么，微博为什么又回来了呢？



什么是微博？

微博（Weibo），即微型博客（MicroBlog）的简称，即是博客的一种，是一种通过关注机制分享简短实时信息的广播式的社交网络平台，也是一个基于用户关系信息分享、传播以及获取的平台。用户可以通过WEB、WAP等各种客户端组建个人社区，以140字（包括标点符号）为限制的文字更新信息，并实现即时分享。微博的关注机制分为可单向、可双向两种。微博作为一种分享和交流平台，其更注重时效性和随意性。



一步一个脚印，慢慢成长


 

2009年8月28日新浪微博正式上线。


 

2010年初，新浪微博推出API开放平台，新浪微博成为国内最有影响力、最受瞩目的微博运营商。


 

2012年4月，新浪微博宣布推出广告平台，全面启动商业化。


 

2013年8月1日，新浪微博、阿里巴巴推出“微博淘宝版”，启动双方平台账户的打通。


 

2014年3月17日晚9点半，新浪微博正式登陆纳斯达克，新浪CEO、新浪微博董事长曹国伟等敲响纳斯达克开市钟。


 

2015年1月20日，微博开放微博140字 的发布限制，少于2000字即可。1月28日对微博会员开放试用权限，2月28日将正式对微博全量用户开放。


 

2016年11月，新浪微博取消了发布器的 140 字限制。此次开放限制后，包括普通用户在内的所有用户均可以发布多达 2000 字的长微博。




微博的功能


根据中国互联网信息中心的统计，截止2011年12月底，中国网民数量达到5.13亿，而当时微博用户就多达两亿，由此看来，每三个网民里，至少有一个会使用微博。微博得到如此辉煌的成绩，在很大程度上得益于微博自身优越的功能。

 


基本功能

1．开放API：新浪微博 API，覆盖了新浪微博的全部功能，可以通过 API 发微博，传照片，加关注，甚至搜索等全部功能。

2．客户端 ：新浪微博正式上线的时候，它会同时推出各种手机、浏览器、IM上的微博客客户端、机器人以及短信平台。

3．图片、视频分享：新浪微博具有图片、视频和音乐分享功能。

4．同步到其它微博客。

5．搜索功能。


主要功能

1. 发布功能：用户可以像博客、聊天工具一样发布内容。

2. 转发功能：用户可以把自己喜欢的内容一键转发到自己的微博，转发时还可以加上自己的评论。

3. 关注功能：用户可以对自己喜欢的用户进行关注，成为这个用户的关注者（即“粉丝”）。

4. 评论功能：用户可以对任何一条微博进行评论。

5. 搜索功能：用户可以用两个#号之间，插入某一话题。像这像，#某一话题XXX#，则发出的微博，可以点击这个，自动搜索微博上所有的包含有“某一话题XXX”的相关微博。可以展开讨论，实现信息的聚合。

6. 私信功能：用户可以点击私信，给新浪微博上任意的一个开放了私信端口的用户发送私信，这条私信将只被对方看到，实现私密的交流。

7. 群聊功能：用户可以创建自己的微群，或选择自己感兴趣的微群，并且为未加入微群的用户随机推荐热门微群。微群能够聚合有相同爱好或者相同标签的朋友们，将所有与之相应的话题全部聚拢在微群里面，让志趣相投的朋友们以微博的形式更加方便的进行参与和交流。



微博的第二春来啦！

根据《第35次中国互联网发展状况统计报告》的数据显示，截至2014年12月，我国微博客用户规模为2.49亿，较2013年底减少3194万，网民使用率为38.4%，与去年年底相比下降了7.1个百分点。其中，手机微博客用户数为1.71亿，相比2013年底下降2562万，使用率为30.7%。根据《第36次中国互联网发展状况统计报告》的数据显示，截至2015年6月，我国微博客用户规模为2.04亿，网民使用率为30.6%，手机端微博客用户数为1.62亿，使用率为27.3%。手机端微博客用户占总体的79.4%，比2014年底上升了10.7%。自2012年微博受到多方面的冲击，网民使用率持续下滑。直至2015年，微博的网民使用率终于止跌回升，微博又火了，微博又回来了。用户为什么要再一次回到微博，经分析，发现有5大原因。



微信朋友圈的弊端与愈发商业化


                       微商发布广告的消息


最初，微信的最大魅力在于“私密”。相比于微博仅需单向关注就能看到对方动态，微信好友是必须在双方意愿下才能建立的紧密关系。在微博和QQ空间中，你可以看见楼主的全体朋友的点赞和评论，但在微信朋友圈，只有自己的通信好友才能看见，用户们可以在朋友圈里相互分享生活点滴。但随着用户数量的急剧膨胀，微信里的好友列表就不仅仅是亲戚朋友，朋友圈挤进了大量不想聊的人，甚至他们是见过一面的陌生人，有些微信用户的工作关系的密集程度甚至远超亲朋关系，用户已经无法感受到当初的那种“私密”。慢慢地，微商、广告、公众号链接等内容充斥着原本用于分享生活点滴的朋友圈。在微信里用户好友数量越来越多的情况，每发一条朋友圈，就意味着越多的人会看到或关注，这导致越来越多的人不敢轻易地在朋友圈里发表自己的感受和想法、分享自己的生活点滴。另外，微信朋友圈里，有些用户会经常发一些广告或链接，微商不断推销产品，中国互联网络信息中心发布的调查数据显示，在各种移动营销推广方式中，微信营销推广使用率最高，2016年高达75.5%，远超企业移动官网、移动搜索营销推广和移动社交推广等方式，由此看来，微信朋友圈愈发商业化。如果是经常出现自己不感兴趣的内容，于是就有了屏蔽朋友圈、朋友圈权限、沉默、换微信号、不经常上微信的情况出现，微信朋友圈也渐渐地失去了最初的意义。

相反的，微博是基于兴趣订阅模式。微博的内容种类多样，有娱乐的，时尚的，美食的，搞笑的等等，你可以查找你感兴趣的博主加关注。如果有一天，你对博主发表的内容不满意或是没兴趣，你可以随时取消关注。



微博功能的优越性


1. 门槛低：每条微博可以发布超过140个字符，可以三言两语，现场记录、也可以发发感慨，晒晒心情。

2. 随时随地：用户可以通过互联网、客户端、手机短信彩信、WAP等多种手段，随时随地地发布信息和接受信息。

3. 快速传播：用户发布一条信息，他的所有粉丝能同步看到，还可以一键转发给自己的粉丝，实现裂变传播。

4. 实时搜索：用户可以通过搜索找到其他微博用户在几秒前发布的信息，比传统搜索引擎的搜索结果更有时效性，更鲜活。



微博粉丝经济的推动


                        微博网页版热门搜索页面


打开微博，微博的热门搜索，很多都与娱乐明星或世界各地实时或最近发生的大事件相关，文字加配图或文字加视频，总能轻易地让微博用户获取信息或了解博主想要表达的内容。在微博推行初期，许多的微博的兴起在很大程度上得益于“大V”政策。一大批精英人士频繁在微博上对公共领域话题发言，随着微博用户的不断增多，他们的粉丝多达数千万，单条微博的转发量和评论量动辄几十万，影响力不亚于一家大型媒体，在数次公共事件和公共话题争论中形成巨大的舆论压力。同样的，如今微博不仅大"V"的粉丝数量多，中“V”小“V”也兴起壮大，特别是网红，他们通过拍摄搞笑的视频、拍摄化妆教程、录制美食制作节目、写段子等方式吸引用户的注意，从而增加粉丝数量和提高知名度，像papi酱、微博搞笑排行榜（榜姐）等博主，他们的粉丝数量有几百万甚至有的上千万，粉丝数量直逼红透半边天的明星微博的粉丝数量，每条微博的转发、评论和点赞的数量惊人，影响能力不言而喻。

说起娱乐明星和网红，粉丝们想要了解他们的动向和消息，微博就是一个获取这些信息的最佳渠道。先说，微信的好友容量数量上限是5000，如果一个娱乐明星想要与粉丝保持互动和联系，那么5000的好友容量是远远不够的。而微博就凸显了这一优势，无论是加关注人的数量还是粉丝数量，都是无上限的。另外，微博还有一个功能就是悄悄关注，如果你有想要关注的对象，但又不想让对方知道，那么用户可以通过偷偷关注而去关注和了解对方的动态。但微信的好友是必须在双方意愿下才能建立的关系，因而达不到“悄悄”这种效果。虽然在微博发展的过程中，受到了Instagram的冲击，用户数量有所下降，但后来大陆对Instagram的使用有所限制，导致使用的人也越来越少，用户最后还是纷纷回归到微博的怀抱！



微博的直播与小视频功能的引入


                              微博热门视频页面


微博投资的科技创业公司“一下科技”，这家公司旗下的短视频应用“秒拍”、“小咖秀”和直播应用“一直播”等爆款产品，成为微博笼络年轻用户群体的利器。娱乐明星和网红拍摄一些各类视频，吸引粉丝去点击观看，“一直播”直播平台更是让明星网红与粉丝能够很好地互动。微博财报显示，2016年12月，微博短视频日均播放量同比增长713%。2017年的春节，微博与央视春晚进行合作。除夕期间，微博全站视频播放量超过33亿，其中与春晚相关的视频播放量约为5.6亿次。微博跨年晚会直播与5家电视台进行了合作，覆盖用户数超过了2000万，互动量超过了4000万。由此带动不少的微博直播增量。摩根士丹利在去年8月发布的研究报告称，视频和直播产品正在推动微博用户增长，预计微博月活跃用户数将在2018年达到4亿。



智能手机的广泛使用和微博平民化


随着智能手机价格降低，智能手机的使用越来越广泛。智能手机刚开始推出的时候，多数都是对新事物好奇，爱尝试新事物的年轻一代的人使用，老一辈的人很多都依旧习惯使用非智能手机。但如今，智能手机使用越来越广泛，不仅在二三四线城市普及，使用的年龄段甚至普及到了老年人，传统CCTV和各省各类的电视节目，已不能满足人们爱知求真的内容需求，电视媒体在国内多个事发不在第一现场，传播速度和跟踪出现断档。但微博的热门搜索，有些媒体或用户在第一现场，能够第一时间发布世界各地所发生的实时事件，从而满足人们爱知求真的需求。另外，在二三四线城市，地方在管理上的能力存在不足，民事和官民各种纠纷较多，导致传统媒体电视、报纸不敢受理。所以需要一个出口来爆料，来吸引更多普通老百姓和政府的关注，而微博就是一个很好的途径和渠道。微博的用户数量多，同时微博文字配图和引进秒拍等有图有真像的工具，一个新闻热点曝光在人们的视线，很容易引起人们的关注，也引起了相关部门的关注，使得问题得到了被解决的可能。正因如此，二三四线城市，各个年龄段的人关注微博，使用微博，推动着微博再次回到人们的日常生活，成为重要的社交软件！

 





总的来说，微博从蓬勃发展到受冲击衰落到再次回到大众的视野，新浪团队一直以精以求精的态度来做这一款社交软件。虽然微博依然存在着不足，但它在不断改进与创新，在各方面不断提高自身的竞争能力和与用户的粘合性，并把最新的成果展现给用户。微博为我们提供了一个很好的社会化传播平台，使我们每一个人都相当是一个自媒体，都有着展现自己的能力与才华的舞台和拥有自己的影响力。微博引领着互联网的风潮，现在已经形成了一种重要的社会现象，甚至形成了一种微博文化，在方方面面影响着人们的生活。如今微博又火了，微博又回来了，那么“今天，你微博了吗？”
