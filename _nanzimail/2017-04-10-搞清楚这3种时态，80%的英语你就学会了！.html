通过昨天的自学，我们知道了只要含有动词就是一个句子，那么这个句子是什么时候发生的事情呢？就通过给动词“换衣服”来实现，以呈现出不同的动词形态。这些衣服叫做时态。


时态分为“时”和“态”。


“时”指动作发生的时段，有一般、现在、过去、将来4种。


“态”指动作完成的进度，有进行、完成、完成进行、一般4种。


不同的“时”和“态”就组成了16种“时态”：


一般现在时，一般过去时，一般将来时，一般过去将来时

现在进行时，过去进行时，将来进行时，过去将来进行时

现在完成时，过去完成时，将来完成时，过去将来完成时

现在完成进行时，过去完成进行时，将来完成进行时，过去将来完成进行时


语言是有相通性的，中文逻辑学得好，英文时态就能学得好。在这16个时态中，有3种最容易搞错，其他13种，只要语文及格了，就能很轻松地学会。可以说，只要把这3种时态搞清楚，学英语只需要有一本字典，就能不断遣词造句了。就像你已经拥有了好看的身材曲线，接下来就可以穿各种各样的漂亮衣服来改变你的造型了。


这3个最难的时态是：



现在完成时

格式：have done


用法1：过去发生的一个动作，已经结束了，但是对现在有影响


e.g.

 I have done my homework.我已经做完作业了。

 e.g.exempli gratia =for example ，举例


 
用法2：过去发生了一个动作，一直持续到现在，并且有可能继续持续下去。


I have smoked for 3 years.

 我已经戒烟3年了。


现在完成进行时

格式：have been doing


用法：现在完成进行时带有抱怨的成分。


e.g.

I have waited for you for 3 hours.

现在完成时：我已经等了你3个小时了。（很平常的口吻）


I have been waiting for you for 3 hour.

现在完成进行时：人家已经等了你3个小时了哎！（带有抱怨的口吻。）


给那些无法表达瞬间的动词，一个可以变成状态的方法。


do就是一个瞬间，“做”只是一个瞬间，“做完了”才是一个状态。

以do为首的无法表达一个瞬间词要用现在完成进行时。


e.g.

I have been doing my home work.

我一直在做作业。（不是做完了，也不是正在做，而是从过去到现在，我一直在做。）


过去完成时

格式：had done

意思：过去的过去


 
用法：只有在两个动作发生的时候用，这两个动作一个发生在前，一个发生在后。发生在前的用had done，发生在后的用过去时/现在完成时。


e.g.

The TV series I have watched today is the one you had told me yesterday.

我今天已经看完的电视节目是昨天你告诉我的那一个系列。


过去时和过去完成时的区别：

格式上，有没有had

意思上，过去时是过去发生的动作，对现在没有影响，而过去完成时对现在有影响。


e.g.

I had done my homework before I went to out.

我出去玩之前做完了作业。

（两个动作，对现在有影响，是过去完成时）


The work which I done is my favorite part.

这个我做完的工作是我最喜欢的部分。

（一个动作，对现在没有影响，是过去时）
