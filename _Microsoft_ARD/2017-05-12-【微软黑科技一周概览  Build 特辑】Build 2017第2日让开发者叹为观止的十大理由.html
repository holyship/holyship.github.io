丹棱君有话说：每个人的内心深处都蕴藏着创意的灵感。无论是一个宏大的理想、一个遥远的梦想，还是一个最简单的想法，只要将其付诸实践，就有可能实实在在地改变我们的生活。Build 2017 开发者大会第二天，微软公布了 Windows 10 助力人们施展创意的最新举措——Windows 10 Fall Creators Update。Windows 10 Fall Creators Update 将在今年下半年为超过 5 亿台 Windows 10 设备带来创新体验，点燃每一位用户的创意灵感。


微软全球执行副总裁、Windows 与设备事业部负责人特里·梅尔森（Terry Myerson）表示：“Windows 10 将帮助每一个人施展自己的创意。Windows 10 Fall Creators Update 更新将为用户带来的一系列令人兴奋的全新功能，这些功能将推进用户的创新体验、带来统一设计的跨设备体验，并帮助开发者更轻松地构建计算的未来。”

10


超过五亿台 Windows 10 设备迎来新体验：Windows 10 Fall Creators Update 将于今年年内正式推出，为全球五亿台 Windows 10 设备带去新体验。Windows 10 Fall Creators Update 最新版本已向 Windows 预览体验计划的成员开放。 



9


非凡的设计语言：微软流畅设计体系（Microsoft Fluent Design System），它将带来直观和谐、响应迅速、兼容广泛的跨设备体验与互动操作。对于开发者来说，流畅设计能帮助他们设计出更具表现力和吸引力的应用程序，并且广泛支持各种类型的设备和多样化的输入方式。



8


时间回溯：Microsoft Graph 能够构建起在 Windows、iOS、Android 设备之间无缝衔接的一致体验。Microsoft Graph 的新功能时间轴能让用户快捷地跳转到任何时间点去访问文件、应用和网站，仿佛从未离开一样。



7


小娜，之前我做到哪一步了？借助小娜，用户可以随意更换 Windows、iOS 或 Android 设备，并在更换设备后继续之前的操作。剪贴板将成为在互联设备之间进行复制粘贴的便捷手段。OneDrive 文件随用随取，可以方便地访问储存在云端的所有文件，而无需下载和占用设备存储空间。



6


Windows Story Remix：Windows Story Remix 由 .NET 构建，并作为通用 Windows 应用（UWP）在 Windows 应用商店发布，它可以利用人工智能和深度学习技术整理照片和视频并将其转变为叙事形态。它利用 Microsoft Graph 跨设备连接你和你的朋友。Story Remix 能够自动将你的记忆甚至是你好友的照片和视频整合成一个故事，并加入音轨、主题和串场切换效果。通过在照片和视频中加入 3D 对象，用户还可以构建出混合现实的视角，以新的方式讲述故事，或是将照片和视频当作画布，利用 Windows 墨迹书写在上面涂鸦绘画。



5


iTunes 等应用加入 Windows 应用商店：今年晚些时候，包括 iTunes 在内的更多人气应用将加入 Windows 应用商店。届时，Windows 用户将能够在任意 Windows 10 或 Windows 10 S 个人电脑上，享受到完整的 iTunes 应用体验，其中包括 Apple Music、iTunes 商店以及对于 iPhone 和其他苹果设备的支持服务。Autodesk SketchBook 是绘画绘图应用领域当之无愧的王者，这个专门为通用 Windows 平台构建的应用程序将充分发挥 Windows 10 手写笔和触控的最大潜力。还有 SAP 最受欢迎的解决方案之一—— SAP Digital Boardroom 能为企业高管提供单一源头的业务情况汇总，帮助高层实时做出业务决策。



4


开发者工具正当时：大会为 Windows 平台开发者推出多个新工具。下半年将推出的 .NET Standard 2.0 for UWP 和 XAML Standard 将帮助网页开发者、C++、.NET 和 UWP 开发者更轻松地在 Windows 平台上工作。



3


入乡随俗：为了让开发者更好地利用现代编码发挥出 Windows 和 Microsoft Graph 的优势，微软推出了面向 UWP、Android 和 iOS 的“罗马计划”。



2


更多选择：有了 Xamarin Live Player，开发者能在 Windows 电脑上针对 iPhone 应用进行构建、测试和排错。有了它，开发者只需要 Visual Studio 和 iPhone，就能上手开发 iOS 原生应用。



1


微软推出全球首个完全无需依赖位置标签的 Windows Mixed Reality 运动控制器。将 Windows Mixed Reality 头戴设备与运动控制器配对使用，将在创意和生产力工具、游戏、娱乐场景中，为用户带来更丰富和更具沉浸感的体验。利用 Windows Mixed Reality 头戴设备中内置的传感器，就能对视野内的动作进行定位准确、响应迅速的动作跟踪，并且完全不需要在墙壁上安装任何硬件设备。



太阳剧团（Cirque du Soleil）宣布他们将利用 Windows Mixed Reality 技术来转变他们想象和创作演出的方式。为了展现出他们需要面对的创意性决策，太阳剧团为演出 Kurios 重塑了一个虚拟的舞台，其中包括了那只著名的巨大机械手。
