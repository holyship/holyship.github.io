1


7 月 31 日，微软中国宣布发售搭载英特尔®酷睿™ i5 处理器和 256GB 固态硬盘的多种颜色版本的 Surface Laptop，包括深酒红、灰钴蓝和石墨金，实现了超轻设计、性能和优雅外观的完美平衡。Surface Laptop 超越传统笔记本电脑，将铝制外壳与内部 Alcantara® 材质进行了完美融合，打造出独具特色的美感；13.5 英寸 PixelSense™ 触控显示屏色彩鲜明，触控体验更加自然轻松，显示屏四周纤薄的边框也为用户提供更宽广的创作空间。



丹棱君 OS：真乃笔记本电脑界的 Mr. Right，爱，爱，爱不完~


2


8 月 1 日，在全球最有影响力的计算机图形和互动技术年度大会 SIGGRAPH 2017 期间，微软发布了 Azure Batch Rendering 公共预览版，用户可以有效对作业进行调度、排序，管理故障、重试及自动缩放。利用 Azure Batch Rendering，艺术家、工程师和设计师能够通过诸如 Autodesk Maya 、3ds Max 或 SDK 等客户端应用程序无缝提交作品及作业，加快设计成果的交付。点击“阅读原文”，即可了解更多 Azure Batch Rendering 详细操作。



丹棱君 OS：咻咻咻……有了 Azure 的强力支持，批量处理就是快。


3

8 月 1 日，微软宣布，源于 2014 年的骇客马拉松项目——为帮助罹患渐冻症的橄榄球运动员所开发的眼球追踪功能，经过三年的研究孵化，将转化为产品，加入 Windows 10 更新。届时，用户可以通过凝视启动应用，以“看键盘”输入单词的方式，替代传统的物理键盘和鼠标。用户只需看向单词的首尾字母，随后看中间的字母，就能依靠单词预测快速完成输入，文字也将转换为语音。在第三方硬件的支持下，这一功能将帮助患有神经肌肉疾病（如渐冻症）的用户使用 Windows 10 系统，完成上网和文字输入等操作，为“僵硬”的生活注入活力。



丹棱君 OS：让我们一起感受我软科技的温度~~


4

 

8 月 2 日，微软宣布为 Android 和 iOS 版 Skype 应用加入名为 “Send Money” 的 PayPal 转账功能，为全球用户带来方便快捷的移动支付体验。美国、英国、加拿大、匈牙利、比利时等 22 个国家的用户，只需下载最新版本 Skype 手机应用，即可在 Skype 聊天过程中为家人和好友实时转账，无需从 Skype 页面退出后，手动打开浏览器或支付软件进行支付。
