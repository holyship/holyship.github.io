丹棱君近日正在为 One Week Hackathon 2017（骇客马拉松）而“憋大招”呢，嘿嘿。不知道什么是骇客松？木关系～，戳本周【微软黑科技一周概览】，一览骇客必须知晓的本周大事记。And 文末有骇客福利~


1


7 月 13 日，微软宣布了与宝马的最新合作进展，继语音助手 Cortana 、云服务 Azure、Office 365 和微软 Exchange 安装在部分宝马车型之后，Skype for Business 将加入宝马 iDrive 智能行车系统，该系统可直接进行拨号会议，也可通过预先设置在微软 Exchange 日历中的会议日程触发会议。用户可以提前在日历中设定会议时间，自动触发，无需人工操作。



丹棱君 OS：吼吼，这是要把车打造成四轮可移动办公室的节奏啊，在车里开电话会，约吗？


2


7 月 13 日，微软的人工智能机器人“小冰”全面入驻封面新闻客户端 3.2 版本，“小冰”将成为封面新闻的专栏作者，定期撰写新闻稿件。微软“小冰”入驻封面新闻前，已多次在媒体行业试水，努力探索新媒体领域的技术能力，通过实时对话和用户交流，实现新闻传播的效果。截至 2017 年 4 月，微软“小冰”已拥有超过 1 亿用户，累计对话量超过 300 亿。



丹棱君 OS：期待，来自记者“小冰”的报道！


3


7 月 18 日，微软和百度宣布，双方将携手推进全球自动驾驶技术的发展与应用。作为 Apollo（百度向汽车行业及自动驾驶领域的合作伙伴提供的软件平台）开放平台的一员，微软将在全球除中国以外的地区为 Apollo 提供微软智能云 Azure 服务。双方还计划在车联网汽车解决方案领域探索更多合作机会，以期为客户提供独一无二的驾驶体验，推动自动驾驶行业的数字化转型。



丹棱君 OS：无人驾驶 + 智能云 Azure，让我们畅想一下，云中驰骋的感觉~自由“智”在，不要太 cool 哦！


4


7 月 18 日，微软同时向 Windows 、Linux 、macOS 以及 Docker 容器推出了SQL Server 2017 RC1 的公共访问，包括 Linux 活动目录下的 SQL Server 集成；用于加密数据的安全传输层（TLS）支持；机器学习服务增强；SQL Server 分析服务（SSAS），Linux 平台上的 SQL Server 集成服务（SSIS）；Windows Server 上的 SSIS 。目前，微软一共为 SQL Server 2017 推出了 7 个社区预览版，引入了图数据处理支持、适应性查询、面向高级分析的 Python 集成等功能更新。同时，SQL Server 2017 在 TPC - H，TPC - E 等多项基准测试成绩，再创佳绩。


丹棱君 OS：SQL Server 在 Linux 的应用场景又大大丰富啦～快来追随我软的步伐，升级你的 IT 基础架构吧。


5


7 月 19 日，微软宣布与企业资信调查信用管理公司邓白氏（Dun & Bradstreet）合作，利用微软云服务进行数字化转型，为客户提供更加高效的服务，让客户能够无缝访问邓白氏的商业数据。利用微软 Dynamics 365，邓白氏能够将业界标准邓氏编码（D-U-N-S® Number）与核心业务数据进行整合，帮助合作客户与邓白氏的全球业务数据库（由 2.65 亿条记录组成）保持同步。


丹棱君 OS：智能云加持商业智能，我看行！


6


7 月 20 日，微软正式发布新一代 Arc 鼠标，包括 Surface Arc 鼠标与微软 Arc 鼠标。新一代 Arc 鼠标采用创新的全滚动平面，浑然一体的触控区域为手指提供了更加自由的移动空间，还可同时支持水平和垂直方向的滚动，实现快速、高效的移动和导航操作；此外，新一代 Arc 鼠标应用蓝影追踪技术，Surface Arc 鼠标与微软 Arc 鼠标几乎可以在各种平面流畅使用。搭配 Surface 系列设备与 Windows 10 设备，Arc 鼠标方便携带到任何地点，并可调节至扁平状态，轻松放入口袋或包中。



丹棱君 OS：第一眼看过去，整个人都“懵”了。是的，Surface 出品，必属精品。荷包君，快来和我一起减肥～


7


7 月 21 日，微软发布 2017 财年第四季度财报：截止到 2017 年 6 月 30 日，营收达到 233 亿美元，运营收入为 53 亿美元，净收益达 65 亿美元。其中，生产力和业务流程业务收入达 84 亿美元，增长 21% ；智能云服务收入达 74 亿美元，增长 11% 。微软首席执行官萨提亚·纳德拉（Satya Nadella）表示，“客户希望通过微软，以及我们充满活力的合作伙伴生态系统来加速他们自己的数字化转型，在智能云及智能边缘计算时代创造新的机遇。”此外，微软还公布了截止到 2017 年 6 月 30 日的 2017 财年业绩：营收达到 900 亿美元，运营收入为 223 亿美元，净收益达 212 亿美元。
