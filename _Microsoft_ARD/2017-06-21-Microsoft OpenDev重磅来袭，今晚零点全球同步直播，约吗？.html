丹棱君有话说：在这里，你能学习各种微软平台开源新知识，获取微软合作伙伴带来的诸多精彩内容，还能通过实际案例，深入浅出地了解如何将开源技术布署在微软云上。没错，丹棱君说的就是 Microsoft OpenDev！这场由微软总部发起、微软的开源合作伙伴共同主持的云端开源技术在线研讨会，将于今晚 0 点准时上线！这么多干货，这么多大咖，还不快报名，和我们一起玩转开源技术！


本次 Microsoft OpenDev 直播将由微软云首席架构师 John Gossman 带领各位进入微软的开源新世界。


内容包含 Redhat on Azure、Canonical on Azure、Docker & 混和云架构实战，Pivotal Cloud Foundry 应用在现代化的企业环境中融合Java 的共构，Chef on Azure for DevOps、Deis Kubernetes 容器管理平台与 Helm、Steward、Kops套件的解析，以及 Skype 如何透过在 Azure 全球布署 Linux 基础设施服务全世界的用户等丰富内容。



此外，我们也邀请到国内著名的合作伙伴 Redhat 专家洪涛、Redhat 专家王磊、灵雀云架构师 Daniel Morinigo、微软开源云市场经理何谚锜、微软技术顾问李铮、微软云计算企业事业部首席研发经理缪瑾，以及微软云技术架构师王添，将为您详细介绍快速运用 Java、Java Script 建立您的第一个容器微服务解决方案、运用您已经熟悉的各项开源技术结合 DevOps 做快速产品迭代、让您的既有服务能够更有效的管理与开发维护等内容。


我们在北京为大家同步转播这场开源技术盛宴，让您可以一边享受国际大咖的技术直播，一边吸收来自国内技术专家的第一手消息，一起玩转开源技术，在微软开源云 Azure 上快速拓展与布建您们的解决方案！
