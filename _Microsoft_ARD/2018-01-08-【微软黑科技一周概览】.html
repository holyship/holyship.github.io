微软发布最新 Azure TSI 开发工具


1 月 3 日，微软正式发布了 Azure Time Series Insights（TSI）的最新开发包，包括可自动管理环境，访问策略、事件源和参考数据的 Azure Resource Manager （ARM） 模板，可编写 TSI API 应用程序的 C# 和 JavaScript 样本以及易于理解的使用说明文档。开发者可将 TSI 的平台嵌入到自定义应用程序中，生成图表，比较不同时间点的数据，动态研究趋势和数据相关性。点击“阅读原文”，了解详情。

微软正式宣布收购混合云数据存储公司 Avere Systems


1 月 3 日，微软正式宣布收购混合云数据存储公司 Avere Systems。Avere 能够为 Linux 和 Windows 用户提供高性能的 NFS 和 SMB 文件存储服务，该服务可以在云、本地和混合环境中兼容运行。此次收购将有机整合 Avere 的存储能力，进一步为行业用户提供承载更大型、复杂的高性能工作负载的微软 Azure 云服务。



丹棱君 OS：携手 AVERE，微软云更强大、更靠谱~



微软宣布与西雅图生物技术公司 

Adaptive Biotechnologies 达成合作


1 月 4 日，微软宣布与西雅图生物技术公司 Adaptive Biotechnologies 达成合作，将结合最新的人工智能和机器学习领域的技术突破，全面分析血液测试大数据，对人类免疫系统进行测绘和解码，检测各种疾病（包括传染病、癌症和自身免疫性疾病）的发展阶段，有效诊断和及时预防多种疾病。该项目是微软 Healthcare NExT 计划的一部分，微软 Healthcare NExT 计划旨在利用微软的人工智能技术和覆盖全球的云服务，为医疗领域带来创新。
