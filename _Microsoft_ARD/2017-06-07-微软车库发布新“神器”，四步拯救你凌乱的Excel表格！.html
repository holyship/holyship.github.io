丹棱君有话说：面对着一张张被凌乱数据占领的 Excel 表格，除了垂头丧气地一条条整理，你还能做什么？五位微软研究员凭借一番努力，不仅把自己从这种困境中解救出来，还顺手解放了成千上万需要整理数据的分析师们。点击“阅读原文”下载插件，你也能轻松找到你所需的转换函数，从此工作效率再上一个新台阶！


原始数据格式多样，分析师们往往要花费数小时对其进行清理或统一，难免让人头疼。

 

最近，五位微软研究员通过微软车库发布了名为 Transform Data by Example 的 Excel 新插件，在 Azure 平台上开发了后端服务，帮助快速抓取、索引、搜索特定的转换函数和 Web 服务。分析师只需在数据列表中的前几行输入想要的结果示例，就可以完成列表中所有数据的转换。



让我们来看一个例子。


在下面的表格中，左列是一组输入凌乱的美国地址，有的只有城市名，有的含城市名和邮编，有的只有邮编，有的则是州名和邮编。


为了统一提取出每个地址中的城市、州和邮编信息进行后续分析，用户只需先在右列输入了前三个地址的输出示例，然后点击 “Get Transformation!”， 几秒后，Transform Data By Example 就查找到了相关的转换函数。只需点击该函数，用户就能立即获得一列统一格式的地址信息。



目前， Transform Data by Example 插件还处于试水阶段，仅支持日期/时间、地址、电话号码和 URL 地址的批量转换，适用于 Excel 2016、Excel Online、Excel 2016 for Mac 和 Excel 2016 for iPad。

 

Excel 函数爱好者还可以通过插件的扩展接口添加新的转换函数。例如，为了提取下表中每条信息中的浏览器版本信息，用户可在网上搜索相应的 Web 服务，点击插件上的 Add My Service、粘贴 Web 服务的 URL 地址，就可以开始抓取数据了。
