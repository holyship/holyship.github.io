丹棱君有话说：工业 4.0 看哪里？当然是德国。看德国最新工业 4.0 新成果，当然要去 2017 年汉诺威工业博览会。来，丹棱君带你去看看，工业 4.0 的先锋们是如何优雅地实现数字化转型的~

 

你知道吗？到今年底，将有三分之二的来自全球 2000 强企业的 CEO 会把数字化转型作为企业战略的核心。


你知道吗？55% 的企业表示会在一年内开始数字化转型计划，33% 的企业正在制定数字化转型计划。


你知道吗？到 2022 年，智能工厂将成为现实——40% 的操作流程将实现自我修复和自学习。


大家都在谈数字化转型

但为什么要进行数字化转型？


经济学人智库（EIU）的调查显示，39% 的企业认为，在数字化转型上的投入在很大程度上能帮助他们实现实时交易；32% 认为能帮助提高运营效率；28% 认为能帮助赢得新客户。


普华永道的调查显示，45% 的高管预计数字化将提升业务收入，25% 预计数字化将创造更好的客户体验。


IDC 预计，在所有制造业公司中，善于利用数据的公司将有潜力创造约 4000 亿美元的收入。


这年头，圣诞老人都在准备数字化转型了……你还在犹豫吗？



如何优雅地拥抱数字化转型？


在这个“工业 4.0 时代”，许多企业已抓住先机，从最前沿的科技中获益，并和其他尚未发展数字化转型战略的企业拉开了差距。如何引领数字化转型，在市场中获得并保持竞争力？在世界规模最大的工业技术盛会 2017 年汉诺威工业博览会上，一群前辈为我们做出了精彩示范。一起来看看它们如何玩转数字化。



让微软智能云节省您的成本


艺康（Ecolab）旗下的水处理公司纳尔科（Nalco Water）利用微软云平台和高级数据分析工具，帮助客户（包括微软）100% 重复利用和循环使用水资源，并保护系统和设备。纳尔科每年能为微软在圣安东尼奥的数据中心节省超过 14 万美元的水资源成本，九年来总计节省 120 多万美元；每年还节约了 5830 万加仑（约 2.2 亿升）饮用水资源。



全球领先的食品加工和包装解决方案提供商利乐（Tetra Pak），通过将其食品和饮料包装与微软 Azure 云相连接，进一步升级其状态监测系统（Conditioning Monitoring）。2016 年初，利乐启动了一项为期 6 个月的试用，利用升级后的系统支持 11 条客户生产线。试用期间，每条包装流水线的宕机时间最高可减少 48 小时，为客户节省的成本最高达 3 万欧元。



戴姆勒卡车北美公司与微软合作为所有的 Detroit Connect 服务建立了基于云计算的后台管理系统，利用微软智能云处理全球海量的车联网数据。与常见的基于服务器的基本设施相比，云端解决方案效用持久，灵活性更高，经济效应也更好。


让 HoloLens 提高您的效率


为确保高度定制的移动解决方案测量精准，20 多年来，蒂森克虏伯会派出多名工作人员到现场进行测量，并使用一套非常复杂的测量系统。使用 HoloLens 后，销售人员在第一次到达现场时就能测量楼梯数据。数据会通过微软 Azure 云自动与生产部门共享，方便生产部门快速决策，实时批准，无需再次输入手动数据。这种销售和制造过程的数字化帮助蒂森克虏伯将交货时间缩短了四倍。



医疗行业，全球电子制造服务领导者捷普（Jabil）及其咨询机构 Radius 创新与开发部，利用微软 Surface Studio 上的设计软件及微软 HoloLens 的可视化功能，在医疗产品的制造过程中提高协作与创新。其中的全息图及 3D 功能允许厂商在产品进入工厂前，进行产品管理、设计改进，并加速团队合作进程。


柯马（Comau）正与微软和爱康诺（ICONICS）合作，利用微软 HoloLens 及爱诺康全息机器接口（Holographic Machine Interface），远程控制柯马 Racer3 机器人在多种制造环境下组装小部件，为制造厂商提供创新数字化解决方案，帮助他们优化生产流程，提高生产效率。此外，柯马还利用 HoloLens 远程监控车间机器，制造商只需通过自然手势和免提操作，就能够在车间进行数据交互。


让机器人保障您的安全


Sarcos 机器人公司（Sarcos Robotics）与微软合作，通过机器人即服务（robot-as-a-service）为客户保障施工安全，提高施工效率。Sarcos Guardian S 能在千里之外遥控操作。它能够钻过狭窄的下水道、管道、储存槽，还能穿过其它险要地形。Sarcos Guardian S 装有一整套传感器，利用微软 Azure 机器学习及 Windows 10 物联网技术，它还能够解析图案并持续探测异常现象。



让 AI 和机器学习提升您的洞察


德舒特啤酒厂（Deschutes Brewery）的酿酒师们正在用微软的机器学习技术以及基于微软 Azure 的OSIsoft PI Integrator，通过数据科学改进啤酒生产工艺。啤酒发酵通常需要 12 天，德舒特啤酒厂发现如果将发酵期减少 2 天，那么他们的 50 台容器每年就可以多出至少 6 个发酵期。维持现有设备不变，生产能力就可以增长 20%，有望增加 600 万美元的年收益。


山特维克可乐满公司（Sandvik Coromant）是世界领先的切削刀具供应商，目前也在借助微软科技开发解决方案，监控机器加工过程和性能，以减少浪费并提高生产效率。测试过程曾是最熟练的技术工程师的专场，如今已被数字化复制。这也让山特维克可乐满公司多出了一项全新的服务，提供给德马吉森精机公司（DMG MORI）等机器制造商。通过新的监控解决方案，德马吉森可以预测出何时刀片需要维修和保养。


为什么选择微软？


答案是微软独有的四大法宝——安全性承诺、全球云计算规模、合作伙伴生态系统、微软对持续创新和企业标准的投入。


为了促进制造业的发展，微软将继续发展对 OPC UA 和工业自动化机器对机器通信标准的支持。在去年汉诺威博览会关于 OPC UA 公告的基础上，微软上周又宣布，通过微软最新的 Azure IoT Connected Factory 解决方案，制造商使用 OPC UA 在云端预配置工厂设备的过程将变得更加轻松。此外，微软还宣布推出 Microsoft IoT Central，这项全新的软件即服务（SaaS）将大幅提高制造商进行创新和推行新产品的速度，降低创建物联网解决方案的壁垒，为客户创收并改善体验。


除了提供微软自己的解决方案之外，微软 Azure 云还兼容其他主流的物联网平台，如 ABB Ability、罗克韦尔自动化以及西门子 MindSphere，制造商可以自主选择。更重要的是，微软在安全性方面投入了数十亿美元，用于保障制造商的知识产权，防范网络攻击。



2017 年汉诺威工业博览会明天就要闭幕了，但企业的数字化道路不会就此止步。听了这么多大道理，看了这么多成功案例，你的选择是？
