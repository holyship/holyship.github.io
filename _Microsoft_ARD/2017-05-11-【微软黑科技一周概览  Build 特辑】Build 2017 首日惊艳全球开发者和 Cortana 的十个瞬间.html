丹棱君有话说：从昨晚到今天，你是不是已经被微软 Build 2017 大会刷屏啦？机智的你想必已经留意到，国内直播趴上的实时中文字幕用的就是 Microsoft Translator Live！ 


今年 Build 大会第一天，微软 CEO 萨提亚·纳德拉、微软全球执行副总裁兼云计算与企业事业部负责人 Scott Guthrie 以及微软全球执行副总裁兼人工智能及微软研究事业部负责人沈向洋轮番登台，干货满满。


从云计算到边缘计算，微软正在打造更智能的未来。萨提亚 ∙ 纳德拉表示：“我们正处在一个拥有近乎于无限计算能力、并且数据呈指数级增长的世界。在这样的环境下，微软专注于予力每一位开发者，帮助他们开发应用，以适应新的智能云及智能边缘计算时代。”


微软推出一系列全新工具和服务，帮助各平台开发者面向不同平台打造更加智能的应用程序，并对现有应用进行现代化升级。Azure  IoT Edge、Visual Studio 2017 for Mac、Graph  API 接口……三小时的干货，如何瞬间消化？ 贴心如我，已经将精华涵盖在本周【微软黑科技一周概览】特辑。请收好，不谢。


10


Cortana 技能工具包在美国开启公共预览。通过创建一个机器人并将其发布到微软机器人框架新开通的 Cortana 渠道中，开发者可以为 Cortana 增添新技能。这项服务支持 Windows 10、Android、iOS，以及内置了 Cortana 的哈曼卡顿 Invoke 扬声器。



9


Azure IoT Edge：将云计算的智能和更多优势扩展到物联网边缘设备。



8


发布 Azure Cosmos DB：这是业界首个全球分布式的多模型数据库服务，能提供带有运行时间保障、高吞吐、高一致性的水平扩展，并能保证 99% 的个位数毫秒级延迟。



7


在 Azure SQL 数据库基础上新增 MySQL 和 PostgreSQL 托管服务，为开发者提供了更多的选择和灵活性，同时还能确保平台服务的高可用性，尽可能不停机以完成扩展，并可确保数据安全和数据恢复的能力。



6


Visual Studio 2017 for Mac 正式商用，让开发者在 Windows 和 Mac 环境之间实现无缝衔接，全面支持移动、网页和云端工作负载，并提供 Docker 工具、Azure Functions 和 Xamarin.IoT 支持功能的预览。



5


微软英国研究员 ZHANG Haiyan 帮助罹患帕金森病的平面设计师 Emma Lawton 重新拿起笔绘画、写字。微软研究院花了数月时间研究该疾病，开发了一款名为“Emma’s Watch”的可穿戴设备原型机，帮助暂缓 Emma 右手手腕的不自觉颤抖。现在，Project Emma 正在 ZHANG 的协调下，探索利用传感器和人工智能，对与该疾病相关的复杂症状进行检测和监测，包括肌强直、姿势步态障碍、跌倒和静止性震颤等等。



4


开发者将可以在 Microsoft Teams 中发布嵌入式应用。作为 Office 365 提供的一个工作用聊天工具，在 Microsoft Teams 中加入应用，将以全新的应用体验打动用户。开发者还将可以为 Teams 应用添加新的功能，例如在信息流中加入第三方推送通知、复合扩展和可交互信息等。



3


推出全新的 Microsoft Graph API 接口，包括 SharePoint 和 Planner 的 API。Microsoft Graph 让开发者能够访问 Office 365 数据和智能，并在联系人、对话、项目、日程、流程与内容之间建立联系。这些洞察能帮助开发者打造更智能的应用，构建更聪明的工作方式。



2


微软认知服务提供了业界最多的智能 API 接口，也是唯一允许对 API 进行定制的，现在这个数字增加到了 29 个。新加入的 API 服务包括必应定制搜索、定制视觉服务、定制决策服务和视频索引。



还有一个被称为 Presentation Translator 的 PowerPoint 插件，利用微软 Translation API 接口，可以在播放演示文稿的过程中，实时将其翻译成多种语言。访问 http://www.surveygizmo.com/s3/3530821/Translator-Add-in-Sign-Up 进行注册，了解 Presentation Translator 在 Windows 平台的最新进展。
