丹棱君有话说：一年一度的 Build 大会又要明年再见了，你是不是跟丹棱君一样恋恋不舍？今年的 Build 大会上，依旧有惊喜，有震撼，有温情，有感动。你眼中的 Build 大会是什么样？丹棱君翻了翻国内外网友和媒体的评论，有的一本正经讲技术，有的一言不合讲冷笑话，他们有没有说出你的心声！如果没有，请占领留言区，谈谈你对今年 Build 大会的看法。下周一（也就是 5 月 15 日）中午 12 点，丹棱君会给点赞数排名前三的小伙伴送出精美礼品哦！


从云计算到边缘计算，打造更智能的未来


新智元 闻非：边缘计算并不会取代云计算，而是作为后者的补充，在这方面，无论是计算力、数据存储量还是数据分析工具，这届 Build 大会都展示了微软的深厚积累，也成就了如今他们强推的“边缘计算”故事。


爱活网 nic：微软也为所有的 Azure 管理员发布了 Azure 移动应用，它将会在 Windows 10 Mobile、iOS 以及 Android 上架，这意味着管理操作时终于不用时刻守在 PC 面前，在任何地方都可以使用手机监控运行状况。


国外网友 Khaled Chebat：看完 Build 2017 大会第一天的主题演讲，我最大的感受是：如果你还没看到云计算创造的新的就业机会，那你肯定是看错地方了！



为每位开发者、每个机构、每个平台、每个人服务的人工智能


新智元 闻非：如果用一句话总结微软在该公司年度开发者大会 Microsoft Build 2017 上想要强调的微软 AI 发力方向，可以说是“最好的 AI 是让用户感受不到的 AI”。


猫叔在硅谷：要向微软致敬。每年最重要的 Build 开发者大会，微软都会在最黄金的时间，放下自己的产品展示，用几分钟的视频介绍一个微软用最新科技改变残疾人士生活的小故事。去年是通过视觉识别与语音介绍的智能眼镜让残疾人重新“看到”世界，今年则是用智能手表让帕金森患者颤抖的双手也能流畅地写字绘图。


最新的前沿科技非但没有抛弃残疾群体，反而用黑科技改变了他们的生活，一定程度上让他们重新享受到生活本来的乐趣。“科技以人为本”的口号，微软确实在一点点实现。

Mashable 首席记者兼特约编辑 Lance Ulanoff：（不好意思），人工智能将会颠覆每一个商业应用哦！（害怕了吗？）



Windows 10 Fall Creators Update：点燃创意灵感


爱活网 nic：微软也在演讲中公布了一批即将登陆 Windows Store 的用户名单，其中苹果 iTunes 赫然在列，这意味着只能使用 Windows Store 应用的 Windows 10 S 设备将会解决很大的障碍。其次，通过 Windows Store，微软也将允许 Windows 10 设备安装 Ubunt、SUSE Linux 以及 Fedora 子系统，这意味着相关开发者在一个 Windows 10 下就能搞定大多数的开发环境。


品玩 光谱：这次 Build 上，微软很清楚地释放出了对其他平台和公司的善意。


The Verge 高级编辑 Tom Warren：微软的下一个移动战略就是让 iOS 和安卓系统变得更好。
