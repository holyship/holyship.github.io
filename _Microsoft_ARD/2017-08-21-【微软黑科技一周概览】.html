微软凭借强有力的“智能销售生产力”和前瞻性的产品体验，被独立研究机构 Forrester Research 评为 2017 年第二季度销售服务自动化（Sales Force Automation Solutions，以下简称 “SFA”）解决方案领导者。微软 Dynamics 365 SFA 解决方案能基于商业角色为客户互动提供不同解决方案，将客户熟悉的工具贯穿销售流程的所有情境；同时，其内置的智能性能为客户提供前瞻性指引，为企业提供智能洞察、个性化互动以及智能客户管理体验，帮助客户提升销售表现，真正通过现代可扩展平台达成业务增长、演变、和转型。

8 月 14 日，微软发布了 .NET Core 2.0 版本。.NET Core 2.0 新增了 Linux 二进制存档、运行包存储、分离式便携和独立可执行文件，并移除了 OSX 上 OpenSSL 的需求。此外，所有平台都包括可移植的二进制归档文件，支持 Linux Arm 构建，以及 Windows Arm32 和 Arm64 构建。.NET Core 2.0 将提供更好地进程管理、JIT 编译器以及服务器系统的体验。目前，.NET Core 2.0 已部署到 Azure Web 应用中。


丹棱君 OS： 又一波我软给开发人员的大礼包，速领~


3


8 月 15 日，微软发布了 Visio custom visual for Power BI 公共预览版，用户可以在其中创建互连工作流程和实际布局等说明性图标，实现操作智能。将 Visio和 Power BI 结合使用，用户可以在一个区域，对以图表和可视化效果呈现的数据进行说明和比较，从而掌控全局，实现操作智能和商业智能。另外，用户还可以连接到 SharePoint 或 OneDrive 上的 Visio 图表，无需手动操作，Power BI 底层数据就能将自动智能链接图表。作为交互式的 Power BI 可视化工具，Visio 图表将帮助用户更快进行决策。

8 月 15 日，微软宣布收购云计算公司 Cycle Computing，以便让企业更好地使用 Azure 的高性能计算、大数据工作负载及其他“大计算”能力。微软 Azure 大型计算基础设施和 Cycle Computing 技术的有机结合，将加快云端运行速度，增强对 Linux HPC 工作负载的支持，帮助客户充分利用云的最高性能和合规性的基础架构，同时更容易地将预置型工作负荷扩展到云端。

丹棱君 OS：如虎添翼，飞上“云”端。

8 月 17 日，微软公布了研究团队进行的一项“无限滑翔机”飞行试验。通过人工智能，滑翔机能够像鸟类一样，自主发现和捕捉自然产生的热气流，在无须马达的情况下停留在空中。滑翔机的 AI 系统分为两部分：高级规划器和低级规划器。高级规划器负责将所有的环境因素纳入考虑，并尝试制定策略，让滑翔机知道应该去哪里寻找热气流。并随着时间推移，根据每次升空所收集的信息，做出更好的预测。低级规划器部分负责“在实干中学习”，使用贝叶斯强化学习方法，根据滑翔机传感器发回的数据实时侦测和锁定热气流。未来，无限滑翔机可以用于执行各种实用任务，例如监测农村地区的农作物生长，或者在条件艰苦地区提供移动互联网服务等。
