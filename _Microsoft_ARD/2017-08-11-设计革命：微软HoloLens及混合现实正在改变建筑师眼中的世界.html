丹棱君有话说：你知道从 2D 到 3D 有多远吗？对于建筑师来说，从创造草图、到水彩美化、生成电脑 3D 效果图、再做成缩微的泡沫模型，平面的图纸设计，变成实际的立体建筑模型，要经历漫长的过程。而 HoloLens 就能让二维的平面“立”起来。从 2D 到 3D 的距离，触手可及。想知道 HoloLens 是如何颠覆设计体验的呢？好奇 HoloLens 是怎么把设计屏幕全息投射到现实环境中，让建筑师在自己设计的建筑场景中身临其境吗？能想象设计师任意缩放、移动和旋转物理设计的炫酷程度吗？接下来请大家就位，和丹棱君一起来一场穿越 2D 到 3D 的旅行。


甘斯勒（Gensler），世界最大的设计与建筑公司，以其出色的建筑设计而闻名。总高 632 米、光彩夺目的上海中心大厦，还有仅用了 17 天就建成的位于迪拜的世界首个 3D 打印办公室，都是出自甘斯勒之手。


所以，当它位于洛杉矶的办公园区要进行改造时，毫无意外，甘斯勒又迈出了创新的一步。为了打造一个崭新、灵活的健身、娱乐、办公空间，甘斯勒采用了颠覆传统的方式——利用混合现实（Mixed Reality）的沉浸式数字化体验进行建筑设计。 


小到定制楼梯和实验室，大到规划体育场和工厂，混合现实技术已成为众多项目的全新设计方法。微软 HoloLens 全息电脑中的 SketchUp Viewer 应用，是 Windows Store 中的第一款可拓展的商业化 HoloLens 应用，由工程建设技术公司 Trimble 开发，可以快捷创建并支持协作的混合现实设计解决方案。


HoloLens 中的 SketchUp Viewer 应用

为用户带来 3D 设计体验


“混合现实给我们的设计带来了革新。”甘斯勒的创意媒体负责人 Alan Robles 表示，“把设计从屏幕转变为现实环境中的体验，让设计师可以在大空间中与设计对话，非常酷！”


利用 HoloLens 上的 SketchUp Viewer 应用，甘斯勒在洛杉矶的办公园区用全息影像“建造”了一座玻璃天桥，连接两栋大楼。


“利用混合现实，我们可以更好地评估设计效果，比如桥板的高度，是否能达到美学要求，是否会出现任何问题。” Robles 说。


以往，建筑设计方案都要经历从草图，美化，到电脑 3D 效果图，再到泡沫模型，整个过程非常耗时，而且微缩模型很难充分展现实际效果。


SketchUp Viewer把Trimble最热门的3D建模软件嵌入到微软的 HoloLens 中，将全息影像投射在桌子上，用户可以绕着桌子从各个角度进行审视，调整显示比例、移动和旋转；还可以利用建筑信息模型（Building Information Modeling）数据，将 3D 模型以 1 : 1 实际大小呈现，“实际体验”建筑的物理场景和功能特点。


HoloLens 中的 SketchUp Viewer 应用

为用户带来可视化设计与实时协作的新体验


“利用混合现实来呈现和沟通复杂的想法，的确是一个了不起的突破。” Trimble 的混合现实项目总监 Aviad Almagor 说，“它将颠覆传统的消费、互动以及沟通的方式。”


这项技术能实现远程、实时的协作，更快速有效地对方案进行调整，让使用者在设计、施工和维护方面更有把握地进行决策。


甘斯勒在设计可容纳 22000 人的加州银行体育场时，同样在 HoloLens 上使用了 SketchUp Viewer 。


“建筑师要做到的是，用设计提升观众的观赛体验。” Robles 说，“在混合现实技术的帮助下，客户能有效参与设计，进而帮助设计师更好地理解空间的用途。”


其他领先的设计公司，比如 AECOM、CallisonRTKL 和 Hensel Phelps， 也在利用 HoloLens SketchUp Viewer 进行设计创新。此外，德国蒂森克虏伯公司（thyssenkrupp）利用微软 HoloLens 为行动不便人士提供了移动家居解决方案的设计服务。


借助 HoloLens 和 Azure，蒂森克虏伯重新设计了工作流程，开始数字化转型。现在，设计师进行楼梯扫描、步数测量、数据保存和实时分享，都十分方便，并且客户可以亲眼看到虚拟出的 3D 电梯的效果，然后做出决策。销售、设计、制作和安装的效率大大提升。


蒂森克虏伯利用 HoloLens 设计移动家居解决方案


蒂森克虏伯创新副总裁 Thomas Felis 表示：“跟微软合作，蒂森克虏伯可以高效地改造和升级家居，让生活更便捷美好。这是一场变革。”


混合现实正是这样一场颠覆性的变革，不仅改变设计流程，还改变设计本身。Robles 设想，在未来，设计桌的尺寸不再受限于电脑显示器，因为那时混合现实应用的可穿戴设备将无处不在。


Almagor 想的更远，未来可穿戴设备将像隐形眼镜一样轻便，自由地呈现混合现实。消费者可以通过混合现实成为“体验设计师”，轻松更换房间的风格和窗口的风景。


“你可以根据自己的心情来决定当天的房间风格。可以是装饰艺术风，可以把梵高的画挂在办公室的墙上，也可以让窗外的风景变成海洋。”


对于支持创新设计的建筑师 Greg Lynn 来说，HoloLens 上的 SketchUp Viewer 带来的是一种全新的工作方式。在 2016 年威尼斯建筑双年展上，他使用这一技术重新“构建”了位于底特律的废弃汽车厂 Packard 。



Lynn 站在巨大的工厂全息图中，看着天花板，计算模型的最佳比例，并对结果进行快速预览——所有的这些操作都是在他位于洛杉矶的工作室里完成。


“我的工作就是尝试把几何图形变成现实。” Lynn 说，“HoloLens 成为了 2D 、3D 和现实世界的桥梁。这是建筑学，也是革命。”
