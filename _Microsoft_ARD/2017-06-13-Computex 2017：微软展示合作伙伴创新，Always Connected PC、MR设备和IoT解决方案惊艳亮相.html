Peter Han，微软合作伙伴设备与解决方案副总裁


丹棱君有话说：昨天的 2017 微软中国开发者大会看得尽兴吗？今天丹棱君带大家回顾一下本月初落幕的年度盛会 Computex。作为全亚洲规模最大的国际电脑展，Computex 汇聚了全球顶尖的科技企业最前沿的科技产品。微软和合作伙伴们在今年的 Computex 上推出了哪些惊艳的产品和服务？除了 PC 市场，我们在混合现实和物联网领域还有哪些亮眼的表现？微软合作伙伴设备与解决方案副总裁 Peter Han 特别撰文，与大家分享微软与合作伙伴们携手创造的重重惊喜。


亚洲最大规模国际电脑展 Computex 在春夏之交如期而至。这一备受瞩目的科技盛宴不仅对微软而言激动人心，它也吸引了众多 OEM、ODM 生态企业及四万多参观者和 1600 多家参展商齐聚一堂，共同把脉前沿技术，展望全新合作未来。微软的合作伙伴不仅在 PC 市场创新不断，在物联网、混合现实等新兴领域也身手不凡。


今年的 Computex，我与微软个人及设备全球销售副总裁 Nick Parker 、微软 Windows 市场全球副总裁 Matt Barlow 一起，非常荣幸地和与会者分享了微软如何与合作伙伴共同创造无限可能。


其中包括：

  • 首次亮相的多个来自我们合作伙伴的创新产品

  • OEM 和运营商承诺支持 Always Connected PC 服务

  • 将于今年下半年发布的 Windows Mixed Reality 开发套件

  • 针对制造业和公共事业等关键垂直行业的 Windows 和 Azure IoT 解决方案


Windows 混合现实头戴式设备：“百”花齐放


今年晚些时候，我们将会与大家分享来自包括宏碁、华硕、戴尔、惠普、联想等合作伙伴全新设计的混合现实头戴式设备，届时人人都将有机会体验到让人耳目一新的混合现实！这些设备借助微软平台，将为开发者提供统一的用户界面、标准化的输入和通用的应用平台。


在前不久的 Build 大会上，微软 HoloLens 项目负责人 Alex Kipman 宣布宏碁和惠普推出的 Windows Mixed Reality 开发套件已在美国和加拿大开启预售。开发者们反响积极，这让我们对开发生态的未来充满期待。


华硕：未来派


除了 Oculus-Ready ROG 游戏桌面和 VR-Ready VivoPC X 之外，华硕还将推出一款未来派的 HMD（头盔显示器），拥有独特设计的多边形 3D 表面，舒适的人体工程学设计，可调节头带，可单手轻松启动。华硕与微软合作，通过六自由度运动跟踪控制器，使 HMD 响应快速、功能强大、机身轻薄，给佩戴者带来身临其境的混合现实体验。



戴尔：实惠舒适


戴尔与微软合作，产品定位于价格实惠的消费级混合现实头戴式设备，面向大众需求。时尚的白色头戴式设备，通过可自由安装或移除的衬垫、重量平衡头带等创新设计让用户使用时感觉更舒适、操作更方便。参与这款设备设计的正是 Alienware 个人电脑设计的团队。



联想：价格打折，技术不打折


微软的另一合作伙伴联想，通过极其实惠的价格将炫酷的混合现实技术带给每一个人，让人人可享受沉浸式的、超越屏幕的混合现实体验。联想的混合现实头戴式设备将内置传感器，以实现由内而外的追踪并简化调试过程，让用户在虚拟世界里也可以自由走动。



关键垂直行业的物联网解决方案：方兴未艾


企业、政府、学术机构利用物联网驱动业务发展。在 Windows 10 IoT core、Azure IoT、Cortana analytics 和 business intelligence 的支持下，微软提供稳定的端对端平台，帮助合作伙伴为客户搭建企业级的 IoT 解决方案。


物联网并非只是将设备连接到云，而是大量设备智能互联以提供深入的洞察。我的同事微软全球设备销售与合作伙伴关系副总裁 Mike Quinn 通过“物联网时代的一天”向大家展示了设备互联合作给人类生活带来的极大便利。例如，Airbox 设备可以采集空气质量数据，实时显示在汽车仪表盘上。餐厅里有数字招牌、数字菜单、自动贩卖机等，还有一个智能机器人，可以通过历史记录等信息为顾客推荐菜品。


Always Connected PC：电脑再无断网之忧


学生、创作者、自由职业者、小公司、大企业……新时代的用户需要电脑随时随地可以上网。这意味着，屏幕间共享信息以及支持千兆 LTE 和 eSIM 将是 PC 的发展趋势。基于共同的愿景，微软正与合作伙伴们通力合作。微软设备合作伙伴华硕、惠普、华为、联想、VAIO 和小米等将是 Always Connected PC 的终端厂商。此外，华硕、惠普和联想生产的 Always Connected 设备将搭载高通骁龙 835 处理器。该 PC 的特色还包括 Windows 10 操作系统、永远在线的 LTE 连接以及超长续航时间等。


我们的合作伙伴生态：生机勃勃


新设备和新领域为我们的合作伙伴创造了新的商机。从动态存储捆绑和云等新型业务模型到生态合作伙伴进军混合现实头盔显示器和运动控制器等新兴领域，微软平台正不断为合作伙伴创造新的机会。


微软合作伙伴们将搭载 Windows 10 的设备带入市场，推出价格亲民的教育设备，打造激动人心的游戏体验……在合作伙伴们勃勃的创新力推动下，我们看到，新的产品类别正在不断涌现蓬勃发展。


我对于未来充满自信，对微软合作伙伴将要创造的无限可能充满期待！
