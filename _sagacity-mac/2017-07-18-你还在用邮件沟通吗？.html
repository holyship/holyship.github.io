随着微信微博和钉钉 Tower 等各种协作工具的兴起，邮件似乎越来越不被人们提及了，这并不是邮件用的少了，而是邮件成为了基础设施，就像水、电，就像云端的各种服务。好的服务是无感知的，就像空气，只有空气状况变糟的时候我们才会感受到空气的存在。邮件也是一样，互联网发展到现在，邮件成为了现代人最常用的沟通和协作工具，谁没有几个邮箱地址呢？

当一个技术发展成基础设施的时候，创新的机会是不是变少了？恰恰相反，如果这个服务是有生命力的，有用户基础的，创新的机会依然存在，无论是技术层面还是应用层面。就邮箱服务本身来说，更安全更快的协议，跟精准的垃圾邮件过滤规则，更智能的邮箱分类，更有效和有用的 EDM，更方便的使用方式，更好的协作方式等，这些功能依然值得期待。我是一个邮件重度使用者，常常与一些喜爱使用邮件的朋友通过 Gmail 进行会话式的沟通，效率不低，而且容易检索和记录。我提供的服务「卖桃者说」也是基于邮件订阅的，每每想到这个长达十年的「一千万」服务，真是惊喜加刺激（微笑脸）。

由于 macOS 自带的邮箱略显厚重，我曾经使用并推荐过各种邮件客户端，比如 Sparrow、MailBox、CloudMagic 等。MailBox 横空出世的时候，简直是邮件客户端界的一缕清流，我当时把所有的赞美都给了这款邮件 App，结果好景不长，MailBox 被 DropBox 这股泥石流收购以后后继乏力，版本停滞，最后干脆 shutdown 了，这特么能怪谁呢？估计是一山不能容俩 Box 吧。

CloudMagic 也是轻量级邮箱的佼佼者，如果没记错的话，这款软件的费用是一次性128块。后来厂商发神经把名字改成了 Newton，这个名字让我想起了伟大的艾萨克·牛顿先生，但是和邮件有毛关系呢？并且，定价策略也改了。完全功能版的价格成了按年订阅，也就是每年都要付费的意思。50刀一年，用不用随你，态度就这么个态度。

有时候我忍不住无耻的想，有没有更好的解决方案呢？比如收发邮件流畅有力（最重要的一条），免费，对国内外各大邮箱完美支持，交互与 UI 简洁优雅，符合 Mac 用户的时用习惯，等等。

7月份网易发布了 Mac 版网易邮箱大师，终于补足了网易邮箱生态的最后一块木板，成为全平台的邮箱客户端。我提前拿到了内测版，用了几周之后，觉得这是 Mac 邮箱的一个新选择。

邮箱智能配置

网易的软件我一向比较喜欢，比如网易云音乐，有道云笔记，有道词典等等。网易邮箱大师保持了一惯的网易风格，简约，有效，自然，流畅。网易自己的邮箱服务差不多有近20年的历史，在这方面有深厚的技术和应用基础。网易邮箱大师能够登录并使用超过 120 万个邮箱服务，大部分常用邮箱，比如163、QQ、Outlook、Sohu、Sina 等都可以直接通过邮箱和密码自动登录并进行配置，IMAP/SMTP 这样的服务器科学用语，基本上就算告别普通用户了。对于需要进行二次验证的邮箱，比如 Gmail 和 iCloud 用户，只要你敢输入邮箱后缀，系统就会提示你应该怎么办，比如你输入了 geek@icloud.com，系统就会提示「如开启了双重认证，请使用APP专用密码登录」，非常方便。

网易邮箱大师通过云端账号配置引擎，帮助用户自动选择最佳的配置协议，并加以机器学习，自我优化，大大简化了以往邮箱登录时的繁琐配置。对于 Gmail 重度用户的一大利好是，一旦登录之后，你就可以扔掉梯子，在曲折的道路上平坦前行。能做到这一点，难能可贵。

UI 和交互

对于 Mac 用户最关心的 UI 和交互设计，网易邮箱大师采用了分栏式设计风格。最左侧的黑色功能区提供了邮件、联系人和配置功能。在邮件功能去包括邮件帐号、邮件列表和邮件内容，一共四个分栏的设计，操作简洁明了，没有学习成本。联系人视图采用了三分栏设计，使用的是 macOS 的联系人数据，你可以在这里维护联系人信息，有邮件地址的联系人，可以直接发起邮件沟通。设置功能区在一个界面中整合了系统设置、帐号管理、邮件签名、来信分类和关于五个功能特征，切换顺畅，使用方便。比如在来信分类里，我们可以快速设置规则，并将其应用到现有的邮件中去，同时支持快速新建来信分类。

网易邮箱大师的整个设计十分优雅，独特的分栏设计，功能菜单、邮件列表与阅读视窗一目了然，极具互联网软件风格。文本编辑器的字体、字号、字体颜色、文字背景色设置恰到好处，让写邮件变得赏心悦目。

产品细节是打动人心的最佳方式

这款软件有几个细节设计非常打动我，第一个是搜索，网易邮箱的模糊搜索功能也十分强大，无论是标题地址内容，输入之后，即可精确定位邮件。在此基础之上，系统还提供了「指定搜索范围」的功能，你可以指定搜索范围是整封邮件、联系人、主题/正文和附件，这个特性能够帮助我们凭借一点蛛丝马迹迅速在海量邮件中找到心中所想。

第二个是围绕联系人设计的「查看来往邮件」和「移动他的所有邮件」功能。在任意一封邮件点击任意联系人，都会弹出这个界面：

点击查看来往邮件，你可以看到你与这个联系人所有的邮件往来，这个功能远远超过了传统的邮件会话，让我们查找和处理单一联系人的邮件信息变得无比方便。我还可以把这个联系人的所有邮件一次性存档放入某个文件夹里。

第三个是群发单显模式。发送邮件给多人的时候，如果你想做到单独显示对方的邮件地址，只需要在发送的时候选择「群发单显」模式，这样每个人都会收到「发给自己」的邮件，邮件中不会显示其他联系人的信息。这种场景我们在工作中会常常遇到，比如产品调研，客户回访等等，没有人希望自己是收件人之一，我们都希望自己是唯一的那一个。以前有个男生追女生，本着广撒网的捞鱼原则，一封情书群发出去忘了使用密送，结果就没有结果了。我想他现在一定会喜爱这个「群发单显」功能。

技术驱动产品

所有的产品创新都离不开技术，比如网易邮箱采用了自己研发的私有协议 WZP（Wireless-Z Protocol），更快，更安全，压缩比更高；采用了智能服务器配置，让普通用户更容易一键登录；通过专有的新邮件推送通道实现了新邮件实时提醒，比如 IMAP IDLE。对于不支持 IMAP IDLE的邮箱服务，为减少邮箱的流量损耗，邮箱大师还采用服务器端轮询检查的方式，为这部分用户提供新邮件推送服务……

技术驱动产品前行，一向如此。

如果你还常用邮件沟通，打开 Mac App Store，搜索「网易邮箱大师」，即可安装使用这款优秀的邮件客户端软件。目前 Mac 版网易邮箱大师的版本是1.0.3，他们正在进行快速的产品迭代，相信未来会有更多惊喜。
