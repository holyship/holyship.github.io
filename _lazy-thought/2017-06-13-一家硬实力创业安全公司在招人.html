岗位一：技术服务工程师


人数：1-2人

地点：湖南长沙

岗位职责：

  1. 开展网络安全应急响应工作，协助业务人员了解客户的需求；

  2. 参与公司产品相关技术文档的编写；

  3. 参与解决客户在产品使用过程中的技术问题；

  4. 负责相关培训工作，为客户讲解方案并协助实施；

岗位要求：

  1. 计算机相关专业本科生及以上；

  2. 有较强的适应能力和学习能力，善于总结归纳；

  3. 具有良好的沟通能力和文档撰写能力；

  4. 具备一定的抗压能力，能适应短期出差；

  5. 有软件开发经验者优先；

  6. 熟悉网络安全技术者优先；

  7. 具有良好的团队合作精神，有意留长沙发展的优先。


岗位二：安全事件分析师


人数：1-2人

地点：湖南长沙

岗位职责：

  1. 负责各类安全信息源信息的获取与分析，梳理各类安全事件反映出的攻击者画像；

  2. 完成客户提出的安全业务响应工作。

岗位要求：

  1. 熟悉安全攻防的各类技术，有一定的逆向工程能力；

  2. 熟练掌握某种脚本语言(Python、Perl等)，能够快速将想法落地实现；

  3. 掌握一定的数据分析基础，能够快速对数据进行关联分析、探索挖掘优先；

  4. 思路清晰，善于主动思考，有创新、能独立分析解决问题，具有良好沟通能力和团队合作精神；

  5. 需要能够适应短期出差。


岗位三：安全开发工程师


人数：1-2人

地点：湖南长沙

岗位职责：网络扫描引擎的开发、维护及持续改进、优化

岗位要求：

  1. 计算机相关专业本科生及以上；

  2. 熟悉互联网产品和服务的开发过程、了解 TCP/IP 协议；

  3. 有 Python 开发经验；

  4. 熟悉 Linux 系统及 Bash 脚本；

  5. 熟悉 MongoDB、ElasticSearch；

  6. 熟悉 Web 前端开发技能者优先，包括 HTML/CSS/JavaScript


岗位四：恶意代码分析工程师


人数：1-2人

地点：湖南长沙

岗位职责：分析木马病毒技术原理，撰写技术分析报告

岗位要求：

  1. 本科以上学历, 热爱反病毒, 对计算机安全有浓厚兴趣；

  2. 了解病毒, 木马, 恶意软件工作原理；

  3. 熟练掌握 OD、IDA 等调试分析工具；（有逆向经验者优先）；

  4. 熟悉 C/C++ 以及至少一门脚本类语言，具有一定的编程能力；

  5. 熟悉 PE 结构，熟悉 Windows 内核原理以及编程；

  6. 良好的团队协作能力及用户沟通能力。
