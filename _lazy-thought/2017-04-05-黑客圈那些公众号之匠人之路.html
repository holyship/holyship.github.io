好久没推荐优秀公众号了，推荐原则：对味。


黑客这条路不易走，没有足够的创造力，何谈黑客？本期推荐的几个，无论是个人，还是已经做出的事业，那些独特的思维就已经让我受益。希望对你有帮助。


qz安全情报分析


qz，茄子，他来写安全情报我觉得是足够分量的。2008 年熟知他，就一直是一线的干货型黑客，搞了三年 Web 安全、三年二进制安全、三年移动安全，想进一步了解他，只需百度/Google：RAyH4c，这个 ID 即可。


微信号：lookvul

锦行信息安全


几个一线黑客牛人创建的新品牌，提供基于攻击者视角的新型安全解决方案，帮助政府、企业保障信息资产安全。胡鹏，是锦行的产品总监，那款攻击欺骗类产品“锦行幻云”正是由他负责。下面这个公众号也经常发布技术方向干货。

bigsec岂安科技


老板 amxku 是我的老朋友，也是国内知名漏洞库平台 Seebug 前身“Sebug”的创始人，后来由我对接，卖给了我当时所在的知道创宇。岂安科技专注于互联网业务风险控制，这也是时下热潮了。


上一篇文章“如何用最小的代价完成爬虫需求”就来自他们的分享，本来我做了充分的编辑与一些引导的，可惜一发出就变成了“原汁原味的转载”，醉了。


微信号：bigsec

虚拟机


厉哥的公众号，一名很有创意的黑客，这两年走上了跨界创业不归路，搞出了个 Ruff（一种物联网操作系统），非常酷，软件定义硬件，可以用 JavaScript 进行编程。当年他给我说要搞这个的时候，我并不觉得惊讶，毕竟他是厉哥。


微信号：mintshow

黑白之道


怪狗，华盟网的老板，很有个性的一名黑客，尤其是前段时间的“中国黑客对乐天宣战”，他是直接发起人。我们观点有时向左，但这并不影响我和怪狗成为朋友。他主力负责的公众号“黑白之道”，可以了解到这个圈子的一些及时动态。


微信号：i77169


上面推荐的不少朋友都已经走在创业路上，他们一个人时就已经很优秀，在各自的黑客分支领域如匠人一般存在。现在都有自己的团队，有更大的事业等着他们去雕琢。


这条路，我们几个懒人也在路上:-)


最后推荐个特别的公众号：


内推小王子


阿混，专注于互联网内部推荐，国内任何一家互联网公司都可以帮你内推，目前各大知名互联网公司都有成功案例。不管是需要找工作或招聘的你，都应该关注一下。


微信号：neituixiaowangzi

我们需要的人才也会找阿混推荐，绝不客气。
