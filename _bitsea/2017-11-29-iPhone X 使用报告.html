iPhone X 入手半个多月了，我想从使用者的角度来谈谈自己的感受。


首先，它的名字不是很好叫。X是罗马数字10的意思，所以应该念成：iPhone Ten，爱疯疼。但爱疯不疼，疼的是我的肾，这个叫法不科学。我们中国人习惯性地把字幕X念成“叉”，比如说著名的Windows XP，它的民间发音是：瘟倒死 叉屁，提醒大家要远离传染病，否则死了之后屁股都还不得安生。所以，iPhone X 目前的流行叫法是：爱疯叉，或者简称：粪叉。


对此，我实在是无能为力，iPhone X 可能是我买过的最贵的粪叉。


起先我非常担心自己成为了苹果公司的小白鼠，iPhone X 不同于之前的历代产品，我并不确信iPhone X 能够完美运行，也许我会在很长一段时间里忍受它的缺陷。但用下来的结果还好，基本上没有什么大的毛病。信号不错，声音清晰，联网稳定，发热量也不高，无非是有些时候会自己重启。但这又能怪罪谁呢？喜欢自主重启，说明iPhone X 有了自我，产生了性格，它的性格就是喜欢重启，相当于人类的暂时离家出走。可谁从小到大没有动过离家出走的念头呢？我们又何必对一台手机如此苛责。


我第二个担心是iPhone X 的Face ID 问题。


事实上，我觉得苹果最早推出的指纹解锁并不好用，在两代更新之后才达到80%的解锁成功率。我甚至因此养成了解锁前先舔大拇指一下的习惯，有天在动物园熊馆门口那么做，熊看见了，立即流露出非常熊性化的表情，冲着我扬起下巴打招呼，意思大概是：兄弟，不错啊，你怎么逃出去的？Face ID比指纹锁要好太多了，基本上拿起手机就已经悄然解锁。在夜间很黑暗的环境里，也能靠屏幕发光照亮面部，直接解锁。我对这个功能挺感动的，以前无论是输入密码也好，还是用指纹解锁也罢，都有种操弄机器的感觉。但现在我有一种手机“认识”我自己的感觉，打个照面它就知道我是谁，让人觉得iPhone X 真挺温暖的。


就一点不好，如果你是躺在床上侧卧玩手机，因为有一部分脸陷入了枕头，iPhone X 就死活认不出你来，非得翻身仰面朝天不可。但这也说明iPhone X 是一台天真无邪的手机，比人类强多了。人类都是从床上起身，穿起裤子才不认人。而iPhone X 像是个小婴儿，躺在你身边发现你的脸不见了半边，就立即哇哇大哭了起来，觉得爸爸变掉了。


然后就是“留海”问题。


因为iPhone X 上市不久，许多公司还在观望，并没有第一时间做了针对iPhone X 屏幕的适配工作。而且，中国人你懂的，即便做了适配，也是用最廉价的手法实现---直接把屏幕上方变成很宽的一根黑条，让内容在留海之下显示，反而让屏幕显得更小了。这直接带来了两个后果，一个是屏幕显示比例出问题。例如B站，在iPhone X 上就看不了字幕，直接遮盖掉了。另外一个是屏幕上的操作按钮位移，你老想点左上角退出，结果怎么点死活都点不中，这种感觉非常讨厌。就像是偶尔出差，半夜起来上厕所，按照家里的习惯走到地方开尿，睁开眼发现自己正冲着沙发滋水，而你的同事正一脸惊恐地看着你，头发湿漉漉地。


但这一点也不构成多大问题，随着iPhone X 进一步铺货，大部分APP公司会屈服的，迟早还是会投入人力物力，做好iPhone X 的屏幕适配工作。毕竟，跟人较劲那叫节操风骨，和钱较劲那要么是真有钱，要么就是真有病了。


至于说到手势控制问题，也没有我想象中那么困难。


iPhone X 发明了一整套新的手势操作系统，截屏、关机、切换APP、关闭APP、呼叫控制中心都和过去不一样了。但是，苹果就是有这种本事，每次新设计一套手势操作系统出来，很快就会让你忘记之前的那一套东西。用了不到两天，当我切回我的iPhone 7 Plus 的时候，就已经彻底忘记了Home键的存在。老不自觉地往上划屏幕，让它觉得非常困惑，不明白为什么我老要呼叫出控制中心来。到了这种时候，我才感觉原来的物理Home键有多么生硬和不便，完全比如我用手指在屏幕上下拉上滑。


不过有一点我非常不爽：iPhone X 设计的关机方式非常诡异，要求你同时按下屏幕右边的电源键，和屏幕左边的音量“+”键。唯有这样，才能呼叫出一个单独的关机页面。但是在那个页面里，同时还有个紧急求助按钮。只要关机页面出现，3秒钟之内紧急求助功能就会激活，发出震耳欲聋的警报声。这样，固然可以在起飞前向全客舱的人宣布自己买了一台iPhone X ，但手机报警这件事情真的很愚蠢。而且，不知道苹果公司是怎么想的，如果我真的需要用到这个功能，警报声响起，歹徒第一时间就知道我在报警，那么本来死的可以是隔壁老王，但是现在挂掉的肯定是我，有没有考虑到我的感受？


话说iPhone X 的听筒声真的是比前代强大了不少，不单声音洪亮，而且有立体声的效果，感觉就在对方边上聊天。希望它能撑久一点，因为我的前两部手机最后都是听筒损坏，声音越来越小，狠不得给手机挖一下耳屎；有些人可能会比较关心iPhone X 的摄像头，坦白说我没有发现什么特别的地方。每天早上起来，我都要拍一张天空照，换了iPhone X 之后，天空特别蓝而且冷，和我肉眼看到的景色不大一样。换句话来说，在iPhone X 里霾的影响被消除了不少，可以说iPhone X 是一部有觉悟的手机了。


我自己基本不会自拍，因为一则不好看，二则会同步到电脑上，而这年头电脑硬盘容易损坏，我还是得省着用。所以，在自拍效果这个问题上我没有什么发言权。但库克毕竟是一个有觉悟，追求进步的CEO，想必他在这个问题上能够处理妥当，拍出一个你更满意的自己。你高兴就好，反正我们都不信。


总之，iPhone X 是一部比较成功的手机，虽然不如iPhone 4 那么惊艳，但也已经足够好了。价格的确是贵了一点，但是考虑到只要一关机，半层楼的人都知道你换了iPhone X ，也算是非常符合中国用户的使用心理了。九千八百八，这钱不白花，库克送了广播小喇叭。抱歉我不能用比较硬件参数的方法来谈iPhone X ，毕竟我吃辣子鸡的时候，并没有把盘子里的鸡块重新拼回一只鸡，检查有没有给足两条腿、两只翅膀的习惯。


我用了，觉得挺好，希望我的经验能对你有所帮助，事情就是这样。
