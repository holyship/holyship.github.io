演技这种东西吧批判起来不难，甚至我偶尔也跟大家聊大幂幂、大宝贝等等一线明星的演技，然而真正去演一演大家都会觉得不容易，因为不仅仅要控制自身的眼神和面部表情，更是要对于肢体动作有较为系统的规范。


在一部剧中，往往吸引我们的是有爆发力的表现，如哭号等情绪，或者演一些和正常人反差太大的角色，比如说变态（小丑、汉尼拔等），精神分裂（24个比例，《致命ID》），或者傻子。



没错我们今天就来说说电影里的傻子角色，有好多都是童年回忆类，不知道大家还记得几个？



《Hello！树先生》 (2011)

王宝强 / 树



有很多小伙伴不喜欢王宝强，因为王宝强卖座的电影大多数都是装疯卖傻，一口不标准的普通话重重复复演雷同角色非常令人厌烦，而在《hello!树先生》这部小制作电影中，王宝强奉献了影帝级别的演技。


从刚开始的社会边缘人物，到疯疯癫癫成为村里的“先知”，非常有黑色幽默的气氛，而王宝强对于傻子的演绎包括他的一些小动作，都非常写实。




《一个勺子》 (2014)

金世佳 / 勺子



名字说是一个傻子，但是却在写一家子遇见傻子之后的转变和生活，金世佳在其他电影里演个啥都被人说成是展博，在这部电影中终于没人说他是展博了，因为他演的是一个可能连数都不会算的傻子。电影给傻子的发挥的余地不是很多，不过也算是交了个及格卷。




《阳光灿烂的日子》 (1994)

姚二嘎 / 傻子



这部电影有一个非常出名的傻子：古伦木，当大家叫他的时候，他总会回一句“欧巴”，就像是样板戏中的口号一样，这个傻子的设置无疑植入了姜文的个人政治意图和倾向，最后回的不是欧巴而是傻逼实在是点睛之笔，原来傻的不是他，是那段青春。



《海洋天堂》 (2010)

文章 / 王大福



在曾经文章非常火的时候，眼袋君还是个学生党，我们全宿舍窝在一起看了他演的催泪电影《海洋天堂》，这部电影中文章和李连杰搭戏，亲情十分感人。演员人品不说，感觉文章非常适合演这种极端人士……




《傻瓜》 (2008)

车太贤 / 李乘龙



如同往常韩国爱情喜剧的套路，善良老实的傻瓜和忧愁的有诸多心事的妹子，车太贤无论什么情况面对着大家都是善良的笑容，每个细微的小动作都非常暖心，这才是一张没有被污染的白纸嘛。




《搜神传》 (2008)

钟嘉欣 / 好彩妹



不能说好彩妹是傻子，但是她说“笑口常开好彩自然来”的时候我觉得她很有智障气质……




《肥猫寻亲记》 (2000)

郑则仕 / 肥猫 



童年回忆，现在剧情忘得七七八八，大概只能记住这剧非常催泪，还有肥猫的各种神态表情，小时候我一直以为他真的是个傻子……
