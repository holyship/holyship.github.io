漫长的暑假就要结束，是不是发现自己既没有按照计划中的那样好好看书看电影，也没有按照想象中的那样好好出去社交出去玩闹，是不是发现自己根本没有减肥不说，还肥了大一圈？


是不是突然因为自己什么都没干暑假却莫名其妙结束了而觉得懊悔不已？


不用担心，接下来我给各位推荐几部能够一口气解决掉你剩余休假和睡眠时间的高分精彩韩剧，让你能够彻底忘记“自己什么都没做”的客观事实，在自欺欺人中完美的度过这个炎热的夏天



信号

(2016)

导演: 金元锡
编剧: 金恩熙
主演: 李帝勋 / 赵震雄 / 金惠秀 / 张铉诚 / 金元海 / 郑涵妃


根据真实事件改编。


剧情围绕着一个可以与过去通话的无线电对讲机展开，讲述了两个拥有正义感却身处不同时空的警察利用这部对讲机破获那些长期未结案件的故事，在主线认真破案逻辑合理的基础上，慢慢交代着几个主角那些令人有共感的经历，像《迷失》，却比《迷失》更加精悍短小，主线明确，充满人情味。



实力派影星赵振雄镇场，青龙奖最佳男演员李帝勋和性感知性美女金惠秀辅助，搭配上张炫诚，郑海均，金元海这些老戏骨，配合这细致精巧充满戏剧冲突的剧本，能让你看着看着，莫名其妙，就过了一个通宵。





特殊失踪专案组：失踪的黑色M

(2015)

导演: 李胜英
编剧: 李幼贞
主演: 金康宇 / 朴熙顺 / 赵宝儿 / 朴素贤


以失踪案件为主题的刑侦单元剧，每一个案件都是极高水准，有着扎实的现实基础和感情核心。



但这都不是最重要的，令这部电影大放异彩的，是由金康宇饰演的男主角吉秀贤——一个有着黑暗过去，亦正亦邪办案执法长期游走于灰色地带，秉行着自己的正义标准和办事手段的冷静克制高智商高情商FBI出身天才刑警。



他是整部剧中最亮的光，金康宇身上的知性气质完全赋予在这个角色之上，让人很自然的对他目不转睛——尤其，是看完第四集之后。




九回时间旅行

(2013)

导演: 金炳洙
编剧: 宋载正 / 金允珠
主演: 李阵郁 / 赵允熙 / 全卢民 / 李成俊

李振郁帅得宇宙爆炸，这是其次。

赵允熙萌得天地无光，这是添头。


这部《九回时间旅行》最牛逼的地方，就是它基本上无可挑剔的神级剧本。



在这部剧中，你不会察觉到任何时光穿越剧中会出现的一些BUG戏码，电视剧从头到尾遵守着它一开始便提出的诸多规则，二十集看下来，你根本察觉不到自己是在看一部“戏”，你只会觉得自己看完了一个传奇男人精彩没有遗憾的一生。



在李振郁那如大海般清澈的眼光中，你能很自然的为了看完这部电视剧而熬过一个又一个的漫漫长夜。


如果你还没有看过任何韩剧，建议你从这一部看起，它一定能够改变你对韩剧的偏见。





未生

(2014)

导演: 金元锡
编剧: 郑有贞 / 尹胎镐
主演: 任时完 / 李圣旻 / 姜素拉 / 姜河那


《未生》讲述了一个没有资历，没有人脉，只会努力的年轻人在社会上磕磕绊绊的故事，向观众展现了一个没有那么多情感，没有那么多贵人，充满了勾心斗角的残酷职场。



这是一部几乎没有任何狗血剧情的写实韩剧。


在里面你能看现实中的基本规则——坏人不一定会受到惩罚，努力的人不一定会成功，拼了命完成的工作不一定是正确的。


以及，大部分的人出了社会，都得从孙子做起，职场和工作，总会教你什么是长大，总会慢慢消磨掉你所有的任性和坏脾气。



在冷峻克制的镜头语言和剧本下，这部韩剧，能让没有进社会的学生提早认识到现实的残酷，也能让忙忙碌碌整天做牛做马的上班族得到感同身受的宽慰。


打开它，让男主角张克莱那苦逼又会出现点小阳光的日子，陪伴你暑期的最后几天。
