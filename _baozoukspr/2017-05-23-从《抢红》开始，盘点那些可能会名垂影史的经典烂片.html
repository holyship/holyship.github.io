最近眼袋君的朋友圈都因为《抢红》一片沸腾（上一次这么激动还是《摆渡人》上映的时候）虽然大家都一副不太想花钱去电影院看的样子，但是并不妨碍他们讨论（吐槽）的热情。



说到我国的烂片史，可能要追溯到很多年前，张艺谋拍的那一堆意在进军好莱坞的商业大片。那时候观众和影评人对烂片的容忍度都很低，从《英雄》、《夜宴》到《满城尽带黄金甲》，无一不被批得体无完肤。但是现在回头看，我不禁要真情实感地心疼一把张艺谋……


因为从那以后，国产烂片的下限可以说越来越低，这三部片子放到如今，简直不值一提、无足挂齿。


在我国，烂片并不稀奇，但拍出一部现象级的烂片，却需要天时地利的条件。


首先，它要烂得有个性、有爆点，其次，它还要有足够的知名度，缺一不可。


下面眼袋君就来给大家回顾一下，那些可能会在中国影史上留名的——经典大烂片！



抢红

2017

导演: 黎明

主演: 张涵予 / 黎明 / 王耀庆 / 杜鹃

类型: 动作 / 犯罪


上一次在电影方面听说黎明，还是因为那部和美国某ffff团经典电影同名的另一个烂片《消失的爱人》。在演员们一窝蜂跨界涉足导演事业后，黎明也不甘示弱地交出了自己的答卷——《抢红》


据看过的人反馈：你完全猜不出电影下一分钟的剧情还能有什么转折，完全刷新观影认知的电影。年中还未到，华语最烂就被提前预定了。


邓超可能是唯一觉得高兴的人吧。



无极

2005

导演: 陈凯歌

主演: 张东健 / 张柏芝 / 真田广之 / 谢霆锋 / 刘烨 / 陈红 / 于小伟

类型: 剧情 / 动作 / 奇幻


这是一部恶搞视频比电影还火的电影。这是一部电影改编小说由另一位本榜知名导演撰写的电影。这是一部直接把一个九分导演拖到五分的电影。



这也是一部让导演玻璃心发作亲自上阵手撕视频阿婆主的电影。


试问当年，有谁不知道《一个馒头引发的血案》？


如今，我早已忘记了电影讲了个什么故事，但我依然深深地记得，一个馒头引发的血案……


不过现在我相信了，陈凯歌是冤枉的！



富春山居图

2013

导演: 孙健君

主演: 刘德华 / 林志玲 / 张静初 / 佟大为 

类型: 剧情 / 动作 / 惊悚


这部电影，可以说开启了一个中国烂片史崭新的时代，大制作、大明星、高票房，以及不知所云的剧情、一塌糊涂的节奏、莫名其妙的对白，用wifi看都嫌浪费时间。


它直接导致了某知名电影营销博主从如日中天到糊进地心的致命转折，为我国写电影软文盈利的一批影评人敲响了警钟。


真是一部当之无愧的现象级烂片！



小时代

2013

导演: 郭敬明

主演: 杨幂 / 柯震东 / 郭采洁 / 郭碧婷 / 谢依霖 / 陈学冬 

类型: 剧情 / 爱情


《小时代》系列，相信不用眼袋君介绍了。它曾一度被奉为中国观影群众的智商检测标准，并让无数少男少女为之疯狂。



它开启了非专业人士（以作家为主）投身电影行业，为我国烂片添砖加瓦的新潮流。



它养活了一大批以吐槽电影为己任的阿婆主和影评人，当我们检测另一半真爱的时候，往往会以是否愿意陪同观看《小时代》为试金石。



直到如今，它仍然是人们茶余饭后的谈资……



爵迹

2016

导演: 郭敬明

主演: 范冰冰 / 吴亦凡 / 陈学冬 / 陈伟霆 / 郭采洁 / 杨幂 / 林允 / 严屹宽 / 李治廷 / 王源

类型: 奇幻


国内首部真人CG电影，集齐了微博流量榜上80%的明星和演员，启用了据称国内动画最高水准的制作团队，投入的资金大约可以拍十部《师父》、二十部《路边野餐》，却在上映后收获了票房与口碑的双重失败，导演泪洒当场，曰出了“是不是我死了你们才不骂”的肺腑之言。


然而这并没有什么卵用。



恶棍天使

2015

导演: 邓超

主演: 邓超 / 孙俪 / 代乐乐 / 梁超 / 杨新鸣 / 王砚辉

类型: 喜剧


又一部演员跨行导演的经典烂片，上映于年底，如同一匹黑马，一上来就直接pk掉了之前十二个月的所有烂片，荣登烂片榜首。引发了一次观众与影评人的集体狂欢。


邓超身体力行地向观众们揭示了一个道理：好演员，不一定是好导演！



长城

2016

导演: 张艺谋

主演: 马特·达蒙 / 景甜 / 佩德罗·帕斯卡 / 刘德华 / 威廉·达福 / 张涵予 / 鹿晗 / 彭于晏 / 林更新 / 郑恺 / 黄轩 / 陈学冬 / 王俊凯

类型: 剧情 / 动作 / 奇幻 / 冒险


如果说《英雄》《夜宴》《满城尽带黄金甲》时的张艺谋被喷很冤，那么拍《长城》的老谋子可以说一点也不冤了。


可能是《建国大业》以后启用大腕最多的国产商业电影，请来了流量小生、好莱坞巨星为某背景成谜的小花抬轿作配，国师团队倾力打造，世界级特效水平，与西方文明亲密接轨，英文对话高端大气上档次。


然而除后期包装以外与电影内容有关的部分一塌糊涂，它的投入有多高，拍得就有多烂。在海外市场更是石沉大海扑得悄无声息。


有句话怎么说来着：丢脸丢到国外去了！



摆渡人

2016

导演: 张嘉佳

主演: 梁朝伟 / 金城武 / 陈奕迅 / 杨颖 / 张榕容/ 杜鹃 / 熊黛林 / 大鹏 / 马苏 / 崔志佳 / 贾玲 /李宇春 / 鹿晗 / 欧弟 / 纳豆 / 金士杰 / 柯佳嬿 /李灿森 / 喜翔 / 曹可凡

类型: 喜剧 / 爱情


这是一部三流作家拍的十八流电影。它的烂，直接导致了另一位影迷们心中的白月光导演的晚节不保。



毫无逻辑的智障剧情、矫揉造作的愚蠢台词，让两位国内的男神级演员都无力回天甚至惹得一身骚。无数影迷为此肝肠寸断，含泪脱粉。


对此我只想代表广大群众问张导一句：王家卫梁朝伟金城武欠了你多少钱，我替你还好不好？！
