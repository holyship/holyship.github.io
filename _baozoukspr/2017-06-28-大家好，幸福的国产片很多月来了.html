各位小鲶鱼们你们好，一转眼六月份又要过去了，在上一周，鲶鱼君还被卖拷贝的狂轰乱炸弄得神经衰弱，还在被《异形：契约》中可爱的异形宝宝萌的肝颤，然而这些代表着资产阶级低级趣味的电影，根本就没办法和我们的国产正能量电影相比！


所以鲶鱼君是非常非常期待接下来的国产电影那啥月的，因为我们国家也有非常非常多的优秀电影，这些电影完全不能被埋没，理应得到更多的票房支持，理应在豆瓣上得到很多很好的评价！



明 月 几 时 有

导演: 许鞍华
主演: 周迅 / 彭于晏 / 霍建华 / 郭涛 / 春夏 / 黄志忠 / 蒋雯丽 / 梁家辉 / 王菀之 / 吕良伟 / 叶德娴 / 鲍起静 / 李灿森 / 梁文道 / 张兆辉 / 冯淬帆/ 苑琼丹 / 唐宁 / 卢巧音



上映时间：2017年7月1日


上海电影节金爵奖提名电影，许鞍华的新片，香港沦陷下时代背景中的人物们，女性细腻的人文关照下有血有肉的人，然而这次许鞍华可能有些失手了，豆瓣评价6.7分。


大多数的负面评价在于剧情节奏散乱，人物也站不住脚，还有演技问题，但至于成片鲶鱼君也没有看过，不过你去电影院看这片，肯定是比看《怦然星动》之流值回票价的。



与 君 相 恋 100 次

导演: 月川翔
主演: miwa / 坂口健太郎 / 龙星凉 / 真野惠里菜



上映时间：2017年7月6日


这部电影2月份在日本上映，现在豆瓣评分是7.5分，可以说是不错的分数了。


《贤者之爱》中“小狼狗”龙星凉以及盐系美少年坂口健太郎，讲述了一个非常日系的爱情故事，为了拯救自己喜欢的女孩，男主穿越过去和她相恋，这种又甜又虐的故事，如果能在细节上动人就更为难得，上映之后估计会赚很多人的眼泪吧。



京 城 81 号 Ⅱ

导演: 钱人豪
主演: 张智霖 / 梅婷 / 钟欣潼 / 耿乐 / 邬君梅 /王水林 / 陈冲



上映时间：2017年7月16日


我没想到这部电影会拍第二部，在大家的印象里，院线国产恐怖片都是什么女生宿舍X仙的低成本软色情，《京城81号》用较为靠谱的卡司和比较精良的制作一定程度上扭转了大众的想法，不过剧情还是非常薄弱，剧本还是非常不行，人是没吓着，倒是让人在电影院笑了一场，所以这个第二部嘛……


应该会吸取教训大力改进，肯定不负众望！



神 偷 奶 爸 3

导演: 凯尔·巴尔达 / 皮艾尔·柯芬
主演: 史蒂夫·卡瑞尔 / 克里斯汀·韦格



上映时间：2017年7月7日


这种电影，只要口碑不flop到地心，维持6分左右就还是会有大批大批的人去看的，老少皆宜合家欢，不过真摸着良心说，神偷奶爸这系列早就应该完结了……不过有小黄人啊！小黄人啊！为了小黄人干啥都行！（来自一个酷爱小黄人的鲶鱼）



绝 世 高 手

导演: 卢正雨

主演: 卢正雨 / 郭采洁 / 范伟 / 蔡国庆 / 陈冲 /仓田保昭 / 孔连顺 / 黄龄 / 杨迪 / 柯达



上映时间：2017年7月7日


本月非常期待的一部国产片，卢正雨从恶搞邵氏的短片到《嘻哈四重奏》，无厘头的搞笑风格深得周星驰精髓。



在国产喜剧赶着卖低俗卖黄段子的时代，在尬喜剧堆满天的时代，鲶鱼君期待能从卢正雨作品中看到不同的喜剧元素，再加上范伟这个广大观众一看就想笑的影帝加持，希望卢正雨能给国产喜剧片带来点不一样的东西。



秘 果

导演: 连奕琦
主演: 陈飞宇 / 欧阳娜娜



上映时间：2017年7月7日


饶雪漫操刀的青春疼痛剧本，靠谱程度高一些的导演连奕琦，鹿小葵少女欧阳娜娜，陈凯歌的儿子陈飞宇。


应该会非常好看的！一定要去看！



悟 空 传

导演: 郭子健
主演: 彭于晏 / 倪妮 / 欧豪 / 余文乐 / 郑爽 / 乔杉 / 杨迪 / 俞飞鸿



上映时间：2017年7月13日


如果你是85后，那么你一定经常在贴吧或者QQ签名中看到这句话：我要这天，再遮不住我眼，要这地，再埋不了我心。


今何在的原著《悟空传》，本身就带着愤世嫉俗桀骜不驯的气质，不知道燃起了多少中二少年的热血魂，这个小说被搬到银幕上，鲶鱼君是有点担心的，预告片中彭于晏操着台湾腔大吼，各路神魔特效都合格，却总是能让人联想起《西游记之孙悟空三打白骨精》。



观众期待在电影院看到与众不同的狂与锐气，也期待再次感受到“让诸佛都烟消云散”的气势，但说实在话，这种题材真的不好拍，剧本影像化的改编是一个方面，导演的功力是另一方面，某些不能说的是再一个方面，上映观望吧。



大 护 法

导演: 不思凡



上映时间：2017年7月13日


这片又有个名字叫《大护法之黑花生》，怎么看都是在考验人的普通话。


这部动画电影自己给自己分级，建议13岁以下的小盆友们不要观看，预告片中就能感受到这部电影的重口味，爆头，血流遍地，奇异怪诞的造型，颇有深层含义的剧情，都注定了这是一部不服务于低幼儿童的电影。



剧情的架构也十分cult，大护法来到的小镇，居民形似花生，这些花生们反对一切外来事物，同时当他们身上长出蘑菇的时候，就会被枪决，这片能上院线的确挺让人惊讶的，非常期待。



青 禾 男 高

导演: 蒋卓原
主演: 景甜 / 欧豪 / 黑木美纱



上映时间：2017年7月13日


我大甜甜宇宙再添力作，类似《热血高校》的剧情，一排荷尔蒙炸裂的帅哥，把我甜众星捧月，这种罕见的题材，这种引导青少年走正路的正能量电影，当然是要去看啦！！



父 子 雄 兵

导演: 袁卫东
主演: 大鹏 / 范伟 / 张天爱 / 乔杉 / 邬君梅 



上映时间：2017年7月21日


哟，大鹏宇宙也有新作品了，看海报质量心里就有数系列，当然是去资辞一个！



建 军 大 业

导演: 刘伟强
主演: 刘烨 / 朱亚文 / 黄志忠 / 王景春 / 欧豪 /刘昊然 / 马天宇 / 刘浩 / 张艺兴 / 李易峰 / 白宇/ 梁大维 / 于和伟 / 吴樾 / 白客 / 刘循子墨 / 董子健 / 陈晓 / 杨大鹏 / 保剑锋 / 宋洋 / 叶筱玮 /释小龙 / 李现 / 李岷城 / 王昭 / 李沁 / 宋佳 / 关晓彤 / 马伊琍 / 周冬雨 / 冯文娟 / 王庆祥 / 韩庚/ 王挺 / 陈赫 / 霍建华 / 张天爱 / 余少群 / 张涵予 / 余皑磊 / 周一围 / 杨祐宁 / 马浴柯 / 郑元畅/ 佟瑞欣

Cast非常厉害，李易峰、关晓彤、韩庚、陈赫、张艺兴等等鲜肉小花纷纷加盟，业内各界都非常重视这次演出，你想想，当你们学校组织看电影的时候，你居然能够在那个场合看到自己的爱豆，是不是非常兴奋？
