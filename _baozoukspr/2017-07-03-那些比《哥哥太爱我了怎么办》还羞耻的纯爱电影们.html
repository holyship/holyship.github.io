自从上次介绍了魔幻言情日剧《哥哥太爱我了怎么办》之后（文章戳→套路玩得深，谁把谁当真），眼袋君就经常在后台收到求推荐纯爱片的信息。说到纯爱（不是那个写作chun ai读作dan mei的纯爱），将这一片种玩到极致的当然是盛产纯爱漫的岛国人了。


今天我们就不提已经脍炙人口的《东京爱情故事》《一吻定情》《流星花园》《花样少男少女》了，来安利一些近几年比较新的日本纯爱电影吧。



昨日的我与明日的你约会

2016

导演: 三木孝浩
主演: 福士苍汰 / 小松菜奈 / 东出昌大 / 山田裕贵



改编自七月隆文著作的轻小说作品《昨日的我与明日的你约会》。


影片讲述京都美术大学学生南山高寿和他一见钟情的女生福寿爱美的爱情故事。


男主角某日遇见了一位和自己同岁的神秘女子，这位女子在我的心中如同高岭之花一般神圣不可侵犯，后来才知道在小时候就早已与她认识，我和她的关系发展迅速二人心心相印，但是她的身上却隐藏着一个秘密……



一部玩平行时空法则梗的催泪电影。看似偶然的命运其实是必然，在错误的时间遇到正确的人，造就了一段虽遗憾却也美好的奇缘。


推荐指数：❤❤❤❤



狼少女与黑王子

2016

导演: 广木隆一
主演: 二阶堂富美 / 山崎贤人 / 铃木伸之 / 门胁麦



爱慕虚荣的高中女生绘里香偷拍了一张帅气男生的照片来向朋友炫耀“男友”，结果被偷拍者竟是同校校草恭也。为了圆谎绘里香只得请求恭也假扮她的男朋友，却因此得知原来外表光鲜亮丽如王子般温柔地佐田恭也的本质竟然是腹黑+抖S的结合……



光看名字就是十分典型日系少女漫画展开，抖S男孩与虚荣女孩的青春校园喜剧一看就是一出“霸道校草爱上我”的戏码，虽然是陈年老套路，但是套路之所以是套路，也是因为它总是有效嘛！


推荐指数：❤❤❤♡



闪烁的爱情

2015

导演: 广木隆一
主演: 福士苍汰 / 有村架纯 / 山田裕贵 / 佐藤亚璃纱



改编自同名少女漫画。


高一女生木下仁菜子悄悄喜欢着同年级的一之濑莲，一之濑虽然堪称多数女生心仪的对象，但是他冷若冰川，将斗胆表白的女孩们拒之千里之外。不过幸运之神不经意间眷顾了仁菜子，电车上偶然的交谈消除了横在她和一之濑之间的樊篱，她以笨拙的方式追随者对方的脚步，发自心底为对方呐喊和鸣不平，而一之濑似乎也对这个多少有些特别的女孩青眼相看。


只是青春的爱情绝非如此简单，是永大树、永麻由香、安堂拓海、杉本真央等人的出现都让初恋物语变得复杂起来……



冷酷王子和可爱少女的纯爱故事，但是港真男主不够帅，有点撑不起这个角色，反正眼袋君是冲着有村架纯妹子看下去的……


推荐指数：❤❤❤



我才不会对黑崎君说的话言听计从

2016

导演: 月川翔
主演: 中岛健人 / 小松菜奈 / 千叶雄大



电影改编自マキノ创作的少女爱情类漫画。因为父亲人事调动而开始了宿舍生活的由宇，本以为能够和白王子同居人兴奋，但却被人称黑恶魔的黑崎盯上。因为反抗黑崎而被夺取初吻，之后一直要对黑崎言听计从……



少女漫届的红白玫瑰套路，一边是以正亦邪坏坏的“黑恶魔”，一边是彬彬有礼男神范儿的白王子。一次过满足两个愿望，虽然通片充斥着让一个理智成年人忍不住捂脸的羞耻play，但是为了少女心，理智算什么东西？！


推荐指数：❤❤❤



只要你说你爱我

2014

导演: 日向朝子
主演: 川口春奈 / 福士苍汰 / 市川知宏



本片根据漫画家葉月かなえ的同名原作改编。

 

东明高中的黑泽大和是所有女孩崇拜追逐的对象，而他在一次意外遭遇后，偏偏对几乎没有人注意的女孩橘梅产生了兴趣。他自顾自将电话号码留给对方，偏巧小梅某晚被人跟踪，不得已打电话给黑泽，赶来解围的黑泽借机夺去了女孩的初吻。


当年因某起事件，小梅封锁了自己的内心，可是阳光开朗的黑泽却蛮横地开启了她的心扉。随着交往时间的延长，他们对彼此的了解也越来越深。曾经单调的校园生活变得多姿多彩，当然悲伤和苦涩则成为恋爱之路上不可或缺的调味剂……



万人迷男主爱上普通女主的故事。似乎每个日本少女漫里都存在着这样一个瞎眼的高富帅和一堆瞎眼的追星少女。故事没有什么起伏，最大的看点可能就是男主和女主之间的强吻戏了……


推荐指数：❤❤♡



女主角失格

2015

导演: 英勉
主演: 桐谷美玲 / 山崎贤人 / 坂口健太郎 / 福田彩乃 



本片根据幸田桃子的同名漫画改编。


一座平凡的高中内，不善与人交流的帅哥寺坂利太超受女孩欢迎。不过在超爱幻想的女孩松崎羽鸟眼里，那些女孩都是形同杂鱼般的人物。小学时候，利太因为母亲与人私奔而受到欺负，这时候羽鸟出手相助，两人也因此成为好友。自信的羽鸟相信自己和利太就是命中注定的一对。


谁曾想童话般的爱情故事居然朝着意想不到的方向展开。同班其貌不扬的女孩安达未帆被人欺负，利太出面帮助了她，两人竟由此成为男女朋友。眼看自己从女主角沦为杂鱼配角，羽鸟的内心产生前所未有的裂变……



设定上有点反套路但结局依然走向套路的纯爱片。这是一个看到开头就能猜到结局的故事，因为，在日本纯爱电影的世界里，青梅竹马，四个字就是最好的剧透。


推荐指数：❤❤❤



邻居同居LDK

2014

导演: 川村泰祐
主演: 刚力彩芽 / 山崎贤人 / 中尾明庆 / 冈本玲



本片根据渡边あゆ的同名漫画改编。


为了帮好朋友小萌出气，独自在公寓居住的高中女生西森葵找到学校头号帅哥久我山柊圣理论，结果不慎将对方弄伤。小葵好不容易将柊圣带到家中，却发现这个坏心眼的男生就住在自己隔壁。葵帮柊圣做饭时不慎引发火灾，无处居住的柊圣自顾自住进西森家中，两个高中生就这样展开别别扭扭的同居生活。


柊圣似乎尤其喜欢捉弄小葵，而小葵对之也从最初的厌恶悄悄转为喜欢。在此期间，一直照顾小葵的大学生三条亘和柊圣的前女友水野樱月相继出现，在两个当事人交替激起波澜……



狗血白烂的少女言情套路，我帮我的闺蜜打了拒绝闺蜜的男神，男神却爱上了画风耿直的我。还有让人少女心砰砰乱跳的同居环节，女主和男主为了向好友们隐瞒同居做的傻事让眼袋君忍俊不禁。


推荐指数：❤❤❤
