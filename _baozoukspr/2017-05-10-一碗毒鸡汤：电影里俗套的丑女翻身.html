在这个鲶鱼君已经热成鲶鱼汁的炎炎夏日，是时候来一点毒鸡汤为生活增加一些绝望了。各位僵尸粉们应该知道，在这年头，废柴逆袭已经成为各种网文屡见不鲜的题材，尽管老套，却仍然能够让观众看得美滋滋，同样，丑女翻身在影视剧中也占据了一席之地，能够给人一种另类的爽感，尽管一辈子也变不成女主，看看过过瘾也是很吼的！



《窈窕淑女》

My Fair Lady

导演: 乔治·库克
主演: 奥黛丽·赫本 /  雷克斯·哈里森 / 斯坦利·霍洛威

奇怪口音粗俗卖花女被语言学家调教一朝变贵妇，气质高贵甚至得到皇室垂青，这部电影里，丑女变身不是因为人家本身就是丑女，而是没钱的美女变得有钱可以打扮自己之后变得更美了而已，尽管电影的表现和萧伯纳的原著有些差距，最后结局也让人比较憋屈，但是赫本的时装秀实在是太抓人眼球了。



《丑女大翻身》

미녀는 괴로워

导演: 金容华

主演: 金雅中 /  朱镇模 / 林玄植

大家应该都看过的电影，翻身界的殿堂级影片。超胖女主出车祸后整容减肥变身天仙，从此凭借自己甜美的歌声走上了俘获男神的道路，这片将一般网络爽文的剧情拔高了一个level，最终引向了真诚、心灵美的主题。这片演技在线有很强的感染力，不知道比唐嫣的《克拉恋人》高到哪儿去了。



《公主日记》

The Princess Diaries

导演: 盖瑞·马歇尔

主演: 安妮·海瑟薇 / 朱莉·安德鲁斯 / 赫克托·埃里仲杜

平凡女高中生竟然发现自己是公主，从此她的人生发生了翻天覆地的变化，从戴眼镜一头乱发的普通少女变成身穿礼服的优雅公主，反差效果不是很大，怎么整都是安妮·海瑟薇。



《穿Prada的女王》

The Devil Wears Prada

导演: 大卫·弗兰科尔

主演: 安妮·海瑟薇 / 梅丽尔·斯特里普

同样是安妮·海瑟薇，不过当初胖胖的穿着过时毛线衣的她和浑身名牌散发slay气场的她，明显不是一个人了，这部外国版的《杜拉拉升职记》给我们展示了时尚编辑的福利到底能好到什么程度，同样也在大家眼中炸开了女神的各种时装秀。



《八美图》

8 femmes

导演: 大卫·弗兰科尔

主演: 伊莎贝尔·于佩尔 / 达尼尔·达黎欧 / 凯瑟琳·德纳芙

神经质、敏感、瘦弱、不讨全家人喜欢的奥古斯汀一直嫁不出去，并且以古板、保守的老处女形象出现在大家面前，不过当她有了变美的心并实践之后，从楼梯上走下来的一幕简直美翻。于佩尔阿姨果然能够hold住所有角色。



《情癫大圣》

导演: 刘振伟

主演: 蔡卓妍 / 谢霆锋 / 陈柏霖

阿sa为演这部剧可谓是十分的自毁形象，在剧中，谢霆锋饰演的唐僧也经常吐槽她其丑无比，尽管电影拍的挺前卫的，但最后岳美艳涅槃重生如同天使一般救了心上人也足够让人老泪纵横。



《食神》

导演: 李力持 / 周星驰

主演:周星驰 / 莫文蔚 / 刘以达 / 吴孟达

莫文蔚饰演的双刀火鸡尽管长得不咋地，却对史蒂芬·周情深意重，最后为她挡了子弹却无意之中促成了自己的美貌变身，做了整形手术的她穿着时髦，更是有一双标志性的大长腿，最终和史蒂芬·周终成眷属。



《初恋这件小事儿》

A Little Thing Called Love

导演: 普特鹏·普罗萨卡·那·萨克那卡林 / 华森·波克彭

主演:马里奥·毛瑞尔 / 平采娜·乐维瑟派布恩

你永远不知道上学时你身边的柴火妞长大之后会变得多么美，曾经平平凡凡的小水为了接近男神每天敷面膜、箍牙、化妆并且提升自己的气质，最终变成了校花级人物。注意一点：小水以前本来就是五官精致，还有点刻意扮丑的嫌疑，所以长大之后可以变成白天鹅是预料之中。



《哈利波特》

Harry Potter

导演: 普特鹏·普罗萨卡·那·萨克那卡林 / 华森·波克彭

主演:丹尼尔·雷德克里夫 / 艾玛·沃森 / 鲁伯特·格林特

一直以来赫敏在我们心中是个可爱的小女孩，直到火焰杯这部。


相信当大家看到这一幕的时候，都惊呆了。



这种丑女翻身的桥段，要不是出个车祸花了几十万整个容；要不是人家天生丽质难自弃，看起来丑只是因为不会打扮；要不就是根植神话背景用一颗善良的心换得美丽的皮，这三点哪一点都不容易达到，所以大家还是，看看就好了。
