各位僵尸粉们大家好，又到了一年一度的高考了。


作为一个过来人，鲶鱼君可是对自己的高考记忆犹新，作为一个从军事化管理学校出来的学生，考完的第一件事儿肯定是疯玩儿了！当然在肢体运动疲乏的时候，我们也可以看个电影来感受下心灵的宁静。


今天鲶鱼君的片单涵盖了大家高考前后的各种心境，虽然我知道看这个的大多数是高考完四五年的单身狗们，但是大家还是一起怀念下曾经的热血，和憧憬中的未来吧。



高三

2005

导演: 周浩

类型: 纪录片

作为一个出身自类似衡水中学学校的人来说，看这部纪录片会回忆很多。


如果说我们见到的青春电影用各种美妙的滤镜将青春期叙述得太过偶像化，那么《高三》这部电影就真实记录了大家在高三备战时的灰头土脸，堆积如山的试卷和参考书，呆板的中国校服，上厕所争分夺秒，在被窝里用手电看书……

这部电影会带你回到那忙碌的一年，然后你会在心中默默给自己点个赞，原来曾经这么能吃苦！



垫底辣妹

2015

导演: 土井裕泰
主演: 有村架纯 / 伊藤淳史 / 吉田羊 / 田中哲司

类型: 剧情 / 家庭

又是一部打鸡血的片子，其实挺适合高考之前看的，不过我这种工作党看的时候仍然被感动得热泪盈眶，剥离了考试这个主题，其实也挺适合日常打鸡血，学英语，学日语啥的。


一部“有志者事竟成”的优质电影，发挥很稳，让人哭得稀里哗啦的。




心灵捕手

1997

导演: 霍拉蒂乌·马拉埃雷
主演: 马特·达蒙 / 罗宾·威廉姆斯 / 本·阿弗莱克/ 斯特兰·斯卡斯加德

类型: 剧情

一部非常经典的电影，编剧就是马特·达蒙和他的基友本·阿弗莱克。


一位天赋异禀的清洁工，他打架斗殴放荡不羁，数学却比麻省理工的学生还要牛逼，这部电影讲述了这位天才逐渐破除心灵魔障爱情友情双丰收的故事，同样他和桑恩教授伟大的友谊也让人心动。

《心灵捕手》给我们奉献了无数语文作文可以引用的金句，不过现在说，也来不及了。（鲶鱼君敲打键盘的时候你们正在考语文）



歌舞青春

2006

导演: 肯尼·奥特加

主演: 扎克·埃夫隆 / 凡妮莎·哈金斯 / 阿什丽·提斯代尔 / 卢卡斯·格拉比

类型: 剧情 / 喜剧 / 家庭

大学是什么样子呢？你肯定在想，《歌舞青春》让你感受张扬放肆的青春，顺便回答下汪峰导师提出的问题：你的梦想是什么？！



美丽心灵

2001

导演: 朗·霍华德

主演: 罗素·克劳 / 詹妮弗·康纳利 / 艾德·哈里斯/ 克里斯托弗·普卢默 / 保罗·贝坦尼

类型: 剧情 / 传记

很多人高考之后（尤其是进入大学之后）会荒废学业，沉浸在来之不易的自由之中，这也就导致大多数人的大学生活变成了吃饭喝酒搞对象，而《美丽心灵》给我们的启发在于，对于自身爱好的专注和坚持不懈。



致青春原来你还在这里

2016

导演: 周拓如

主演: 吴亦凡 / 刘亦菲 / 金世佳 / 李沁 / 李梦

类型: 剧情 / 爱情

没事，就是让大家看看大学生活不是这样。



洛奇

1976

导演: 约翰·G·艾维尔森

主演: 西尔维斯特·史泰龙 / 塔莉娅·夏尔 / 波特·杨 / 卡尔·韦瑟斯 / 布吉斯·梅迪斯

类型: 剧情 / 爱情 / 运动

一位业余拳击手过着咸鱼一样的生活，然而当他坠入情网之后他决定变成一条有梦想的咸鱼，他明知道自己没有办法击败曾经的世界冠军，却仍然用心准备比赛，每天刻苦训练，最终他在和世界冠军过招的时候撑了15回合，尽管并没有俗套励志电影中咸鱼翻身成为冠军的桥段，但在洛奇心中，自己已经成为了自己的英雄。

当考试成绩出来之后，肯定是有人欢喜有人愁，成绩不理想的小伙伴不要沮丧，只要你成为了最好的自己，就没有遗憾。



百元之恋

2014

导演: 武正晴

主演: 安藤樱 / 新井浩文 / 稻川实代子 / 早織 

类型: 剧情

曾经没有工作天天吃膨化食品的废柴，突然想改变自己走出家门工作、学习拳击，最终没有像《摔跤吧！爸爸》那样扬名立万，也没有收获美满的爱情，现实毕竟是现实，很多事情没办法通过个人的奋斗去完成，也没必要用传统意义的成功去框定你曾经努力的过程。


你的改变只需要你自己知道。



奋斗

2007

导演: 赵宝刚 / 王迎

主演: 马伊琍 / 佟大为 / 文章 / 李小璐 / 朱雨辰

类型: 喜剧 / 爱情


欢乐颂

2016

第一季导演: 孔笙 / 简川訸

第二季导演: 简川訸 / 张开宙

主演: 刘涛 / 蒋欣 / 王子文 / 杨紫 / 乔欣 / 祖峰/ 王凯 / 张陆 / 靳东 / 杨烁

类型: 剧情 / 喜剧

成绩不理想的话呢，除了转回头重新过一遍《高三》之外，还能在电视剧《奋斗》、《欢乐颂》中得到启发。


如果你爹是《奋斗》中佟大为的那种爹，或者是《欢乐颂》中曲筱绡的那种爹。


一切也都是so easy了。
