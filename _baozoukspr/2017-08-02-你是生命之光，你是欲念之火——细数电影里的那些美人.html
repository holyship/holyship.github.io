最近大家都在看《战狼2》吧，眼袋君看完《战狼》，又去刷了很多那种热血沸腾的直男电影，感觉肾上腺素有点爆表。今天就为大家平衡一下荷尔蒙，安利几部非常养眼的电影，这些电影不光有一些羞羞的镜头，还有很多美得肝颤儿的美人儿，无论是男人还是女人，看了都会由衷的赞叹：好美啊！


用《洛丽塔》中的一句话来说就是：“她可以褪色，可以枯萎，怎样都可以。但我只望她一眼，万般柔情，便涌上心头。”



《洛丽塔》97版（又名《一树梨花压海棠》）

导演: 阿德里安·莱恩
主演: 杰瑞米·艾恩斯 / 梅兰尼·格里菲斯 / 弗兰克·兰格拉


Lolita实在美的让人窒息，天真中带着一丝邪恶。




《洛丽塔》62版

导演: 斯坦利·库布里克
主演: 詹姆斯·梅森 / 谢利·温特斯 / 苏·莱恩


这版的洛丽塔也超美的！





《情人》

 (1992)

导演: 让-雅克·阿诺
主演: 珍·玛奇 / 梁家辉


湿漉漉的湄公河、沉默而又迷人的少女，还有深情地梁家辉，珍·玛琪单薄而又性感的背影和杜拉斯的原著相当符合，整部电影都透着一种细腻的性感。





《巴黎野玫瑰》

(1986)

导演: 让-雅克·贝奈克斯

主演: 让·雨果·安格拉德 / 碧翠斯·黛尔 / 热拉尔·达尔蒙


女主角贝蒂长得不算传统意义上的美人儿，但是浑身透着的那种疯狂和歇斯底劲儿里倒也有几分灼热的性感。





《戏梦巴黎》

(2003)

导演: 贝纳尔多·贝托鲁奇
主演: 迈克尔·皮特 / 伊娃·格林 / 路易·加雷尔


不画眼线的伊娃·格林，眉眼依旧透着邪恶的诱惑。




《本能》

 (1992)

导演: 保罗·范霍文

影史最风骚的镜头之一，没穿内裤的莎朗·斯通性感度爆表。




《玛戈皇后》

 (1994)

导演: 帕特里斯·谢罗

实在难以用语言来描述……伊莎贝尔·阿佳妮美的令人肝儿颤。




《原罪》

(2001)

(2000)

(1995)

导演: 米开朗基罗·安东尼奥尼 / 维姆·文德斯

《阿黛尔的生活》

 (2013)

导演: 阿布戴·柯西胥

《女性瘾者》

（2013）

导演: 拉斯·冯·提尔
主演: 夏洛特·甘斯布 / 斯特兰·斯卡斯加德 / 斯塔西·马汀
