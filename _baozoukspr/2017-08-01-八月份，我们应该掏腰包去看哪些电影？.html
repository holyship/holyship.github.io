本月继续是国产电影那啥月，在这个那啥月里，我们创造了《战狼2》几天破十亿的佳绩，那么在八月份，我们应该掏腰包去看哪些电影呢？



三生三世十里桃花

导演: 赵小丁 / 安东尼·拉默里纳拉
主演: 刘亦菲 / 杨洋 / 罗晋 / 严屹宽 / 李纯


上映时间：8月3日

推荐度：★★☆☆☆


当红鲜肉杨洋以及小花刘亦菲无疑会给这部电影带来巨大的流量，所以说票房什么的，只要请对了人，就会有一大批的粉丝抢着去买账。


导演赵小丁，曾经老谋子御用摄影师，宏大的场面能hold住，仙侠场面也不用担心。


不过两位主演的演绎鲶鱼君比较担心，看预告片，感觉杨洋同学仍然是一脸沉浸在自己美貌中的表情。


美貌而自知并且灵动演绎出来，或美貌而不自知让观众挖掘其一面惊鸿，都挺好，最怕的是自诩美貌而钳制了自身的表情，那，大概可以叫油腻了吧。


这片的原著，三流小说的水平，至于改编成电影的水平，并不是很期待。



皮绳上的魂

导演: 张杨
编剧: 张杨 / 扎西达娃
主演: 金巴 / 曲尼次仁


上映时间：8月4日

推荐度：★★★★☆


导演张杨的《冈仁波齐》，内核故事以及表现方式，都不是叫座的类型，但却卖出了将近一亿，鲶鱼君在疑惑的同时还是觉得开心的，毕竟这代表着更多人去欣赏这种小众电影，也身体力行的支持了独立片导演。



《皮绳上的魂》同样是讲述西藏，讲述宗教和禅意，豆瓣评分6.8，风景同样出色，希望张杨能给我们带来更多的惊喜吧。



龙之战

导演: 高峰
主演: 刘佩琦 / 曹云金


上映时间：8月4日

推荐度：★★☆☆☆


冯子材抗法的故事，从预告片和剧照都透出浓浓的CCTV-6的风格，内核是家国大义，场景比较朴实，导演高峰拍的片大多数是这种风格，比较“体制内”，去不去电影院看，看各位喜欢什么风格了。



我是马布里

导演: 杨子
主演: 斯蒂芬·马布里 / 吴尊 / 何冰


上映时间：8月4日

推荐度：★★☆☆☆


与上面那个《龙之战》差不多，CCTV-6风，体育励志电影大家应该也看了不少，套路基本上摸个门儿清了，啥时候崛起啥时候失败受挫啥时候咸鱼翻身都能想个差不多，这片基本上大概就是这种吧，看了没啥惊喜，催泪一把应该还可以。



心理罪

导演: 谢东燊
编剧: 顾小白
主演: 廖凡 / 李易峰 / 万茜 / 李纯


上映时间：8月11日

推荐度：★★☆☆☆


看了预告片，感觉廖凡和李易峰不是一个画风不是一个世界的，尤其是当廖凡叔恶狠狠的说话的时候，峰峰仍然不为所动，让我想起被《麻雀》支配的恐惧。


导演谢东燊十年前拍过些好片，但近几年拍的电影是在是有点不太行。



万茜、廖凡这个三带一阵容，是否真的能让整部电影的演技不垮，就要看天命了。



侠盗联盟

导演: 冯德伦
主演: 刘德华 / 舒淇 / 张静初 / 杨祐宁 / 让·雷诺


上映时间：8月11日

推荐度：★★☆☆☆


冯德伦当了导演之后，拍的电影都不太行，这次又当导演拍电影了，演员阵容挺强大，刘德华舒淇还有魅力大叔让雷诺。


剧情是一个宝物当做麦高芬，然后一堆人争夺来争夺去，最后引发惊天大秘密，是不是有一种《富春山居图》以及《抢红》的feel？


不太期待。



鲛珠传

导演: 杨磊
编剧: 张炭 / 徐照青 / 杨磊
主演: 王大陆 / 张天爱 / 任达华


上映时间：8月11日

推荐度：★★☆☆☆


这片，有一种导演杨磊拖家带口宣传了很久的感觉，前一阵子的《逆袭之星途璀璨》网剧愣是给这片做了不少广告。


杨磊导演，鲶鱼君很喜欢，人挺实诚，也很接地气，拍的《红色》也是佳作，不过这部电影的编剧，曾经编出过《无极》、《封神传奇》等具有划时代意义的伟大作品，实在是让大家对《鲛珠传》望而生畏。


希望杨磊导演赚够了钱能够静下心来磨剧本好好拍戏，我看好你哟。



二十二

导演: 郭柯
类型: 纪录片


上映时间：8月14日

推荐度：★★★★★


这部纪录片，鲶鱼君已经看很多人推荐过了。


抗日八年中的慰安妇们，她们现在的生活是什么样的呢？摄制组历时四年，横跨5个省，29个地区，走访22位慰安妇，将逐渐消逝的记忆记录下来。



现在有人能够花时间去用心做这样一件事，仿佛在与飞奔的时间作斗争，希望能在院线看到更多这样的电影。



十万个冷笑话2

导演: 卢恒宇 / 李姝洁
编剧: 卢恒宇 / 李姝洁 / 陈学良 / 隋佳林 / 高信影 / 赵聪


上映时间：8月18日

推荐度：★★☆☆☆


14年的十冷大电影评分6.3，表现比较一般，这次的十冷2，看预告片，如同网友说的那样，这个配置活脱脱是个银河护卫队，雅典娜、雷神、河神还有个小金刚，吐槽的点非常网络，也夹杂着不少网络过时的段子，让人看得有点尴尬，可能因为十冷1上映时正是吐槽风兴盛的时刻，而现在，吐槽这个元素已经被大家广泛的玩儿坏了。


大概受众是十几岁二次元小可爱吧，我这种快三十的老阿姨get不太到这个点了。



破·局

导演: 连奕琦
主演: 郭富城 / 王千源 / 刘涛


上映时间：8月18日

推荐度：★★☆☆☆


不知道谁起的头，现在电影片名利都加个标点符号来表现高端。


这片剧情很像韩国电影《走到尽头》，不知道是翻拍还是咋，导演连奕琦拍出过几部还不错的片子，主演挺厉害，郭富城王千源都是遇见好剧本很有发挥的演员，希望质量能不错吧，不然可以叫破剧了。



星际特工：千星之城

导演: 吕克·贝松
主演: 戴恩·德哈恩 / 卡拉·迪瓦伊 / 克里夫·欧文/ 蕾哈娜 / 伊桑·霍克 / 赫比·汉考克 / 吴亦凡


上映时间：8月25日

推荐度：★★★☆☆


这片烂番茄新鲜度51%，IMDb评分6.8，口碑比较一般，外媒评价表示节奏比较拖沓并且没有给人完整的电影体验，而喜欢的人表示这部电影的想象力和视觉效果非常不错。应该可以去电影院支持下。


PS：不知道女主角的演技现在好点没。



极道车神

导演: 埃德加·赖特
编剧: 埃德加·赖特
主演: 安塞尔·艾尔高特 / 凯文·史派西 / 杰米·福克斯 / 乔·博恩瑟


上映时间：8月25日

推荐度：★★★★★


这片很厉害！烂番茄新鲜指数95%，IMDb评分8.2。



埃德加·莱特是一个非常有风格的导演，看过的朋友们纷纷说飙车戏燃爆全场，音乐也十足有腔调，观感非常棒，五星推荐不看后悔！




赛车总动员3

导演: 布莱恩·菲
主演: 欧文·威尔逊 / 克里斯黛拉·阿朗索 / 克里斯·库珀


上映时间：8月25日

推荐度：★★★☆☆


属于打卡片，看过前两部的观众们应该都会去看，只要整体剧情没有太大的硬伤，大家还是会拖家带口以及熊孩子去看的。
