前几天，眼袋君为了提高自己的姿势水平和文化修养，某宝下单了诺奖得住鲍勃·迪伦文集一套。这文集包装上很下功夫，和别的平常文集一点都不一样，薯片充气包装的外表一看就像鲍勃·迪伦本人，散发着独一份的朋克味儿（而且很有食欲！）。


吃着与诗同行的薯片听着鲍勃·迪伦的民谣读着鲍勃·迪伦的诗集，想一想就美滋滋。然而眼袋君还是naive，撕开这充气的薯片包装，里面根本没有薯片。


？？？！！


你们来评评理，薯片充气包装里没有薯片，这像话吗？还有没有基本法！


伟大的诗人、作曲者，鲍勃·迪伦


诗意薯片是没得吃了，可鲍勃的诗却是上好的精神食粮，这位首位得诺奖的作曲家，唱民谣玩摇滚是一代宗师，重温他的的作品，不禁让眼袋君回想起了曾经迷恋摇滚的少年岁月。


青年时期的鲍勃·迪伦


今天跟你们聊聊关于摇滚的电影，包括知名摇滚乐手或者乐队的传记片，和其他关于摇滚乐的故事。



《摇滚校园》

小孩子玩摇滚



2003

导演: 理查德·林克莱特
主演: 杰克·布莱克 / 琼·库萨克 / 麦克·怀特 / 萨拉·西尔弗曼


摇滚音乐人杜威冒名顶替别人进了一所校风极为严谨的名门中学教学。学生们的死气沉沉一度让他倍感无聊，某天，杜威无意中发现孩子们的音乐天赋。于是，他瞒天过海，带领这群只知道读书的孩子们进入一个全新且乐趣无穷的摇滚世界……



同影片中的主演一样，演员杰克·布莱克在真实生活中也是一位狂热的摇滚乐迷，他甚至有着自己的摇滚乐队，所以他把片中主角，也就是那位冒牌老师对摇滚的那份执着和热情诠释的很到位。


王尼玛告诉我们：小孩子不要看暴漫；杰克·布莱克告诉我们：小孩子可以玩玩摇滚！



《大门》

大门乐队



1991

导演: 奥利佛·斯通
主演: 方·基默 / 梅格·瑞恩 / 凯瑟琳·奎南 / 凯尔·麦克拉克伦 / 迈克尔·马德森


本片记述了1960年代美国迷幻摇滚代表乐队——大门乐队的灵魂主唱吉姆·莫里森从成名到自我毁灭的一生。



这部电影充满迷幻的氛围，将那个年代的狂热和躁动，吉姆的不羁诗人气质都淋漓尽致的展现了。



《黄色潜水艇》

披头士乐队



1968

导演: 乔治·杜宁
主演: The Beatles / 保罗·安吉利斯 / 约翰·克莱夫 / Dick Emery


本片是一部披头士乐队的音乐和波普视觉艺术结合的，二维插画风格的动画片，整部片子中共穿插了披头士的15首经典歌曲。



影片讲述了披头士成员保罗·麦卡特尼在半梦半醒之间来到花椒国，这是一个讨厌音乐的地方，正遭遇蓝色恶魔的侵略，老船长弗莱德召集披头士成员一同乘坐黄色潜水艇共同保卫花椒国。 



《迷墙》

平克·弗洛伊德乐队



1982

导演: 艾伦·帕克

主演: 鲍勃·吉尔道夫 / Christine Hargreaves / James Laurenson / 埃里诺·大卫


平克·弗洛伊德乐队（Pink Floyd）元老成员、主唱兼吉他手西德·巴雷特的梦幻心理成长史。



影片中充满深刻的隐喻直到现在都令人深思，就是这部电影让很多人尊重摇滚，提升了它的意义和价值。


另：这部电影算是一个加长版的平克·弗洛伊德乐队MTV，呃，据说这部电影也是最早期的MTV雏形。



《天鹅绒金矿》

华丽摇滚



1998

导演: 托德·海因斯
主演: 伊万·麦克格雷格 / 乔纳森·莱斯·梅耶斯 /克里斯蒂安·贝尔 / 托妮·科莱特 / 艾迪·伊扎德


摇滚史上最华丽的70年代，同性恋，华丽的装扮和舞台。原声和演员都很棒，两个男主角貌似是在影射同时期的David Bowie和Iggy Pop。




《我不在那儿》

鲍勃·迪伦



2007

导演: 托德·海因斯
主演: 凯特·布兰切特 / 本·卫肖 / 克里斯蒂安·贝尔 / 理查·基尔 / 希斯·莱杰 / 朱丽安·摩尔


这是一部关于鲍勃老爹生平的传记电影，由六位演员扮演鲍勃·迪伦，分别演绎他在不同时代的生活故事和音乐经历。


片中最牛逼的地方是凯特·布兰切特反串出演鲍勃·迪伦，简直精彩到无可挑剔。




《最后的日子》

柯特·科本



2005

导演: 格斯·范·桑特
主演: 迈克尔·皮特 / 卢卡斯·哈斯 / 阿西娅·阿金托


本片根据涅磐乐队（Niravana）主唱柯特·科本（Kurt Cobain）生前最后的一段经历改编，试图还原柯本在自杀前夕的心路历程。主演迈克尔·皮特对柯本的形象还原度极高。




《海盗电台》

摇滚乌托邦



2009

导演: 理查德·柯蒂斯
主演: 比尔·奈伊 / 肯尼思·布拉纳 / 菲利普·塞默·霍夫曼 / 尼克·弗罗斯特


这是一个给人们梦想和欢乐的电台，这是一个乌托邦的电台，这里有爱，有梦，更多的是音乐带给人们的欢乐。



影片可爱，搞笑和人性化的再现了60年代的那场关于爱与和平的小革命，励志而且感人。



《摇滚芭比》

变性少年的摇滚梦



2001

导演: 约翰·卡梅隆·米切尔
主演: 约翰·卡梅隆·米切尔 / 米利亚姆·肖尔 / 安德列·马丁


主人公为了去美国完成自己的摇滚梦，而进行了变性手术……还有比这更摇滚的事情吗？




《制造伍德斯托克音乐节》

五十万嬉皮的狂欢



2009

导演: 李安
主演: 亨利·古德曼 / 爱德华·希伯特 / 艾美达·斯丹顿 / 迪米特利·马丁 / 凯文·张伯伦


相信稍微接触点摇滚乐的人都对60年代的那场著名的音乐节并不陌生，影片改编自真人真事，描述60年代美国同志作家兼设计师埃利奥特·提伯创办1969年伍德斯托克音乐节的经过。



不知道有多少人遗憾自己生错了时间，生错了地方，错过了1969年纽约附近的山坡上，那一场五十万嬉皮的狂欢。



《北京乐与路》

 北漂的摇滚路



2001

导演: 张婉婷
主演: 耿乐 / 舒淇 / 吴彦祖 / 吴耀汉 / 俞飞鸿


中国的摇滚电影，不多见。记得吴彦祖饰演的香港音乐人在片中说了一句：香港没有摇滚。（片中台词，不是我说的！）




《席德与南茜》

性手枪乐队



1986

导演: 亚力克斯·考克斯
主演: 加里·奥德曼 / 科洛·韦伯


本片根据性手枪乐队的贝斯手希德·维舍斯的事迹改编。希德·维舍斯应该是历史上最著名的朋克了，他的事迹基本代表了朋克精神的本质和美学标准——无政府主义、极端暴力、质疑一切、仇恨社会、毒品和性、危险的浪漫，以及自毁的艺术。磕药过量要了他的命，他甚至还莫名其妙地杀了女友南希（这件事一直有争议）。



所以说，好孩子玩玩摇滚可以，但是千万不要试图去做一个朋克。
