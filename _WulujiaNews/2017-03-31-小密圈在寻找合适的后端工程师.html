小密圈最近流量蹭蹭地涨，后端压力也不断增大，需要愿意一块成长的小伙伴来一起折腾。后端研发工程师主要对小密圈 API 等后端组件进行开发。

简历请在标题注明「后端工程师」，发送到：wulujia#gmail.com。

要求：

  1. 精通 PHP，熟悉至少一种 MVC 框架（如 Phalcon/Yaf 等）。熟练使用 Phalcon 者优先。

  2. 熟悉 Linux、MySQL、Redis、Nginx、Git。

  3. 具有 MySQL 索引优化、查询优化、存储优化的经验。

  4. 熟悉 PHP 的缓存技术、静态化设计。

  5. 熟练阅读英文技术文档。

  6. 优先考虑：

    • CET-4 或 CET-6 425 分及以上。

    • 熟悉 Swoole。

    • 熟悉 JavaScript、jQuery、CSS3 等前端技术。

    • 熟悉 Golang、Python。

    • 熟练使用 Docker。

欢迎有个人作品的朋友（包括应届毕业生）投简历。
