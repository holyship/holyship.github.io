神经网络入门

饿了么的 PWA 升级实践

PWA 作为下一代 Web 应用模型，其尝试解决的是 Web 平台本身的根本性问题：对网络与浏览器 UI 的硬依赖。任何 Web 应用都可以从中获益，这与你是多页还是单页、面向桌面还是移动端、是用 React 还是 Vue 无关。
