丁香园杭州办公室环境

Java高级架构师

工作职责

  1. 参与丁香园统一的后端技术架构规划和设计；

  2. 负责公司核心业务系统开发和维护；

  3. 预研新技术，做公司内部的技术培训和推广工作；

  4. 参与公司产品项目的架构评审和代码开发工作。

职位要求

  1. 5 年以上Java服务端开发经验，有系统架构设计经验，设计过微服务架构者优先；

  2. 从事过大规模 Web 应用开发，熟悉代码重构，性能优化，系统安全；

  3. 精通Spring/Hibernate/iBatis等开源框架，在Github上有开源项目者优先；

  4. 有过中型以上的互联网公司主持架构设计和实施的工作经验优先。

搜索系统研发工程师

工作职责

  1. 参与丁香园搜索平台的架构设计及研发，保证系统稳定；

  2. 开发和优化搜索相关的 AB 测试功能，方便对搜索的性能和效果进行持续优化；

  3. 参与具体业务需求的对接，并实现业务相关的搜索开发工作。

职位要求

  1. 计算机相关专业，2 年以上搜索相关研发经验；

  2. 对搜索引擎相关体系架构与算法有比较深入的了解，熟悉全文检索、中文分词、排序及查询分析等相关技术；

  3. 熟悉 Lucene（ElasticSearch/solr），具备一定的定制和维护能力；

  4. 熟练使用 Java/Python 编程语言，对 JVM 原理有一定了解，熟悉Linux；

  5. 具有优秀架构能力者优先，有大型搜索研发经验优先。

大数据运维工程师

工作职责

  1. 负责和参与公司大数据基础架构平台的运维，保障数据平台服务的稳定性和可用性；

  2. 负责和参与自动化运维系统及平台的建设；

  3. 负责优化部门运维流程提升运维效率。

职位要求

  1. 一年以上大数据平台运维经验；

  2. 掌握 Hadoop, Hive/Impala, Spark, Yarn, Kafka, Flume 等组件的安装、配置、使用和优化，熟悉 CDH 者优先；

  3. 掌握 Linux 操作系统的配置，管理及优化，能够独立排查及解决操作系统层的各类问题；

  4. 精通 Python, Shell，能够开发运维自动化组件和工具；

  5. 熟悉 MySQL，熟练掌握SQL。

工作地点：杭州滨江
