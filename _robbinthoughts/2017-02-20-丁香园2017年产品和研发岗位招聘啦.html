2017年是丁香园在互联网产品、研发、和医疗大数据平台全线发力的一年。在过去的半年，我们的产品战略规划已经清晰，研发体系逐步完善，医疗大数据平台搭建成型；而在2017年，我们饥渴的需要更多优秀人才加入我们，在互联网医疗领域加速奔跑，向一流互联网公司的目标前进。

我们能给你什么？

  • 弹性工作时间，不管你是早睡早起还是夜猫子，都能在这里精力充沛地工作

  • 舒适的办公环境，健身房、咖啡吧、练歌房、足球场、冷饮

  • 双休制、五险一金、法定假期、年假……当然都是有的

  • 最重要的是充分的成长空间和锻炼机会，只要你敢想，就放手让你去做


丁香园杭州办公室环境

高级Android研发工程师

工作职责

  1. 主导开发具有良好的性能、良好的用户体验的 Android 应用程序；

  2. 负责Android应用架构的设计和规范化工作；

  3. 负责Android工程师的培训，能够指导Android工程师代码开发；

职位要求

  1. 5年以上Android应用相关的开发经验，具有优秀的编程能力；

  2. 熟悉移动客户端内存管理，线程管理，Crash分析，以及开源框架的使用和优化；

  3. 基础扎实，精通常用数据结构与算法，熟悉常用的设计模式； 

  4. 有很强的责任心和团队精神，有带领研发团队经验的优先。

高级Java工程师/架构师

工作职责

  1. 参与丁香园统一的后端技术架构规划和设计；

  2. 负责公司核心业务系统开发和维护；

  3. 预研新技术，做公司内部的技术培训和推广工作；

  4. 参与公司产品项目的架构评审和代码开发工作。

职位要求

  1. 5 年以上Java服务端开发经验，有系统架构设计经验；

  2. 从事过大规模 Web 应用开发，熟悉代码重构，性能优化，系统安全；

  3. 精通Spring/Hibernate/iBatis等开源框架，在Github上有开源项目者优先；

  4. 有过中型以上的互联网公司主持架构设计和实施的工作经验优先。

大数据研发工程师

工作职责

  1. 参与公司大数据系统平台与应用的设计、开发、维护；

  2. 参与构建日志收集、数据仓库、ETL、调度系统、分析系统等环节；

  3. 参与机器学习相关的研发工作，我们有高质量数据可供挖掘。

职位要求

  1. 2 年以上 Hadoop 相关工具使用经验；

  2. 至少精通Java/Scala/Python编程语言之一，具备熟练的编程能力；

  3. 熟悉常用的数据指标、数据分析工具和平台（Hadoop、Hive、Spark、Kafka 等），了解数据分析的基本方法和常见问题；

  4. 做过机器学习/NLP相关的优先。

搜索/推荐研发工程师

工作职责

  1. 参与丁香园搜索/推荐系统的开发工作；

  2. 负责医疗产品线的垂直领域搜索和精准推荐业务相关的研发。

职位要求

  1. 至少精通Java/Scala/Python编程语言之一，具备熟练的编程能力；；

  2. 熟悉 SolrCloud, Lucene 开源搜索引擎及方案，有过二次排序算法开发经验；

  3. 熟悉 ElasticSearch及其生态可加分；

  4. 熟悉流式计算相关编程可加分（如Spark Streaming, Storm等）。

高级前端工程师

工作职责

  1. 依据产品需求完成高质量的Web前端开发和维护；

  2. 对具体的产品进行性能优化，实现极致的Web加载、执行和渲染时间；

  3. 在理解前端开发流程的基础上，结合前端实践建立或优化提升工作效率的工具；

  4. 在理解产品业务的基础上，结合前端技术建立或优化Web的交互方式。 

职位要求

  1. 具备5年以上互联网web前端研发经验；

  2. 熟练使用如下框架中的两种或以上：jQuery， React，React Native， AngularJS，Backbone，VUE；

  3. 有基于Hybrid 或者 React Native开发经验优先考虑。

移动产品经理

工作职责

  1. 负责医药领域移动客户端产品的产品设计和产品运营工作；

  2. 线上产品数据分析、手机产品的用户研究，针对数据指导优化，持续改进产品；

  3. 进行竞品分析、信息整理，提出产品优化方案。

职位要求

  1. 1年以上产品经理经验，1年以上独立负责无线产品的经验；

  2. 熟悉手机软件产品，对手机应用层面有深入的了解，熟悉各种智能手机操作系统；

  3. 有从需求分析到产品发布全流程的经验，能独立设计产品。

Web产品经理

工作职责

  1. 负责丁香园相关 Web 产品线整体规划及产品设计；

  2. 负责收集和分析业务部门的需求优先级，协调项目组，把控项目进度；

  3. 参与和配合相应产品线运营策划及落地实施。

职位要求

  1. 一年以上 Web 产品经理（或产品运营）经验，产品基本功扎实，能高效描述用户流程和产品逻辑；

  2. 具有良好的项目管理经验，能够有条不紊的推进项目进度；

  3. 具有良好的沟通能力，善于跨团队和跨部门协调，能够有效整合资源。

工作地点：杭州滨江
