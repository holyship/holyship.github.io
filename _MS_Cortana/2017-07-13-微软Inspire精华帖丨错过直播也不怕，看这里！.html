2017微软全球合作伙伴大会 精华帖

Microsoft 365


2017微软Inspire全球合作伙伴大会上，微软首席执行官萨提亚·纳德拉发布了Microsoft 365，它包含了Office 365、Windows 10和企业移动性+安全性（Enterprise Mobility +Security）套件，为企业用户提供一套完整且智能的安全解决方案，它必将帮助我们的客户和合作伙伴取得更快速的发展，实现更伟大的创新。

微软首席执行官萨提亚·纳德拉


Microsoft 365企业版

  • 通过统一管理用户、设备、应用和服务的方式来简化IT。

  • 通过内置的智能安全性来保护客户/公司信息和知识财产。

*Microsoft 365企业版提供两种选项，即Microsoft 365 E3和Microsoft 365 E5，并可于2017年8月1日开始订购。


Microsoft 365商业版

  • 凭借不间断的安全保护，实现公司数据在所有设备上的安全性。

  • 凭借统一IT控制台来简化员工设备和服务的设置和管理。

*Microsoft365商业版的公众预览版将于8月2日发布，并将于2017年秋季面向全球正式发售，价格为每用户每月20美元。


嵌入式模块，业务应用


Microsoft Azure一直受到独立软件开发商（ISV）的青睐，目前已有超过35,000 个合作伙伴在利用Azure的IaaS和PaaS服务来开发自己的应用，而且这些合作伙伴希望我们以嵌入式模块的形式提供更高级的云服务。

微软云


为此我们推出了名为ISV Cloud Embed的新计划，让ISV能够更轻松地对现有业务应用进行现代化升级，或者是开发全新应用。


  • Dynamics 365 Embedded —— 无需从头开发，就能为特定行业或区域快速定制业务应用。

  • Microsoft Flow Embedded —— ISV将得以让高级用户和信息工作人员添加自定义工作流，以便将其作为应用的一部分，自动处理业务流程。

  • PowerApps Embedded——为应用提供了轻松扩展模型，能够在浏览器或移动应用中交付。

  • Power BI Embedded——用户可以轻松嵌入完全交互式的报告和仪表板。


合作伙伴为先


将成立微软统一商业合作伙伴部门（One Commercial Partner），它将充分利用我们的专业技术和知识——包括在技术、营销、商务开发和计划等方面的积累，并整合一切有用的资源，让每一个合作伙伴——不论其规模大小、采用什么商业模式或身处何处，都能从中受益。

微软公司副总裁Ron Huddleston


 共同开发（Build-with）

不论合作伙伴是需要建立业务、构建知识产权还是打造自己的技术能力，我们的合作伙伴开发团队的专家都将提供专业支持，他们具备商业模式专业知识，并拥有众多技术和专家资源，他们的使命就是在合适的时间为合作伙伴提供合适的资源，以支持合作伙伴的成功和业务成长。 


营销（Go-To-Market）

我们的合作伙伴营销团队的专家将帮助合作伙伴通过营销措施把解决方案推向市场，他们将与其它两个团队相互配合，包括与合作伙伴共同开发市场机会的团队，以及与合作伙伴共同销售以满足客户需求的团队。


共同销售（Sell-with）

我们为合作伙伴业务增加了全新职位，渠道经理将负责在合适的时间把合适的合作伙伴解决方案推荐给合适的客户，配备最佳实践和解决方案路线图的他们将与微软销售团队以及各行各业各种规模的客户携手合作。
