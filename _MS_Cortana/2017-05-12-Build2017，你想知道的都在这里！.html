2017年5月10日至12日在西雅图举行Build 2017大会已经落下帷幕，大会期间小娜摇身变主播，出道首秀就是为大家直播Build 2017，小娜主播的表现大家还满意么？~

 

今天，软妹子亲自搬砖，带大家回顾Build 2017！~

 

微软“移动为先，云为先”的理念创造了庞大的生态体系，在这个庞大的生态体系下，可以预见：未来将会是是智能云（Intelligent Cloud）与智能边缘（Intelligent Edge）的世界，它会改变所有的事，而技术发展的新方向势必会是多设备、跨平台、人工智能、边缘计算与智能云相互交织而成。


▼
Azure IoT Edge

这项技术能够将云计算的智能和更多优势扩展到物联网边缘设备上，它可以在本地计算设备上进行计算，节省时间，它同时也支持在 Linux 和 Windows 上跨平台运行。

▼
Azure Cosmos DB

它是“首个全球分布的多模型数据库服务，支持一站式横向扩展，同时99%保证运行时间和毫秒之内的延迟”，具有：高可用性、低性能延迟、高吞吐量以及数据一致性这几个特性。

▼
Visual Studio for Mac

MacOS版的Visual Studio正式发布，这一举动意味着微软对开发者的全方位支持，不论其码代码用的什么平台。

▼
Cortana

沈向洋博士在现场宣布：继与哈曼卡顿合作Cortana智能音响之后，英特尔和惠普两家公司正式和微软小娜达成合作关系，未来将基于小娜生产更多的智能硬件产品，让小娜成为用户跨平台的人工智能助手，不论在 PC 、手机、智能家居、甚至车载系统，小娜可以出现在用户需要的任何地方，为全球一亿多小娜用户提供更便捷的生活服务。

同时，推出Cortana Skills Kit支持开发人员为小娜（Cortana）智能助手创建语音应用，这套工具支持跨平台跨设备，未来小娜（Cortana）将作为微软融合度最高的AI产品，向更多的领域和场景整合渗透。

▼
AI领域新动向

沈向洋博士表示：“人工智能需要强大的计算能力，算法和大量的数据，而微软的云平台、一流的研究院以及在办公软件方面的积累，都为大家提供了上面的三个条件。”沈向洋博士特别提到了一个中国企业Airdoc的例子，这家公司主要借助医疗领域人工智能，辅助分析医学影像，微软则为其提供了认知工具开发包。另外，他还展示了一个实时PPT翻译功能，它可以自动实时翻译PPT中的文本，同时在协作对话中以字幕的形式翻译语音内容。

▼
Xamarin Live Player

发布新工具Xamarin Live Player （即刻的编程环境），也就是开发者无需安装SDK或模拟器，即可创建、测试应用以及排除bug。可以安装Visual Studio 2017和Visual Studio 2017 for Mac的Live Player扩展包，也可以从Google Play和苹果应用商店中下载Xamarin Live Player。

▼
混合现实

发布首款Windows混合现实运动控制器，与头显配合可运用于开发创意工具、提高工作效率，打游戏以及进行娱乐活动。

全球首款完全无需依赖标签的运动控制器。依赖于混合现实头戴设备的内置传感器，就能对视野内的动作进行精确的动作跟踪，无需任何外部的定位设备。

▼
Windows 10

Windows 10 秋季 Creator Update，Win10将会迎来几个新功能：包括图片视频创作工具Windows Story Remix以及UWP（Universal Windows Platform）的独立音量控制。Win10 继支持Ubuntu 子系统之后，现在又支持了 Fedora 和 OpenSUSE 子系统，这三个子系统都可以通过Windows Store 下载安装。
