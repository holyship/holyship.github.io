你想用什么方式，感受、拥抱这个瞬息万变的互联网时代？

优秀如你，选择一定不会少，有没有想过亲自见证一个产品从零到一孵化与成长？这也许是让自己完成蜕变最好的方式。而如果这款产品恰好是为技术人群服务的呢？为自己也为同行打磨一款产品，体验不能再爽！眼下就有这样一个机会——加入极客时间。

极客时间是极客邦科技旗下一款为技术人开发的产品，上线短短一个多月的时间，安装量、专栏订阅等数据就取得了不错的表现（此处用不错，因为谦虚是我们的品质）。相信，在未来可以预见的时间内，互联网的技术人都会记住这个名字：极客时间。

极客时间背后拥有一支实力雄厚的技术团队。池建强，主管研发、产品和设计团队，曾先后任职于洪恩软件、用友集团和锤子科技，有丰富的产品规划、架构设计和团队管理经验；孙涛，主管前端团队，大前端工程师，精通 Vue 和 Weex 开发，极客时间 Web iOS 和安卓主力工程师...... 更多技术大牛等你加入慢慢探索！

招聘岗位：Web 前端工程师

人数：多名

职位描述：

负责极客时间官网和 App 相关的各种 Weex、Web、原生开发 / 负责公司相关运营活动开发 / 负责公司各种大会网站开发

 我们期望你：


  • HTML 5、CSS 3、JavaScript 基本功扎实，能编写兼容多种浏览器的前端页面代码；

  • 熟悉主流移动 Web 开发框架，使用 Vue / React 在工作中开发过若干项目；

  • 熟悉至少一种后端语言，如果是 Java / PHP / Node.js / Python 等那真是极好的；

  • 有 Weex 或者原生 App 开发经验优先，对 Weex 非常感兴趣也可以；

  • 丰富的移动端开发经验、熟练使用伸缩盒布局、熟悉微信 JS SDK；

  • 对产品的交互、体验有自己的见解，并乐于改善产品的用户体验；

  • 思路清晰、逻辑性强，具备较好的问题分析与解决能力，具备工程化的前端思维；

  • 3 年及以上前端开发工作经验，精力充沛、最好是喜欢运动、爱好广泛；

  • 对「技术媒体」感兴趣，乐于「IT 知识」传播与学习；

  • 必须要有工匠精神，认可分享的价值。

 在极客邦工作，你将会：


  • 和有相同价值观的队友，每天开开心心把事情做好；

  • 真正的弹性工作制（目前的 Team Leader 是弹性工作信仰者）；

  • 从 0 到 1，亲手打造有趣有用的产品 (You Can You Up)；

  • 成为被自家产品服务的对象：免费学习 IT 课程；

  • 成为被自家产品服务的对象：零距离接触各种技术大神；

  • 成为被自家产品服务的对象：免费参加公司举办的各种技术大会：比如 QCon 和 ArchSummit；

在极客邦工作可不止上面这些福利，还有，

 高大上的办公环境

最近我们换了新家，新的办公室超大，装修也超用心，这里就直接抛图让你感受一下~

 完善的员工福利

无限零食供应，年终体检、六险一金、生日庆典、周年纪念、节日福利卡、集体出游、部门团建经费这些福利应有尽有。

饭补、通讯补助、交通补助、电脑补助、季度奖金、年终奖、人才推荐奖是我们的工资奖金结构，没有虚设，只要你是我们的员工。

 精进学堂

公司至少每周举办一次“精进学堂”，邀请内部员工和外部大咖分享经验与心得。相互学习，打造综合实力最强学习型团队。

这么有温情，又快速成长的团队，你准备好加入我们了吗？

如果你符合上述职位要求，同时认同我们的价值观，欢迎投递简历。
