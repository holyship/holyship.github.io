AWS 技术峰会2017 北京


AWS 技术峰会2017 北京 将于2017年7月26日在北京国家会议中心盛大开幕。

我们将为您呈现更加全面的云端商机，为您搭建一个倾听与分享的平台，您将有机会和来自 AWS 专业技术团队的大咖们共同探讨云计算的创新发展之路。

这个盛夏，我们期待与您相遇，共商云生态，共创云未来。


灵雀云展位：S19          我们期待您的莅临！
