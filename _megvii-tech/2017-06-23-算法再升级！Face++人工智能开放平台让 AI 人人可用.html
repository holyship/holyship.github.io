为了贴近用户并为开发者带来更人性的体验，旷视 Face++ 人工智能开放平台日前正式上线了产品博客。在这里，你不仅可以找到平台所有的官方公告以及合成各种识别能力的 Tips，还有限时优惠等活动的通知。此外，我们将持续改进平台功能，以满足用户的需求，提供更好的服务。



那么近期 Face++ 都做了哪些重要调整呢，下面为大家做一个汇总：


人脸算法更强更稳


1. 大角度 抗逆光

为满足各位开发者的使用需要，Face++ 人工智能开放平台团队对 Detect API 进行了全新升级。升级后的 API 可支持【任意角度】的人脸检测，甚至是完全颠倒的人脸也能识别出来，且可以通过 3D pose 返回人脸扭动角度值，并直观地展现图像中人物的【头部姿态】（headpose）。


2. 高精度多人种识别

同时新的算法对逆光、多人种的适应性也有较大提高，人脸比对精度大幅提升，尤其在【人种识别】的结果中表现强劲。


3. 喜怒哀乐 一键解读

新的 Analyze API 能够返回表情识别结果，目前 Face++ 人工智能开放平台根据人脸的变化程度将面部表情分为以下七种情绪，并在每一张人脸的识别结果的 attributes 中给出相关情绪的数值。



身份证识别算法更新

在实际业务场景中，少数民族同胞的身份证信息录入成为了用户和平台的困扰：例如有的少数民族姓名中会有一个“中圆点”，但光这一个“·”就有 4 种格式的输入法，无法识别“·”的格式，使用户在线上绑定银行卡等环节出现错误提示。



针对这一业务需求，Face++ 对线上身份证的识别算法也进行了更新，方便产品能够更好地识别少数民族的身份证信息，以提升业务处理效率和用户体验，具体更新如下：

1  对少数民族的身份证识别做了优化

  • 不同于通用的身份证样式可识别；

  • 少数民族名字更好的适应，包括生僻字、超长字、名字中出现“·”的问题。

2. 文字识别准确率大幅提升，对于采集过程中图片质量较差的情况，比如畸变，都很好适应。



付费版火力全开

Face++ 团队承诺免费版和付费版所享受的算法、性能是一致的，而为了同时满足多元化的用户需求，本次对付费版本（正式 key）的功能做了一些更新。具体如下：

1. 在 Detect API 中可一次性计算所有人脸的属性和 Landmark。

过去用户通过 Detect API 只能得到最多 5 张人脸的 attributes 和 landmark，其他的就需要再调用 Face Analyze API。本次升级之后付费用户（正式 key)可以一次性返回全部人脸的 attributes 和 landmark ，节省费用和缩短调用耗时。


2. 在 Detect API、Compare API、Search API 中可传入人脸位置进行人脸检测、比对、搜索。

用户在使用离线 SDK 或非 Face++ 的 Detect 功能，检测到了人脸出现无法识别或识别不准确的情况时，即可直接传入人脸框位置，强制进行人脸检测和后续操作。更方便用户离线 SDK 与云端 API 配合使用。


3. 在 OCR ID Card API 中身份证五分类的能力将只限制为付费用户（正式key）可用。

身份证五分类：即身份证照片的合法性检查结果，包含 ID Photo （正式身份证照片）、Temporary ID Photo （临时身份证照片）、Photocopy （正式身份证的复印件）、Screen （手机或电脑屏幕翻拍的照片）、Edited （用工具合成或者编辑过的身份证图片） 。
