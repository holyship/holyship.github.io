7月21日-26日，备受瞩目的 CVPR (Computer Vision and Pattern Recognition) 在美国夏威夷火奴鲁鲁 Hawaii Convention Center 召开。一年一度的 CVPR 是全世界计算机视觉领域的顶级盛会，作为今年 CVPR 的金牌赞助商和中国具有代表性的计算机视觉创新企业，旷视(Face++)的研究员们在旷视研究院院长孙剑博士带队下，将最新的研究成果带到了现场，这也是旷视(Face++)连续第 4 年赞助CVPR。


 共赴 AI 盛世


随着深度学习在各个领域的深化应用，计算机视觉迎来了黄金时代，而所有从业者们都能够从今年的 CVPR 的空前盛况中明显感受得到新时代中蕴含的无限光热——无论从参会人数还是论文数量来看，2017年的 CVPR 都是有史以来规模最大的一次。当地时间7月22日一早的大会开幕式上，主办方便公布了 CVPR 2017 的一组官方数据。



从上图中我们可以看到，相比过去的一年：

  • CVPR 2017 的论文有效提交数量增长了 40%，达到 2620 篇；

  • 大会录取的论文数量增长 22%，共计 783 篇；

  • 论文长演讲（Oral）、短演讲（Spotlight）共设 215 场，增长 4%；

  • 论文演讲的分会场由两个增加为三个（这也是 20 多年来的首次增长）；

  • 赞助商增长了 30%，达到 127 家，尤其是初创企业面孔大幅增加；

  • 赞助费总额的增长幅度为 79%，达到 85.9 万美元；

  • 参会人员注册数增长 37%，达 4950 人。


虽然是一场学术盛会，但是自古产学不分家。不管是就论文的投递、赞助的支持还是参会的积极性来看，近几年来产业界都为大会带来了越来越大的影响力。据悉，今年全球共有超过 90 家企业参与到本次大会中，包括谷歌、微软、Facebook、亚马逊、英伟达等国际科技巨头，也包括 25 家来自来自中国的大型互联网公司和初创企业。


 前沿应用多样化


会议期间，旷视(Face++)将诸多前沿技术应用带到了展台，其中专门为大会准备的CVPR 作者识别系统（CVPR Author Recognition）吸引了不少观众驻足，当有作者走进识别区域，屏幕上就会显示出作者提交的 Paper 信息，同时还会根据作者身份弹出活动提醒。



此外，旷视研究院还展示了跨摄像头行人 ReID 识别系统（Pedestrian Re-Identification）、人脸解锁（Face Unlock）、人形抠像（Mobile Portrait Cutout）、FPGA 人脸抓拍机（Low-bit ShuffleNet in Camera）、以及人体姿态实时估计（Multi-person Pose Tracking）等多类型产品形态。孙剑博士表示：“我们对 Demo 的选择原则很简单---展示系统一定是现场 Live Demo，真实的展现我们的研究和技术可以做到什么、还有什么做的不好，让大家有最直观的感受。”

而这些产品都是旷视(Face++)结合切实的商业场景需求而研发的前沿应用。如作者的身份识别系统可用于大型会议注册签到和企业访客管理；人脸解锁和人型抠像可以应用于手机、平板电脑等移动设备或应用程序中的设备/应用解锁和相机特效；双摄像头的 ReID 系统可以对视频流进行行人检测和特征提取，能够帮助公共安防领域实现从对单一的人脸追踪到对人型的综合追踪等；而 FPGA 人脸抓拍机则是旷视研究院最新公布的研究成果—— ShuffleNet 在嵌入式设备(智能摄像头)的应用。


微软全球执行副总裁沈向阳博士做客旷视(Face++)展台


 从研到产 从产到研


作为曾经两次（2009、2016）获得 CVPR Best Paper 的学术界代表性人物，孙剑博士表示这是他第 14 次参加 CVPR ，同时担任了本届会议的领域主席（Area Chair）。他表示，“2001 年我第一次参加 CVPR 的时候是以学生的身份，非常巧合的是当时也在夏威夷举办，不过是在考艾岛；印象最深刻的是参会人员穿着都异常随便，有个著名教授光着脚在会场走来走去，我就不说是谁了”。


恰恰在一年前的此时孙剑选择从大公司的研究部门投身创业公司，加入旷视(Face++)担任首席科学家，并带领旷视的研究团队围绕计算机视觉在基础研究和应用研究进行前沿探索。“这是一个剧烈变革的时代，我们看到世界上最聪明的脑力都投入到了人工智能的事业中。”在此前的分享中，孙剑博士不止一次表现出了对人工智能光明未来的信心。而今天，孙剑博士带领着旷视(Face++) CVPR 的小作者们走向了全球顶尖的学术舞台，这些平均年龄不到 25 岁的少年正是旷视的未来。


此次 CVPR ，旷视(Face++)被录取的论文共计 5 篇（其中包含作者现场长篇报告 1 篇、短篇报告 2 篇，海报展示 3 篇），内容涉及旷视研究院近期在做的三维深度学习、行人检测、文字检测、图像检测和分割等方面的工作成果。


  • A Point Set Generation Network for 3D Object Reconstruction From a Single Image（Oral 2-1C）

  • FastMask: Segment Multi-scale Object Candidates in One Shot (Spotlights S2-1B)

  • What Can Help Pedestrian Detection? (CVPR 2017 Poster)

  • EAST: An Efficient and Accurate Scene Text Detector (CVPR 2017 Poster)

  • Delving Deep into Coarse-to-fine Framework for Facial Landmark Localization (CVPR 2017 Poster)


旷视研究员范浩强作为第一作者

在 CVPR 2017 上发表长篇口头报告



孙剑博士表示：“旷视研究院的主要工作其实都在围绕计算机视觉中分类、检测、分割、回归、测度学习、序列学习等的若干核心问题展开的，不同应用只是对应不同的训练数据。而且公司搭建了强大的数据平台和团队来解决数据获取和标注问题，这样研究员们可以专注于问题的创新性研究。我们内部每天都有非常多的 ideas 被提出和检验，‘创业公司的基因 x 对基础问题坚持不懈的探索’保证了旷视科技强大的、持续地技术前沿性。经常有投资人问我们如何能保持技术壁垒和领先，我说在这个快速变化的世界没有绝对的壁垒，只有强大的团队才是领先的根本保障。”。


正如前面提到的，学术界和产业界永远无法分离而独立发展。事实上，学术界和产业界的研究本质是一致的，共同目标都是为了推动计算机视觉和人工智能走向繁荣，只不过作为产业派，旷视(Face++)希望进行研究工作同时能够让技术实实在在地、第一时间地产生商业或社会效益，让技术真正的改变生活。


“It is the end of beginning to search for the holy grail of computer vision."——孙剑博士在会议最后一天时，在他的朋友圈有感而发。
