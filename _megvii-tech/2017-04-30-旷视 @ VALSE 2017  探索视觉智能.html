第七届视觉与学习青年学者研讨会 ( Vision And Learning SEminar, VALSE ) 于 2017年4月21日至23日在厦门大学建南大礼堂举办。本届大会由厦门大学纪荣嵘教授团队承办，并由浙江大学潘纲教授担任轮值主席，参会人员超过2200人，所属单位涵盖了机器学习、机器视觉相关领域的主要国内大学和科研机构，同时大会也吸引了 BAT 在内的多家知名企业的鼎力支持。而作为铂金赞助商和 CV 产业界代表，旷视(Face++)也应邀出席，并由旷视(Face++)首席科学家孙剑带队出席。

VALSE 2017 盛况空前

VALSE 年度研讨会于2011年发起，此前，VALSE 已先后在杭州、西安、南京、青岛、成都和武汉成功举办 6 届。VALSE 取“华尔兹舞”的轻松、自然、飘逸和洒脱之意，旨在为计算机视觉、图像处理、模式识别与机器学习研究领域的华人青年学者（70后和80后为主）搭建一个自由、平等、高层次的学术交流舞台。

六年以来，这个舞台不仅促进了国内青年学者的思想交流和学术合作，也为相关领域做出重量级学术贡献，大大提升中国学者在国际学术舞台上的学术影响力。

计算机科学前沿分享

大会首日，主办方在厦门大学科学艺术中心音乐厅设置了 FSC ( 计算机科学前沿 Frontiers in Computer Science ) 专场，旷视(Face++)研究员范浩强作为嘉宾出席，并以《基于对抗学习的数据分布建模与应用》为题向在座专家、学者们分享了旷视(Face++)研究院的最新研究成果。

在有监督的机器学习方法中，生成模型 ( Generative Models ) 是一个常见的概念，旨在刻画数据的分布特性，因此针对图像预测与处理中普遍存在的不确定性与歧义性问题，生成模型能够发挥很关键的作用。在分享中，范浩强结合人脸识别、3D 重建等方面的研究实践经验，介绍了生成模型，尤其是对抗生成模型的原理与应用，并深入探讨了对抗生成模型中的若干问题。

深度学习的好时代

“这是我第三次参加 VALSE，从几百人到数千人的规模证明越来越多的学者们参与到了机器学习和计算机视觉的研究中，也证明深度学习迎来了最好的时代。” 旷视(Face++)首席科学家、研究院院长孙剑在特邀报告中表示，“计算机视觉是人工智能的一个子领域，在实际的产业化发展中有非常多的应用。”

在计算机视觉领域，拥有20年学术研究经验的孙剑博士是产学研三界的代表性人物，其主要研究方向是计算摄影学 ( Computational Photography ), 人脸识别 ( Face Recognition ), 和基于深度学习的图像理解 ( Deep Learning Based Image  Understanding )。自2002年以来，孙剑曾在 CVPR, ICCV, ECCV, SIGGRAPH, PAMI 五个顶级学术会议和期刊上发表学术论文百余篇，并分别于 2009年和2016年两次获得 CVPR Best Paper Award。2016年的夏天，孙剑正式加入旷视(Face++)，并带领旷视研究院的团队推进计算机视觉技术在工业和商业上的研究和实践。

聚焦感知智能 & 视觉智能

人工智能分为感知智能和认知智能，在讲解如何区分两者间的区别时，孙剑提到：“感知智能解决‘河边出现了一只狮子’的事情。也就是让机器能够像《人类简史》中的智人一样可以通过感官去获得河边出现了狮子的信息。而认知智能解决的则是’明天河边下午可能会出现一只狮子’的事情。也就是要实现计算机的认知革命，不仅要让机器能够感知到既有事物，还要基于环境和条件做出抽象的思考和预测。”

相比认知智能，感知智能领域随着深度学习方法的爆发而发展迅速。而感知智能的众多应用领域中，旷视(Face++)选择聚焦于智能传感器网络和智能云，并始终致力于解决一个问题：在教会计算机自动理解照片或视频中的信息之后，我们能用它来做什么事情？

与研究机构不同，旷视(Face++)是一家以实现技术产业化为目的的创新企业。基于核心的人脸识别和图像视频理解技术，目前旷视主要专注于泛金融和 IoT 两个领域以人为核心的智能化产品开发和智能数据服务。

视觉智能的核心问题


旷视研究院主要为公司的商业和产品提供底层的核心技术，包括图像分类、检测、语义分割和序列学习四大类，随后孙剑针对这些学科和旷视最新的研究进展做了详细的阐述。

▋图像分类

图像分类技术是指对图片做分类，可以是对图片中不同区域的分类，也可以是对图片中像素的分类。深度学习是目前最前沿的图像分类研究方法，学者们通过不断的增加神经网络模型的层次和深度来降低图像分类的错误率，从几十层到上千层的进步使得图像分类的效果已经取得了很大的突破。

深度卷积神经网络是开展相关研究的核心技术。而随着系统复杂性的提升，对于神经网络的有效训练显得尤为必要。为了提高网络的能力，研究员们使用了各种方法对网络进行优化，包括优化 filter 大小或 Low-rank 逼近法等。2015年，孙剑博士在微软时期领导的团队研发出了残差学习方法 ( ResNet ) ，基于 ResNet 跳层的基本技术原理，它可以帮助计算机很好地解决深层神经网络层级与准确度之间的矛盾。



在分享中，孙剑博士介绍：”在旷视(Face++)，我们希望将 ResNet 设计的更好。” 为了找到更好、更小的网络以适用于实际商业应用中更广泛的设备，孙剑带领旷视的研究员们提出了三种优化残差网络的解决方案：

1. 基于 ResNet 设计更高效的网络结构；

2. 对结构进行优化的三种方法（稀疏链接、tensor 分解、channel 精简）；

3. 通过 Low-bit 表示将 float 表示切换为低 weight 表示。


有了这些三种方式组合在一起能够让网络更小更快地运行在各种设备中，比如在非常低功耗的 FPGA 上可以实现 1080P 高清视频的实时人脸检测。



▋物体/文本检测

物体和文本是计算机视觉需要认识和学习的重要内容，我们不仅要让计算机能够识别出一张图片中是否有特定物体或文字，还有让它指出这些物体和文字在图片的哪一位置，且分别都是什么——这并不简单。

在针对图物体检测的研究中，孙剑的团队提出了一个新的 Faster R-CNN end-to-end 的训练方法，即先通过对图片卷积得到 feature map 后来预测 Proposal ，在做 Faster R-CNN，这种方式大大地缩减了检测的时长，并提升了检测效果的精度。


和物体检测相比，文本检测在我们的日常生活中有更强烈的市场需求。但是在研究和突破上，文字的检测比物体检测更难，研究方法也很不相同。“首先文字检测的主要特点是它的流程（Pipeline）非常长，其次从设计模块、调整参数到训练都要花非常多的时间和心力。” 孙剑提到，“但是旷视(Face++)在文字检测和识别中有了新的进展。”


他介绍，基于端到端的训练和 FCN (Fully Convolutional Network，全卷积网络）的简化文本检测算法，旷视(Face++)的小研究员们用一个极小模型就在公开评测集上取得了非常好的成绩。

▋语义分割



语义分割是图像分析的一项基本的计算机视觉技术，通过对图像的语义结构进行分割处理能够帮助计算机把人们感兴趣的目标物体或区域从图像中分离出来，其在现实场景中也有这广泛的商业价值，例如无人车、医疗、图像处理等诸多领域中。

深度学习的方法帮助语义分割的通用性和精度更高。通常学者们在语义分割研究中仍是基于 FCN 的方法，用下采样抽取后再上采样回来，输出一个 feature map 或是有语义的 map，来完成 end-to-end 的训练。而为了能够突破在大的感受野 ( receptive field ) 中实现很好的效果，旷视(Face++)的实习研究员设计了一种新的大 kernel 的方法，其结果在 VOC 2012 Segmentation on Benchmark 排名首位，同时其相关的研究论文也被 CVPR 2017 接收。


▋3D 感知


除了以上几个最为核心的计算机视觉问题，3D 感知也是计算机视觉科学中亟待突破的一个研究领域，研究者们关心的是如何用深度学习技术来为单张照片实现 3D 重建，也就是仅通过一张照片来还原照片中特定物体的三维形态。



在 3D 表示上，传统的方式是需要用深度图和 Mesh 的方法来实现的，但是这些方式通常具有不连续性给学习和研究带来很大的障碍。要用深度学习方法来实现有什么方法呢？针对从单一图像中的 3D 重建问题，旷视研究员们在最新的研究论文中提出了基于点云的方法来进行建模，并在基于单张图的三维重建问题中进行应用。

▋序列学习

随着深度学习研究的深入，深度学习不仅在机器视觉处理中，还有在音频和自然语言理解领域都得到了应用，而这些领域的同一特点便是需要对针对时序数据进行建模，因此就涉及到了序列学习的概念。

旷视(Face++)也十分重视序列学习的相关研究。目前，研究者们在序列学习的研究中采取的主要方法是通过递归神经网络 ( RNN ) 实现的 ，而旷视同样通过 RNN 来处理图像和视频数据中的时序结构。



“序列学习解决的是输入端可能是多个或一个，或者输出结果是一个或多个的问题，比如在典型的 OCR 识别问题中，图像中的文字是隐含着一定的顺序的，经过卷积后再通过 encode 和 decode 的方法把文字抽取出来，同时用 RNN 的方法为序列信息建模，才能正确地输出图像内的上下文信息。”孙剑介绍，“目前，旷视(Face++)已经在人工智能开放平台上向开发者们开放了 OCR 的能力，并证在大力推进 OCR 技术在商业领域中的落地。”

▋产生式模型

产生式模型 ( GAN ) 在近几年来非常热，其核心问题是如何能够生成序样，如何在极少甚至没有 label 的情况下从无到有地产生出观测数据。尽管 GAN 在被提出时存在训练不稳定等诸多问题，但后来的研究者们分别从模型、训练技巧和理论等方面对它做了改进，如今 GAN 的训练结果已经从不稳定逐渐变得鲁棒、可靠，其生成的数据达到以假乱真的地步。随后孙剑向观众们分享了旷视(Face++)针对产生式模型做的一项具体应用：

通过对大量人脸照片的 GAN 训练，能够在符合原图角度、光线环境的条件下为人脸添加眼镜、胡子等配饰。


和旷视一起 All in


回到最初的问题，旷视在让计算机识别出人、物之后会做什么？孙剑在演讲中提到，“我们用以上这些技术打造了各种智能产品或服务，其中云上的智能服务包括 Face++ 人工智能开放平台和 FaceID 在线身份验证平台，而另一条产品线主要覆盖智能传感网络，涉及智能安防和智能商业两个场景。”

中国的摄像头数量占全世界的 50%，但这些摄像头所产生的数据大多数是处于被闲置的状态，旷视(Face++)希望通过视觉智能解决这些问题。随着业务的开拓，除了主打的身份认证、视频结构化产品以及人员管理服务，旷视(Face++)也激发了市场中层出不穷的创新性商业应用。所以，现在不仅是深度学习的最好时代，从市场需求的角度来看，也是实现人工智能脚踏实地进入各行业的最好时代。

为了不辜负这个最好的时代，旷视(Face++)每一个参与者都保持着 All in 的心态。孙剑表示，研发人才是旷视保持核心竞争力的坚强后盾。会上孙博士面向全国顶尖的青年学者发出了邀请，力邀数学好、编程好和态度好的“三好学生”（其实只要“一好”就行）和旷视一起共赴人工智能革命。
