继今年3月份政府在两会工作报告中首提人工智能之后，5月14日，国家主席习近平在出席“一带一路”国际合作高峰论坛开幕式发表的演讲中，再次提到“‘一带一路’建设要坚持创新驱动发展，加强在数字经济、人工智能、纳米技术、量子计算机等前沿领域的合作。”如果两会政府工作报告是为人工智能在国内的发展打通了官渠，那习主席在“一带一路”的国际合作会议上则是将中国的人工智能发展汇入到了更深层次的世界海洋。


人工智能研究的是如何使机器具备类人的智能从而做出对事物的反应，它的关键起始点就在于让机器具备像人类感知万物那样的视觉功能，即机器视觉。机器视觉可以定义为给机器赋能，赋予机器一双慧眼，使它可以像人类那样去“看”、去“想”，从而实现各种检测、判断、识别、测量等功能。即便机器视觉只是人工智能的开源一环 ，但它已足够让人类的生活更加美好。



机器视觉为人类生活带来的改变


机器视觉让你的生活更加安全。人脸具备个人生物特征的唯一性，当机器具备高精度的人脸识别能力，那么当我们在管理个人金融账户时，具备人脸识别验证服务的金融机构将会让我们的资产更有保障；当我们在搭乘交通工具时，经过人脸识别核实过司机身份的网约车会让你的出行更加安全。当机器视觉进一步能够对人体、对文字、对车、对所有能被感知的事物进行识别后，个人的信息、财产都将因为有了一把智能锁而让你更加放心。


机器视觉让你的生活更加便捷。我们在做任何事情时，不用等待、没有更多步骤的操作流畅感是我们追求的极致体验，当机器具备了快速的人脸识别能力后，我们拿起手机检测到你的人脸便可自动解锁；对着镜头笑一笑，就可以开启电脑或手机上的任一应用；临近公司门口，具备人脸识别系统的门禁便已自动响应；在超市购物，拿完所需物品走出店门就已完成自动支付。如此种种流畅的操作体验，让我们的工作、生活享受到了更加便捷的服务。


机器视觉让你的生活更加精彩。现在我们钟爱的美图、Faceu 等照片编辑应用，因为具备强大的美颜、表情识别、人脸动画合成等功能而备受人们追捧，这些强大功能背后正是人脸检测、人脸识别的技术支撑。此外，一些在线婚恋网站也启用人脸识别技术帮助用户在数以万计的资料中筛选他们心仪的人，大大提高了匹配的成功率。机器视觉的应用已在不知不觉中带给我们更多惊喜。


旷视 Face++ 人工智能开放平台


采用先进的“深度学习”算法，用神经网络模拟人类的大脑可大大提升计算机的视觉能力，使机器更“聪明”地识别到人脸。旷视(Face++)于2012年开始使用深度学习并推出了Face++ 人工智能开放平台，以 API 或 SDK 的形式为开发者提供高效率、高精度的人脸识别、文字识别、图像识别等计算机视觉技术，帮助开发者的应用更好地读懂世界。

用Face++ 体验人工智能的澎湃动力


Face++ 人工智能开放平台提供了旷视业内顶尖的人脸识别技术，包括人脸检测、人脸比对、人脸搜索、人脸关键点定位、人脸属性分析等能力。



基于强大的深度学习算法，Face++ 的人脸识别技术可以准确的对照片中的人脸检测、定位，返回高精度的人脸框坐标，并进一步反馈包括人脸轮廓、眼睛、眉毛、嘴唇以及鼻子轮廓的关键点信息。利用这些高精度的关键点信息，Face++ 的人脸识别技术便可分析年龄、性别、微笑、头部朝向、眼睛状态等人脸属性，为开发者提供精准的人脸比对结果和完备的人脸分析数据。同时Face++ 的人脸识别技术还可在大规模的人脸集合上进行秒级运算，以极低的错误率完成人脸搜索，适应人证合一、复杂监控摄像多种应用场景的人脸搜索需求。



Face++  在多种复杂场景下的人脸关键点检测演示


Face++ 人工智能开放平台为用户开放了人脸检测 API、人脸分析 API 以及人脸关键点 SDK 。和 API 接口在线调用不同， 旷视提供的人脸关键点 SDK 是支持在移动设备上离线运行 Face++ 人脸识别算法的开发包，且贴合度高、稳定性强、跟踪速度快、鲁棒性强。可以实时追踪并完美贴合视频流中的所有人脸，在包括遮挡、大侧脸、运动、恶劣光照等各种极端场景下保持稳定的质量，并返回 106 个高精度人脸关键点，帮助用户在应用中实现人脸妆容、滤镜贴纸和3D动态模型等各类复杂变换操作。

在接入旷视(Face++)为企业和开发者打造的具备高普适性的产品API 或 SDK后，Face++ 的人脸识别技术可在零售、比赛、大学毕业典礼等场景下实现人流量的统计、笑脸抓拍，还可以在在线支付、安保、门禁等场景下通过人脸比对进行用户身份核验、人证合一等安防举措，减少安全风险，而人脸关键点检测及人脸属性分析则实现了照片编辑应用中的人脸照片编辑功能和企业的精准广告投放需求。宽广的应用场景，使Face++人工智能开放平台 API 日均调用量已超过2400万次，不仅成为人脸识别领域最具影响力的品牌，也促使Face++ 在实际应用场景中不断优化算法，使其在任何设备都能拥有强劲的表现。


目前，旷视 Face++ 人工智能开放平台已为超过50000名客户与开发者提供了人脸识别、图像识别等技术服务，得到包括支付宝、滴滴打车、Camera 360、美图、世纪佳缘、联想等众多行业巨头的青睐，未来旷视 Face++ 人工智能开放平台还将收获更多可能。


除核心的人脸识别技术外，Face++ 人工智能开放平台还提供人体检测、人体抠像、人体属性分析、证件识别、文件识别、场景与物体识别技术。对于技术的极致追求，是促进人工智能改变人类生活的根本动力，旷视(Face++)相信更多的能力、更多的可能将让人们的生活更加美好。
