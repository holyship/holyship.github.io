大赛主题及规则说明


 通过高效RDMA通讯协议 

 加速深度学习最新并行应用框架 


  • 参赛者会在报名后收到详细的竞赛题目以及培训资料，我们还将根据报名情况组织区域性培训；

  • 应用程序需在拥有 RDMA 能力的网络环境中运行（如 InfiniBand ），运行节点数与配置根据本校集群条件决定。如参赛队不具备 RDMA 环境，组委将会提供相应运行环境；

  • 参赛人员提交移植的应用程序源代码与执行文件，并提交详细报告说明改动前后的实际运行性能差异对比，并作出性能提升的解释（英文）；

  • 评审委员会审查源程序及报告，并安排远程或面对面的答辩，最终决定优势者；

  • 比赛结果将在HPC Advisory Council中国研讨会上颁布， 大会届时将邀请多位国外知名专家为获胜者颁奖。

▼

▉ Research Intern

职位描述

  • 协助或独立推进计算机视觉和深度学习领域的核心算法；

  • 协助或独立构建计算机视觉或深度学习领域的关键应用。

任职要求

  • 扎实的编程基础；

  • 对 CV / DL 有浓厚的兴趣，致力于在此领域发展。包括但不限于计算机视觉（包括分类，检测，分割，跟踪，和三维重建）；图像或信号处理；计算摄影学和计算机图形学等；

  • 熟悉 Python 等至少一门脚本语言，使用过 Theano, Caffe, Torch, TensorFlow 等开源深度学习框架优先；

  • 有深度模型训练，图像分类、物体检测与分割、视频分析、三维建模、计算机图形学等相关科研经历者（例如顶级会议第一作者）优先。



▉ Software Engineering Intern

职位描述 

  • 参与公司产品线核心架构、业务平台；

  • 参与云服务网站的开发、优化与维护工作。

任职要求

  • 计算机相关专业；

  • 对编程基础知识掌握熟练，包括编程、算法、数据结构、操作系统、面向对象编程等；

  • 良好的编程习惯，追求极致的代码品质，了解常用设计模式；

  • 开朗上进，积极沟通；

  • 有极强的独立解决问题能力，思想灵活，极强的学习能力；

  • 有开发类实习经验者优先。



▉ Product Manager Intern  

职位描述

  • 参与旷视公有云产品的策划与设计，将旷视的智能技术以云服务的形式服务客户；

  • 负责一个主要产品模块或者参与一个产品的需求定义、原型设计和优化、和产品上线计划。

任职要求

  • 产品直觉：热爱智能技术、热爱互联网，对产品的敏感度非常自信；

  • 用户思维：能从用户视角梳理需求、设计产品流程；

  • 设计能力：能使用工具完成产品原型设计；能完成 API 设计；

  • 数据能力：能完成有效的数据分析，具备海量日志分析能力者尤佳；

  • 协作抗压：沟通能力好，对有压力的工作氛围有充分准备。
