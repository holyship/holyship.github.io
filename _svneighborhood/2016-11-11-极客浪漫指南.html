又到了一年一度的光棍节～ 周五有好几个同事请假。问他们去做什么，他们认真地说：去庆祝Single's Day! 可见我们的节日已经深深影响到了广大硅谷的青年。都说程序员们都不修边幅，还不懂浪漫，所以今天我们就来写一写极客浪漫指南～！


《The Five Love Languages》这本书中说，爱的语言有5种：

Words of Affirmation: 肯定的话语

Physical Touch: 身体接触

Quality Time: 高质量的时间

Acts of Service: 做实事

Receiving Gifts: 收到礼物


一般来说，程序员和极客们对于“做实事”这一点，都做得十分到位 － 谁让咱们人实诚呢？所以今天我们来看看送礼物，讲讲极客们如何别出心裁地送出高水平、高质量、高品位、低风险的礼物～


送程序！

送礼嘛，当然要从最拿手的开始。无论你擅长哪种编程语言，都能送出充满爱心的程序。


Processing是一种类似于Java的编程语言，但是它能够非常轻松地创造出图形。简单的几行程序，就能画出一个爱心啦！

background(255);

size(100, 50); 

smooth();

noStroke();

fill(255,0,0);

beginShape();

vertex(50, 15); 

bezierVertex(50, -5, 90, 5, 50, 40); 

vertex(50, 15); 

bezierVertex(50, -5, 10, 5, 50, 40); 

endShape();


不用下载安装IDE，在网上（http://sketchpad.cc/）就可以运行。画出来效果是这个样子：



喜欢C++的小伙伴，也可以轻松地表达你那滔滔如江水的爱～


#include <iostream>

using namespace std;


int main() {

    cout << "I love you ";

    for (unsigned int i = 0; i < (unsigned int)(0 - 1); ++i) {

        cout << "so ";

    }

    cout << "much!" << endl;

    return 0;

}


这段程序，会打印出I love you so so so so so... much, 这其中的"so", 会被重复2^32遍哦！

送书！

还有什么礼物，能比把你们的故事做成一本书更特别呢～ 这个网站（http://lovebookonline.com/）就可以做到！



网站上的卡通都非常可爱


可以先创造长的像你们的两个卡通娃娃，再用它们做出你们的故事就好啦～ 制作的过程基本上是用鼠标拖拽，加上简单的几个词，非常方便。




送游戏！

说到送游戏，学问就大啦。送FPS游戏，对方肯定就一头扎进去不理人了。送实施战略游戏，玩的过程中压力太大，容易因为战略部署观点不一致吵架。而有的游戏就很适合！最近发现一个游戏叫做"Don't Starve Together", 中文名叫“饥荒”，真的是很适合送给另一半一起玩～




这个游戏有点类似Minecraft和Terraria, 都是要在晚上前收集好资源、做好准备，不然就会被怪兽杀掉。不过这个游戏的画面十分可爱，玩法也多样。不论是喜欢做农夫种菜、杀蜘蛛捡火腿肠、还是做科学研究制造各种魔法，都可以自由发挥。


游戏中的生存环境十分恶劣，尤其是冬天。这就是最适合增进感情的季节！在寒冷的冬天，两个人又冷又饿地缩在火堆边，讨论着下一餐怎么解决，真的是很好玩～ 因为游戏的图像做得十分不错，玩的时候感觉游戏里的冷和饿都蔓延到了现实生活中一样，在硅谷十几度的高温下瑟瑟发抖... 如此恶劣的生存环境，不仅让两个人更团结，还能更快的变饿，一起出去吃个宵夜什么的～
