在我的记忆里，好像所有跟青春有关的故事都发生在夏天。


毕业后那年的我，组了乐队，跟喜欢的朋友们在一起，做着自己想做的事，感觉就连呼吸都是自由的。预谋已久的我，自己给自己的手臂上纹上了人生中的第一个纹身。因为瞒着家人，在那段时间里，再炎热的夏天，我也总是穿着长袖。


几天后，某天晚上我爸跟往常一样做了一大堆卤味。我一边啃着鸡脚一边从冰箱拿了一瓶啤酒，刚准备喝，听见我爸说，“少喝点，容易发炎。”


发炎？我奇怪的看着我爸。而他指指我用袖子，那里遮住的正是我的新纹身。


那天，虽然我只是哦了一声。但对于不说破的我爸，我既惭愧又感激。


时光荏苒，最近公司来了不少刚毕业的新面孔，做了一大锅卤味和大家分享，看着90后的小朋友们吵吵闹闹，我突然觉得时间停格在了那个无知无畏的少年身上，不曾改变过。


时间被食物赋予了记忆的味道。总有一天，大家都会从青葱的模样，卤上时间的印记。回首被标记上青春那个滋味，最为弥足珍贵。




▲酥饼大人：我也是青葱少年！


食材


八角4颗 / 桂皮2段 / 白蔻5颗 / 砂仁4颗 / 香叶3片

甘草4片 / 草果2颗 / 山奈5颗 /白芷3片 / 干辣椒6个

花椒 / 小茴香 / 丁香



冰糖 / 姜 / 大葱 / 高汤

牛肚 / 鸡脚 / 鸭翅 / 鸭脖 / 牛腱肉 /

豆干 / 西兰花 / 土豆 / 藕

 


步骤


1、把所有香料放进热水中浸泡十分钟，捞出装入纱布封口制成香料包，备用。

 


2、锅中热油，放入冰糖小火炒化，加入高汤，放葱，姜，盐，冰糖，料酒，淋生抽调味，少许老抽着色再放入香料包，烧至沸腾后转小火慢炖，熬成卤水。


 

3、牛肚、鸡脚、鸭翅、鸭脖，牛腱肉等肉类入冷水锅焯水，豆干、西兰花、土豆片、藕片蔬菜豆制品分别切片。

 



4、处理好的牛腱肉及牛肚放入卤水中焖煮1小时；牛腱肉入锅45分钟后，放入鸭脖、鸡爪及鸭翅。豆干、藕片及土豆片只需最后卤煮5分钟即可。




5、关火，所有食材继续泡在卤水中1——2个小时，待卤水完全冷却，食材都入味后即可取出，切片摆盘，淋上卤汁，完成！
