记得我当年的高考，第一门就考砸了。


具体的情况不太记得了，只是那天考完走出考场，外面热得让我发晕。依稀记得是我爸来接的我，至于路上说了什么早就忘了，也可能是，他什么都没问，我也什么都没说。


考试下午还有一场，中午只是简单的吃一些，但我仍记得那天中午吃的凉面。



屋檐遮蔽了阳光的暴晒，让头脑多了一些清醒。面前的一盆凉面，拌着鸡丝和黄瓜，淋上酱料，还有一点点红油，这一直是我夏天的最爱。我吃凉面时顾不上的其他菜，总爱大口吞面，直到现在都是这个习惯。所以经常吃到胸口发噎，还得缓一缓等那口面滑落。


而此时，我爸就会抬手，拍拍我的背。没有更多关照和唠叨，平常到无以复加，但对我已足够。


夏天是个特殊的季节，因为好像在夏天遇到的一些事，在未来的某天，都会变得值得怀念。


今年，又到吃凉面的日子了。



▲酥饼大人：高考的孩子们，沉着面对吧~



鸡丝凉面


辣椒粉 / 白芝麻 / 花椒粉 / 花生碎

葱3根 / 姜3片 / 大葱3小段

碱水面 / 鸡胸肉 / 黄瓜 / 胡萝卜



1、制作椒麻油：锅中倒大量油，加入葱段，姜片烧至冒烟，捞出弃之，把热油倒入装有辣椒粉、花椒粉、白芝麻的碗中，辣椒油完成。




2、面条入冷水锅，煮熟，捞起过凉水，放入篓子里倒入适量熟油拌匀，在电风扇旁吹凉。



3、调一碗酱汁：生抽、麻油、醋、糖，按照4：2：2：1的比例倒入一个碗中，调匀备用。



4、熟鸡胸肉撕成细丝；黄瓜，胡萝卜分别切丝备用。



5、晾凉的面条装碗，铺上黄瓜丝，胡萝卜丝，熟鸡丝，撒花生碎，最后倒入酱汁和椒麻油拌匀，完成！




再配一道可乐排骨

一口菜，一口面：）


可乐排骨


排骨 / 可乐 / 葱姜蒜 / 干辣椒 / 八角 / 桂皮 / 白芝麻



1、800g排骨切块洗净，下冷水锅焯水，捞起备用。


2、锅中热油，下姜蒜碎和干辣椒爆 香，再加入处理好的排骨，翻炒至排骨表层变色。

3、放入2个八角、1个桂皮，加料酒、生抽各一勺、老抽半勺、和适量盐，最后倒入可乐没过排骨，大火烧开之后转小火，加盖再闷半小时，大火收汁。




4、收至汁水浓稠，装盘撒上白芝麻和葱花，完成。
