我工作室人数不多，社团却不少。什么游泳社、篮球社、火锅社、电影社，最近还出了一个“吃鸡社”。没错，就是最近周围人人都在玩的游戏：）




“今天你吃鸡了吗？”“嗨呀，好气啊，差一点就吃鸡了！”群里每天都在互相问候，下班时间大家也组队“吃鸡”，特别热闹。


我最近刚入坑，让他们带着我一起，好在大家也不嫌弃我技术渣，为了报答，我就做了顿真的鸡犒劳同学们——家喻户晓的宫保鸡丁╮(￣⊿￣)╭"


宫保鸡丁有多好吃？毕竟曾是御用名菜，就不用我形容了。鸡肉滑嫩，花生香脆，辣中带甜，油亮亮的汤汁浇在白米饭上，三碗都不嫌多。即使大家平时不常做饭，这也是很有必要学的一道硬菜。



吃了这道红红火火的宫保鸡丁，祝大家大吉大利，晚上吃鸡~




▲为了阻止我沉迷游戏，酥饼占了我的电脑桌……


食材


鸡胸肉 / 鸡蛋 / 黄瓜 / 花生粒 / 姜 / 京葱

干辣椒 / 花椒 / 郫县豆瓣酱 / 白砂糖 / 料酒 / 淀粉 / 醋



步骤


1、一块鸡胸肉洗净，切约2cm的方形小丁，加入1个蛋清、1勺料酒、1勺淀粉和半勺盐拌匀，腌制15分钟备用。



2、花生粒炸香去皮，黄瓜洗净去皮去籽切丁，京葱洗净切小段，干辣椒剪段去籽，姜切末，备用。



3、将3勺生抽、2勺醋、1勺料酒、少许清水、半勺白砂糖和半勺盐混合均匀，调成酱汁备用。



4、 热锅冷油，下京葱段、姜末、干辣椒段、花椒和1勺郫县豆瓣酱炒出香味。放入腌制好的鸡丁，煸炒至其变色。



5、加入调好的酱汁翻炒均匀，放入黄瓜丁和炸花生粒，最后倒一小碗水淀粉勾芡。




宫保鸡丁完成！

宫保鸡丁算是我从小吃到大的一道菜了，简直就是“家的味道”。说到“家的味道”，还有一样东西不得不提，那就是小时候的大冬天，出门前一定会抹的百雀羚。

想起不久前的双十一，国货百雀羚战绩夺冠，2.94亿销售额，并且连续三年夺取化妆品类冠军。作为大家都喜爱的国货化妆品，百雀羚针对不同人群、尤其是年轻人推出个性化定制的跨界产品。

能坚持将一件事做那么久，越做越好，是件值得钦佩的事。希望很多年过去，我也能一样，继续从事喜欢的影视行业，日食记也能一直继续下去：）
