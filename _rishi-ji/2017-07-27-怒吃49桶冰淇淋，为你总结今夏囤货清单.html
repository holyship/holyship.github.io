入夏后公司的冰箱从来就没满过，成箱的冰淇淋运来总是一天不到就被吃光了。


就这还不够，我的同事们把冰箱吃空之后，默默分享了这篇吃了49桶冰淇淋的心得报告给我。什么意思我不是很懂，希望大家看完可以告诉我：）





巧克力口味冰淇淋，21种


香草味，20种


个性口味，8种

（其实就是没法分类）



共计49种冰淇淋，15位探员幸福地哆嗦着完成了本次的冰淇淋评测，共计耗时2.5小时。


价格和购买渠道也一并附上了，方便你们剁手。




// 巧克力组





局座点评：

正常黑巧克力的味道，隐约还能尝出一些些坚果味，也带有黑巧独特的风味——有些发酸，甜苦适中，连吃几口也不觉得腻。


口感很赞，让局座脑海中浮现出德芙的经典广告语——如丝般顺滑。





局座点评：

质感很温柔，绵密软糯，入口香甜味立刻占据整个口腔，随之而来的是很浓很正的巧克力味，里头有一些磨碎的巧克力，让冰淇淋的口感有了些层次感，挺好吃的。


如果巧克力碎是脆的就更嗲了。





局座点评：

冰淇淋的味道，两个字——正点。


入口可可味儿很浓郁，巧克力味自然不做作，空气感很足，口感绵密又顺滑，就像欧洲老贵族一样经得起一口又一口的细细推敲。





局座点评：

凯蓓蒂算是进口超市冰柜里的常驻客了，很多探员都表示曾经不止一次被吸引过，然而看了眼价格又自觉放了回去，属于那种不下点狠心可能会一直走过路过又错过的冰淇淋。


凯蓓蒂来自冰淇淋消费量世界排名第二的国家新西兰，所以奶味你们完全可以放心，相当纯正，尤其是后味，奶香悠长。


巧克力味一开始入口会觉得有些不适应，因为表面的巧克力碎比较坚挺，往往冰淇淋化完了以后还能继续咀嚼一会儿…不过多吃两口以后就会忍不住开始点头，“嗯还不错”。





局座点评：

新西兰真是生产和享受美味冰淇淋的理想国度啊，据说在新西兰每人每年能吃掉38L的冰淇淋，相当于每人每个星期就能吃掉2斤左右的冰淇淋（基本上等于9块光明冰砖），最初看到这些数字的时候局座受到了惊吓，但吃完上面的凯蓓蒂，紧接着又吃了这款柯林高德，似乎理解了新西兰人民的感受——这么好吃的冰淇淋不拼命吃，简直就是暴殄天物！


柯林高德的巧克力味也做得很正，口感比前几款都更丰富一些，有质地顺滑、奶香浓郁的冰淇淋，有需要咀嚼的固体巧克力碎，还有湿润的巧克力酱。除了罐子小了点价格贵了点，真的没什么可挑剔的点。



// 香草组






局座点评：

虽然仔细观察下来，它的冰淇淋里并没有什么香草籽的踪影，但它的香草味却特别香甜浓郁，是香草组里香草味最浓的一款。也有少数几位探员和局座一样，觉得它别的都挺好，只是稍微甜了一些。





局座点评：

巧克力味还不错，香草的表现也挺好，身世坎坷的蜂雪颂似乎不太费劲的就展现了自己的实力。


香草味的香草籽加得特多，可以直接吃到香草籽的口感，也有被不喜欢这种口感的探员描述成“豆渣口感”。


奶味清新，不是齁甜的那种；质地轻盈，像在意式冰淇淋里打了很多空气，所以放置到常温环境里的时候，化得特别特别快。





局座点评：

之前试吃巧克力口味的时候，只是觉得一般，像日本超市里的“伊利”。结果，香草味意外的还不错，虽然完全没吃出什么香草味，但顺滑度可以，奶油味很重，喜欢奶味的同学可以自行一试。





局座点评：

香草味和奶味都做得恰到好处，质地细腻顺滑，像加了很多奶油的感觉，甜而不腻，把它比作香草冰淇淋的标杆也不为过。

 




局座点评：

奶油味非常赞，也非常香。可能你会觉得它的香味浓郁得不太自然，但是把它含进嘴里之后，就是对它讨厌不起来，反而有种陷在热恋中不能自拔的感觉。


大老爷们儿如此形容一款冰淇淋好像太奇怪了，换一句话说好了，要买柯林高德就应该买大罐的！





局座点评：

本次食鉴单价最高的冰淇淋就是它了。分量也多，足足2L，来自新西兰，again，只是牌子比较陌生，平日里没有看见过。


打开盖子以后有种和外包装上的冰淇淋球色差很大的感觉，看起来不怎么像香草味，反而很像哈密瓜味。实际味道…香草味也确实不明显，完全被浓浓的奶味给盖住了，整体还不错，只是别说自己是香草味就行了。




// 雷区警告


好了，赞美的篇章就到这里打住吧，再要继续局座表示已经使不上力了，好吃的冰淇淋你要是不自己亲自吃上一口，穷尽局座毕生所掌握的词汇量也是徒劳。但不怎么好吃的冰淇淋，要描述起来可就形象生动得多，言传意会起来也更容易得多。下面请收看来自食鉴局探员的雷区警告——

02 丨不太好吃的香草冰淇淋

↓↓↓


香草和巧克力应该是冰淇淋的众多口味中，最最基础和大众接受度最高的味道。


按照吃货界正常的推理逻辑，基础口味做的好吃的冰淇淋，可能别的口味也不会太差，至少质地和口感能在基础水准之上吧，大家可以类推着尝试好吃品牌的其它口味。当然，也不排除某些有个性的生产商，就是不擅长做基础口味的冰淇淋，因此也不能一概而论。


反正局座左思右想，总觉得吃完上面这些颇有不满足，于是，一为解“一口气吃掉二十几种巧克力和二十几种香草味冰淇淋”的腻，二为解“既然都找到了其他看起来也很不错的冰淇淋，不买回来尝一尝怎么行”的馋，顺道就多吃了8款“其他口味”，试水结果如下： 



// 无法分类口味组 






局座点评:

在家乐福买八喜的时候意外看见了它，因其雪芭的质地，得到最多的描述就是“一股清流”，酸酸甜甜沁人心脾。





局座点评:

榴莲味浓郁，香味自然悠长，还能吃到榴莲肉；口感顺滑细腻，比榴莲肉的质地更清爽一些，回味很棒，可以说是局座吃到过的所有榴莲味冰淇淋里，还原度最高的一款，当天就被榴莲爱好者探员们瓜分殆尽了。





局座点评:

无论单独吃哪种味道都挺一般的，没有什么特别之处，味道跟小时候吃的光明三色杯的味道一毛一样，但是是浓郁版、加强加大版的三色杯。只是，记忆里的三色杯，总是一个人拿着在吃，而TipTop则可以两个人瘫在沙发上，你一勺我一勺分着吃，所以给探员们带来了格外美妙的感受吧。



丨不太好吃的其它口味冰淇淋



局座点评:

这款无论是电商还是进口超市，都挺常见的，记得千万别买。就像mm的巧克力豆直接被推进了碎肉机里，无论是视觉上还是口感上，颗粒感（尸块）都过于明显了点。


冰淇淋完全沿袭了巧克力豆的风味，味道倒不难吃，如果不放巧克力豆可能还有救，残碎的豆子混入其中，就显得用力过度了。





局座点评:

和上一款一样，也是个很常见，容易上当的雷。冰淇淋和奥利奥饼干的味道一模一样（那我为什么不直接吃冻奥利奥），奥利奥有多甜，这款冰淇淋就有多甜。


只是质地上来说，像在吃一块在水里泡到坨成烂泥的奥利奥，稀啪啪的，稠稠的，脑海中浮现了婴儿辅食…




局座点评:

这味道要如何形容呢——既此又彼，非此非彼。像制作失败的慕斯，又像一块被落在车里、经过炙烤暴晒后变质的蛋糕，在嘴里搅过来又搅过去，总觉得咽下去有些心理障碍，于是只好心想着这种新奇的口味果然还是不适合大部分国人来自我安慰。


另外嘛，文中没有特别提到的冰淇淋，就比较一般，毕竟“没比没伤”。


很多冰淇淋可能单独买来吃的时候都觉得透心凉爽歪歪，怎么吃怎么满足，但在我局，这么多优秀选手集中到了一起，必定会凸显出艳压群芳的，和被群芳碾压的。大家看过吃过，自行衡量判断。
