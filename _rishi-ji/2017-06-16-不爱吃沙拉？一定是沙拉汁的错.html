以前，好像还常常能看到彩虹的，特别是在暑假的暴雨之后。每次看到彩虹时，我都兴冲冲拽着我奶奶一起看，她也会跟小孩一样一惊一乍的开心。每当这个时候，我就有一种许愿成功般的喜悦。


但不知从什么时候开始，再也没在城市里见到彩虹了。


虽然不总能有遇见彩虹的好运气，但制造彩虹的能力，却人人都有。比如今天做的这道彩虹沙拉。


很多人不喜欢吃沙拉，照我说，你们简直不知道自己错过的是什么：）


沙拉算是做起来最简单的菜了，切一切拼一拼就好了，只要你沙拉汁调配得当，它简直就是集好吃方便健康为一体的冠军菜单。所以有句话说，没有不好吃的沙拉，只有不好吃的沙拉汁。这次我浇的是油醋汁，自己调配的，算是我的最爱了。公司里女同事都指着吃这来减肥。


能不能真的减肥我不知道，只是有这样的一道彩虹，让梅雨季节也不那么难熬了。


▲酥饼大人：减肥就不用了，我想看彩虹～


食材


鸡胸肉 / 鸡蛋 / 圣女果 / 牛油果 / 紫甘蓝 / 玉米粒 / 生菜

特级初榨橄榄油 / 苹果醋 / 蜂蜜 / 黄芥末籽酱 / 黑胡椒



步骤


1、调油醋汁：将特级初榨橄榄油，苹果醋，蜂蜜，黄芥末籽酱，盐和胡椒粉都倒入容器中，充分拌均，备用。



2、鸡胸肉加盐、黑胡椒和橄榄油，腌制15分钟后入油锅双面煎至熟。



3、煎好的鸡胸肉、牛油果、圣女果分别切成方便入口的块状。紫甘蓝切丝；玉米粒焯水，鸡蛋煮熟放凉1切4，依次码进铺了生菜垫底的盘中。



4、撒黑胡椒，倒油醋汁，拌匀，完成。






再来杯樱桃特饮


樱桃苏打


1、樱桃洗净去梗去核，放冰糖，加水没过樱桃，烧至冒泡后转小火煮至粘稠，盛出晾凉。



2、舀两勺樱桃酱放杯底，苏打水倒至8分满，加2片柠檬，点缀薄荷叶，完成！




一口色拉一口苏打~

今天来送罐头的是（当当当）——欧丽薇兰特级初榨橄榄油。不得不自曝一个自己的丑事，就是记得我第一次用橄榄油时还不知道特性，总拿来高温炒菜。现在想来，真是暴殄天物。


从小家里厨房都少不了金龙鱼的粮油产品，这次的彩虹沙拉所用的欧丽薇兰特级初榨橄榄油，算是橄榄油界相当有口碑的产品了，查询一下资料：


来自地中海庄园直供的优质橄榄油，充满浓郁果香和清爽口感，可以代替黄油涂在面包上、凉拌、调制酱料，甚至直接可以饮用，单不饱和脂肪酸平均含量高达79%，使用珍珠棉和双面瓦楞纸，在包装上也非常用心环保。
