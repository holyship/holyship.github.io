最近天热到什么程度呢？总爱在窗边晒太阳的酥饼，最近也对阳光绕着走了……再晒连猫都要化了。


因为热，吃所有东西都恨不得加上冰块，居然发展到连吃面条都不例外了。


面条里下冰块这事，还确实有这么干的——韩式冷汤面。美食无国界，我工作室的肉圆妹超爱冷汤面，那面碗就跟个小脸盆似的，而她那小小的个子居然可以连吃两大碗，看来确实是好吃。刚好管家同学这次做了冷汤面，于是我也以猎奇的心理提前蹭吃了一次。


冷汤带一点酸甜口，面条冰凉滑溜，口口生津。虽然这口味不是常规面条该有的路数，但却意外的很好吃，让人着魔似的停不下口，吃完整个人都降温了三度。


汤汁清澈冰爽，以晾凉的牛肉高汤和浸泡过水果的酸甜汤混合而成，它打破了吃汤面时一贯的浓重口味，成为最适合夏天的小清新。经过发酵的大白菜褪去了原有的甘涩味道，一口面，一口汤下肚，鲜美凉滑不可言喻。


从此，我也开始爱上吃放冰块的面条了。

食材

泡菜 / 苹果 / 梨 / 菠萝 / 青椒 / 黄瓜

牛腱肉 / 荞麦面/ 玉米粒 / 鸡蛋 / 京葱 

紫洋葱 / 小苏打 / 白芝麻 / 八角 




步骤


1、半斤牛腱肉入冷水锅焯水，捞出另起一锅放入，依次加入京葱、姜片、八角小火炖煮1小时，撇去浮沫，留汤备用。



2、青椒（去籽）、洋葱、苹果、梨、菠萝、黄瓜切块入碗，加2.5升矿泉水，放糖、适量小苏打、白醋、酱油和盐，浸泡1个小时后捞出蔬菜和水果，汤里撒上白芝麻，酸甜汤汁完成。




3、酸甜汤汁和牛肉汤以1：1的比例倒进碗中混合均匀，加冰块和白芝麻，做成冷汤。



4、荞麦面煮熟后过凉水，再入冰水中浸泡5分钟，捞出入碗，倒入冷汤，铺上黄瓜丝、玉米粒、泡菜和鸡蛋，泡菜冷汤面完成！
