最近年末，我公司的小伙伴开始以各种借口搞活动。


什么圣诞趴体啦，冬日庆典啦， 每天下班后我都很开心，总有好吃的塞到手里。



这两天，他们又在筹备跨年计划，每人准备一道菜。


跨年是一件值得在意的事。当新年零点的钟声响起，身边陪伴着谁，第一句话说什么，第一口吃的食物，这些事都让人在意，就像是一种特别的许愿形式——希望新的一年能跟喜欢的人多在一起，或是过得像吃到的第一口食物一样圆满。


今年的跨年菜谱，是圆满又美好的香芋酥。



粉紫色螺旋纹的外皮，一刀切开后是流油的咸蛋黄和富有嚼劲的麻糬，外皮酥蛋黄粉麻糬糯，三重口感在嘴里爆炸。不爱吃甜点的我居然连吃了两个不带停。


吃完之后觉得还缺点儿什么，小伙伴又端出一碗黄澄澄的南瓜浓汤，淡淡的甜味，一口下肚，从胃里暖到心上。


这个冬天，一点儿也不冷：）




▲酥饼大人：南瓜浓汤，整个冬天都暖暖的



｜香芋酥｜


｜馅｜


芋泥：芋头 / 细砂糖 / 淡奶油 / 无盐黄油 / 紫薯粉



麻薯：糯米粉 / 玉米淀粉 / 牛奶 / 细砂糖 / 无盐黄油



咸蛋黄：新鲜咸鸭蛋洗净，取出咸蛋黄，放在烤架上，表面喷上高度白酒，放入180度烤箱，烤5分钟后取出，备用。



芋泥：

1. 550g芋头去皮切片蒸熟后，加入50g淡奶油和10g紫薯粉，放入破壁机中，搅拌成细腻的泥状即可。




2. 将搅拌好的芋泥倒入锅中，中火加热，加入200g细砂糖和100g无盐黄油，持续搅拌以免锅边过于干燥而结块；待锅内芋泥沸腾后，转小火，继续不断搅拌翻炒，直至糖和黄油充分融入芋泥中，芋泥呈顺滑状即可关火，倒入容器中冷却，备用。





麻薯：

3. 将30g细砂糖和120g牛奶倒入锅中，小火加热至沸腾；将70g糯米粉和20g玉米淀粉放入碗中，一边倒入加热好的牛奶，一边搅拌，搅拌均匀后，上蒸笼大火蒸20分钟。



4. 放置温凉后，加入10g无盐黄油，揉匀至完全吸收，备用。



5.用芋泥把麻薯和咸蛋黄包裹住，搓成团，每个馅重量约为35g，盖保鲜膜，入冰箱冷藏备用。（收口注意要收紧，以免露馅）



｜外皮｜

水油皮：中筋面粉  /  植物油 /  糖粉 / 沸水


油酥：中筋面粉 / 猪油 / 紫薯粉


1. 水油皮：将150g中筋面粉、40g植物油，30g糖粉和80g沸水放入厨师机中，一档起步调至中速，搅拌成团后取出，放在撒上面粉的操作台上，摔打几下面团，盖上保鲜膜，入冰箱松弛40分钟，备用。




2. 油酥：将110g中筋面粉、70g猪油和15g紫薯粉放入厨师机中，一档起步调至中速，搅拌成团后取出，包上保鲜膜，入冰箱冷藏松弛40分钟，备用。



3. 将松弛好的水油皮和油酥，均分为6等份。（水油皮每个50g，油酥每个32.5g）用油皮把油酥包裹住，收口要注意收好，以免漏酥（可以随时盖上保鲜膜，以免操作速度慢导致水分流失）



4. 将包好的面团用擀面杖擀成舌型，然后卷起，盖保鲜膜，松弛30分钟。（天热时，建议放入冰箱冷藏，否则油酥变软不易操作）



5. 将面团竖放，再用擀面杖擀成舌型，然后卷起，盖保鲜膜，松弛15分钟。

*擀面团和卷起的动作要轻柔，不能大力，以免破酥浑酥，破坏最后酥皮的层次感。



6. 用刀将面团对半切开，擀成圆形。（刀需要锋利些，粗钝的刀会破坏切面层次）



｜包馅和烘烤｜


1.在香芋面皮上放入之前做好的馅料，包好收口朝下，放在事先铺好油纸的烤盘上。（收口注意要收紧，以免露馅）



2. 放入预热至170度的烤箱中，烘烤30分钟后取出，放冷却架上冷却，香芋酥完成！

*未及时食用完的香芋酥可以密封保存，也可以冷冻保存。需要食用前，喷点清水，进150度烤箱加热10分钟即可。








｜南瓜浓汤｜


日本绿皮南瓜 / 牛奶 / 淡奶油 / 糖



1. 半个日本绿皮南瓜（约400g）切小块，去皮。待蒸锅中水煮沸后，上蒸笼大火蒸20分钟（直至筷子能轻松插过南瓜块即可）。



2. 将蒸熟的南瓜块、50ml淡奶油、100ml牛奶一同放入破壁机中打成细腻的糊状。 将打好的南瓜倒入锅中，中火加热，加少许糖调味，南瓜汤煮至粘稠即可关火。



3. 盛入碗中，表面可用淡奶油点缀图案，南瓜浓汤完成！
