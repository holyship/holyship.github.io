嘀嗒你好，


看了嘀嗒嘀嗒最近的文章，想说的有点多，但留言不能超过600字，就把一部分留言写在这了：看过文章又想到一个老话题—要不要读研。看过几篇文章，都不是很支持，角度大多是“读研不代表有好工作”“时间成本太高”“导师不靠谱”等等，这些都是实际问题，但不经过这样系统的训练，如何能快速得到或着说找到正确路径获得安姐要求的这样的思维能力呢？安姐能写出如此受大家欢迎的文章，想必和在学校里受到的训练是分不开的吧。安姐如果有空，能不能写一篇关于如何获得好的思维能力的文章，世界观和方法论都写写，感激感激！/玫瑰



玫瑰你好，


关于要不要读研的问题，很多人写过文章了。记得我也有提过几次。今天再分享一下我的感受吧。


我读硕士和博士的主要专业一个程序语言设计，二是 BioInfomatics。


前者加深了我对程序语言的理解，更清楚各种程序语言最基本的设计理念，以及一些在性能和表达上的优缺点。当然，简历上写着“我能写一个编译器”，大部分时候对于找工作并没有什么帮助，但是在这一领域的深入学习，让我更加了解语言的编译过程，更容易从本质上看到一门语言的局限性，这确实让我现在对学习任何一门新的程序语言对更有自信，并可以很快地学习一门语言并上手。


至于在 Bioinfomatics 方向的研究，除了对很多生物学，尤其是染色体和蛋白质网络的构建有了解之外，更多的是培养了我在数据采集、清理、建模、分析方面的能力。生物学大家知道，很多研究方面的数据都有很多的噪音，所以当时练出来的一个技能，就是可以很快写出各种 Python 脚本，各种 SQL query 等对数据进行处理。现在工作中，往往也需要很多 SQL 的技巧，帮助发现线上的问题。尤其我做支付还牵涉到很多线下财报数据管道的建立等，这个技能就让我的工作更有效率。也在一定程度上增加了我对数据的敏感度。


当然，每个专业的研究生后来能用到的知识技能都不太一样，这还要看你学什么了。


除了这些之外，还有哪些收获呢？


第一就是心理素质。国内我不够清楚，但是美国越是好大学，毕业越难。因为毕业论文的要求更高，需要做出的课题难度也越大。想想看，现在这么多好大学，这么多聪明的学生，容易研究出成果的额课题早就被做完了。剩的都很难做出什么让人兴奋的成绩。所以做研究和工作的第一个差别，就是工程你知道一定是能做出来的，而研究很多时候真的仅仅是在尽力而为。


所以，经历了研究生阶段，现在不管职场生涯还是工作中遇到困难，都能有一颗平常心。觉得这能难到哪里去呢？然后就不想太多，只努力去解决问题。人只要用力逼一逼自己，往往就能突破自己。


也因此，很多研究生毕业的职场新人，也会略显稳重。不过这也可能仅仅因为年纪大一些罢了。


第二就是独挡一面的能力。工作中的项目大都是团队合作。即使你的能力在某一方面有缺陷，只要你善于合作，团队的力量都可以弥补彼此的不足。很早期的只有几个人的创业公司可能是例外。读研究生时候的项目大部分是很独立的。因为毕业论文的署名只有一个人。所以，独立负责一个项目，由始至终做出来，也是会得到锻炼的一个能力。


但是这是利也是弊。其实早一些加入工作，学习如何在一个团队中最大程度发挥自己的优势，也是极其重要的一项技能。长远来说，更重要。但凡事如果能试着站在负责人（做不出来算你的）的角度，先试着独立去思考解决方案，即使是团队作战，也是有益处的。不过只要有心，哪里都能培养这个能力。读研究生可能更能逼着你往这方面走罢了。


第三就是一些做学术的态度和训练。写论文，凡事要有理有据。论文的逻辑思维要强，条理要清晰。你做的本来就是很难懂的东西，再写得乱七八糟，几乎就没有人愿意看了。所以，每一篇发表的论文。其实都是经历过一版又一版的修改，反复思考怎么把一件事说清楚，说的有吸引力。措辞严谨、论点清晰。


相关的就是做 Slides 和 Presentation 的能力。经历各种学术会议演讲和答辩，所以也会在这个方面得到很多锻炼。这和写作能力相似却又不尽相同。尤其是英文。记得刚开始，每次演讲即使 Slides 已经定稿，因为英文不是母语的关系，每次连背带练，都是事先要说的每一句话烂熟于心才上台。现在，基本写好 Slides，心里对演讲也有了谱，很多时候连一遍演练都不需要了。


而这些能力，工作中并不会有那么多的机会让你去练习和培养。


第四个是这一纸文凭，虽然大部分时候没有用，但是多少还是有影响的，尤其是名校的学历。除了可以写上简历，更多时候，它代表的是一种肯定和群体效应。内推、申请工作中多少都会有影响。尤其是中途换专业的，或者本科学校不那么好的，读个好一点学校的研究生，在职场初期还是很有用的。


第五是家庭地位，我和老公技术背景很类似，所以一路在学历、工资、职位、Level、写代码等等方面总是半开玩笑地互相打击对方。若比他学历低，这一局岂不是扳不回来了？（纯粹玩笑，不要认真。）



有得必有失。回过头来，再说读研究生的弊端。


我觉得最大的一点就是时间和机遇。大学毕业是精力和学习能力都还在一个巅峰状态的阶段。研究生一读就是三五年，如果你并不会一路走学术路线，其实更多能力的培养在以后职场中能用到的并不会很多。甚至很多人本科毕业时，上面提及的大部分能力已经都很好了。


一个很明显的例子就是写代码的能力。写代码需要的基础，在本科学习的过程中其实基本都已经涵括了。这个时候，直接到职场的项目中锻炼，比读研究生能得到的锻炼要多得多。研究生更重视的是思维能力，动手能力还在其次。而工作是实枪实弹地练。同样的资质和努力，读三年研究生，肯定没有工作三年的人代码写得好。


再一点就是机遇。这个其实是不可控的了。我的有一些朋友，比我出学校早，去了当年 FB 这样的小公司。然后我们出校毕业了，人家已经乘着 FB 的势财务自由且成为高管了。还有一些小朋友，我毕业的时候，人家本科毕业或者硕士毕业。然后去类似的公司，工资和级别差的也不那么多。所以如果早期想在职场快速发展，我觉得早点出来闯荡，成长的空间更大。


话说回来，我也算很走运的，遇到很好的导师（「我的博士生导师」），研究的领域又都算多少对工作能力有益处的。而且也还算比较快就读完的，一边做毕业设计一边还生了个娃，虽然辛苦到差点放弃。在学校就有寒暑假，也是那些年，把美国很多值得去的地方走了个遍。学校里更有时间和同学在一起混，很多美好的时光，却是工作后很难有的了。我的人生经历因此而更加丰富多彩。所以对于我自己而言，虽然耗掉一些青春，错过一些赚钱的机会，但总的说来，还是很值得的。


最后，要不要读研究生因人而异。总的说来，如果你不打算走学术路线，我觉得弊大于利。三五年的光阴，太珍贵了。有直路走，就不要走弯路。除非弯路上有你想要而直路上没有的东西。
