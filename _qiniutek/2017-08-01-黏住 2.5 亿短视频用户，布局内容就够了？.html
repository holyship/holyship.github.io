艾媒网在近期发布的《 2016-2017 中国短视频市场研究报告》中预计到 2017 年底，短视频用户将达到 2.42 亿人。2016 年，短视频用户人均单日使用时长最高达到 68 分钟，人均单日启用次数最高达到 8.2 次，伴随着移动终端普及和不断降低的带宽成本，用户获取内容的场景不断演化，从内容的生产、消费，到智能分发，这条链路上催生了各种商业机会，也重塑着教育、电商、文娱等各个行业。


面对巨大的市场需求，各大公司都从已经从内容供应链上开始布局，但要在白热化的竞争中突出重围，有多少时间和资本可以给你在产品开发上试错？

对创业者来说，选择合适的短视频解决方案提供商对业务的创新和发展至关重要。在服务过千家直播、点播客户后，七牛云又推出短视频解决方案，基于自身快速上传、转码、鉴黄、存储、分发等强大云端能力，集成拍摄、美颜、特效、滤镜、编辑、合成、本地压缩、上传、播放等客户端组件，形成一站式短视频解决方案。

相比于其他短视频解决方案，七牛云积累了服务千家直播客户的经验，除了全链路均衡的能力，在短视频 SDK 和播放器 SDK 上精耕细作，有明显优势。一天完成集成测试，一周即可完成 App 上线。


短视频 SDK


  1. 短视频 SDK 接口设计灵活、易用。


接口设计规范简洁，提供丰富的参数配置，状态回调，数据回调，客户可基于 SDK 开发


  2.  包体小至 2M，还可进一步拆卸剪裁。


更小的第三方库依赖，更少的代码冗余，更灵活的模块划分，避免客户集成 SDK 后应用变得臃肿


   3.  功能丰富，全面覆盖各种短视频应用场景，保证终端用户可以获得各种新潮的玩法


  4.  性能优越，保证应用的交互顺畅以及使用体验。


高质量的代码，全方位地利用 GPU 加速，尽一切可能减少 CPU 和内存消耗


  5. 丰富的生态链，让专业人做专业的事情，支持各种三方的美颜、滤镜、面部特效、AR特效，满足不同场景下的各种客户需求。


播放器 SDK


播放端性能最直接地影响到用户体验，七牛的旧版本的播放器 SDK 在业内已经有一定的口碑，随着这次短视频解决方案的发布，这次播放器 SDK 也有重大升级，植入了全新自研的跨平台播放内核，功能和性能上都更加出色。


1. 毫秒级首开


一般情况下，播放器首开在 200 ms 左右，甚至更快。


2.包体小，可进一步剪裁


集成后包体增量，iOS 包体 4MB，Android  2.3 MB，去掉软解库，仅需 400KB 左右。


3. 新增加不少特色功能:

  • 变速不变调，可实现更平滑的追帧效果，更少的卡顿率

  • 倍速播放（0.5x，1x，2x，4x 等）

  • DRM 数字版权保护

  • H.265 高效解码

  • 边下边播、播放缓存

  • 支持重连逻辑，断网后可自动重连

  • 使用双 IO 技术更高效地播放 moov 在尾部的 mp4 文件

 4.  三方依赖少，减少产品集成后冲突的可能性。


 5.  跨平台支持，底层基于汇编、C、C++ ，可以支持 iOS、Android、Windows 全平台。


七牛云的上传、转码、鉴黄、存储、分发等强大云端能力，给整个短视频解决方案起到了重要的支撑作用，而成熟完善的客户端 SDK，开放易用的 API ，大大加速了开发者的进度。进入内容领域，创业者除了要建立好自己的内容供应链，更需要稳定可靠的技术来支撑业务创新。凭借多年的富媒体技术沉淀与细致专业的服务水平，在内容时代，七牛云努力成为创业者最佳的技术合伙人。
