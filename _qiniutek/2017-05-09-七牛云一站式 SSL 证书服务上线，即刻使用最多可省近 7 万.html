2017 年 ，随着谷歌、苹果和腾讯对原 HTTP 的相继限制，全站 HTTPS 已经成为了当下趋势，所以安装 SSL 证书成为网站建设中必不可少的一步。


在 2016 年底，七牛云已经与 TrustAsia 联合推出了免费版的 DV 证书，使得个人博客能够零成本完成 HTTPS 的建站工作，这项服务也受到广大开发者的好评。



但是免费版的 DV 证书只解决了少部分个人网站的问题，而综合性的企业网站、金融或电商网站在购买 OV 或 EV 证书时依然面临繁杂的过程。

2017 年 5 月，七牛云与国际顶级机构 Symantec、GeoTrust、TrustAsia 合作，联合推出了三个品牌 11 类证书的一站式申购，覆盖所有类型 SSL 证书，为网站、移动应用提供安全可靠 HTTPS 解决方案，最大限度简化了 SSL 证书申购的流程。



七牛云的一站式 SSL 证书申购服务，选择与 Symantec、GeoTrust、TrustAsia 合作，不仅覆盖了证书的种类，而且确保了证书强度和可靠性，同时个人用户依然能够免费申请 TrustAsia 的 DV 单域名证书，满足用户对于安全的不同需求。


另外，配合七牛云的 CDN 服务，用户可自愿选择将申购的证书一键部署到七牛云 CDN，更快捷地完成网站 HTTP 到 HTTPS 的转换。


使用流程

1.登录七牛云控制台，点击 SSL 证书服务->立即添加。



2.选择证书品牌、种类和年限等。



在付款后将进行申购信息填写等步骤，详情请参见七牛开发者中心-> SSL 证书服务。
