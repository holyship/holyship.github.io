云平台架构、容器与微服务、大数据、AI......随着应用场景的不断复杂，「云」领域的技术人员也要广泛涉猎，习得十八般武艺以防止在不断变化的技术发展中落后于行业。



ECUG 就是你所需要的，一场围绕着「云」端技术深入畅谈的领域盛会，由国内云计算领域大咖许式伟和 Asta 联合发起。


阿里云研究员褚霸、爱因互动 CTO 洪强宁、饿了么 CTO 雪峰、PingCAP CEO 刘奇、七牛云技术总监陈超、Docker 专家孙宏亮等。


深度分享，为你而来

一场代表着国内云领域风向的技术盛宴


时间：12 月 16 日 － 17 日

地点：上海宝丰联大酒店



ECUG Con


ECUG 全称为 Effective Cloud User Group（实效云计算用户组），由七牛云 CEO 许式伟发起，集结了一批具有高端视角并仍醉心于技术本身的同仁，共同关注云计算前沿技术的最新成果和分布式开发、运维的最佳实践。从 2007 年至今，ECUG 社区每年都会组织一场全国性的 ECUG Con 大会，期间众多领域内权威技术大神纷纷参与，他们或许在大众技术会议已经鲜少露面，但会在 ECUG Con 畅谈心得，共同奉献一场半公开的高规格技术分享盛会。

详细议题介绍


许式伟

《继占领云计算产业后，Go 的下一个战场——游戏产业》


Go 语言在桌面侧的战场已经开启，第一个进军的领域将会是游戏产业。预测如同快速占领云计算市场那样， Go 语言将以惊人的速度侵占游戏开发的市场。本选题将会对这其中的原因和趋势进行详细阐述，并且会提供一个用 Go 制作游戏引擎的案例，该引擎可完美支持多个终端平台，包括手机端（Android/iOS）、 PC 端（Windows/Mac/Linux）以及 Web 端。


张雪峰

《饿了么混合云架构探索》


为支撑高速增长的业务需求，饿了么运维团队建立起一个包含 25K＋ 服务器节点、承载每日总业务数据量超 12P 的大型混合云平台。在本话题中，饿了么 CTO 张雪峰会首度讲解其全国领先的混合云平台架构的具体实现细节：

  • 混合云：4 个自有 IDC；

  • 多活架构：2 个自有 IDC zone（已上线），2 个 Cloud zone（建设中）；

  • Auto-Scaling 计划、Cloud Native 计划、Serverless 尝试（进行中）；

  • 自有 IDC VM Cluster over ZStack，自有 IDC Docker Cluster over 自研调度系统（Mesos 二次开发）；

  • OLTP、OLAP 混部计划，YARN、ZStack、Mesos 相互适配。


谢孟军

《如何用数据分析颠覆传统的工程制造》


云计算和大数据的快速发展对于传统的各种业务产生了深刻的改变，工业制造由于其生产环境的复杂性、流程的高协作性，以及对于数据指标的精细化要求，成为大数据应用的一个典型场景。积梦科技 CEO、前苹果数据化生产负责人谢孟军将在本议题中分享围智能制造的应用场景以及大数据分析的深度解析，结合实际案例深入浅出的讲解了如何通过集合工业互联网、云计算、大数据在生产制造全流程中，实现制造过程中的数字化控制、状态信息实时监测和自适应控制，其设计思路和技术工具对于其他场景也具有可借鉴意义。


陈超

《万亿级大数据平台的架构设计与演进实践》


Pandora 是七牛自研的一套大数据平台， 目前每天有超过 200T 的增量数据进入 Pandora，每天参与计算的数据量超过 2P。 全平台基于 Go + Scala 语言编写，定制了多个开源项目，基于 Kafka 自研了数据流入、导出以及多集群处理等多个关键组件。本议题将深度揭秘七牛大数据平台的技术选型，组件设计、系统调优及架构演进。此外也会分享在演进过程中遇到的「坑」，以及对应的解决方案。


洪强宁

《Chatbots 中对话式交互系统的分析与应用》


一直以来，人机交互方式在发生着不断的变化，从命令行交互，GUI 交互，GUI＋交互，直到现在的对话交互（CUI）。之前的传统交互都是人在适配机器，而 CUI 则实现了机器来适配人。本议题将分析常用的对话交互技术框架，并通过实践来分享 chatbots 系统平台的技术和挑战。


刘奇

《TiDB 实现过程中的技术选型》


如今实现一个分布式数据库的复杂度已经不低于实现一个操作系统，实现的过程中会面临大量的技术抉择，这次和大家聊聊 TiDB 实现过程中的一些技术考量，也许这些比源码更有意思，毕竟源码只是众多技术选择的一个画像，其它的画像则被隐去。这其中的思考，纠结，验证对团队而言也是前所未有的，如何抵制诱惑，如何在平衡各种技术选择的优缺点，这些属于没有对错的问题往往需要大量的思考。本次议题将分享这些曾令 PingCAP CEO 刘奇辗转反侧的问题。


吴祖洋

《对于创业公司云平台架构的经验与思考》


云计算发展到今天，已经覆盖到多种业务层面，「各类场景皆云化」。云服务器，云存储，云关系数据库，云推送等云服务和产品极大地降低了创业公司的技术门坎，使得创业公司可以快速起步。作为创业公司的技术决策者，是否任何方面都要向云靠拢呢？有没有应该远离云的地方？懒投资 CTO 吴祖洋在本话题中，将结合个人创业心得，分享云架构的经验和思考。


孙宏亮

《阿里巴巴容器技术 Pouch 解析》


容器生态的发展此起彼伏，不管是容器引擎 Docker，还是编排系统 Kubernetes，都为行业创造了很多价值。阿里巴巴的容器技术 Pouch 也在同时代诞生，Pouch 为何物，又能给行业带来什么？此次我将带领大家由浅入深，了解 Pouch 背后的故事。议题分享介绍内容包含：阿里巴巴 Pouch 的使用现状；Pouch 的技术优势；以及 Pouch 的开源进展。。


倪朋飞

《Kubernetes Cloud Provider 演进以及在 Azure 上的应用》


作为最流行的开源容器集群管理系统，Kubernetes 已经被大量用在生产环境中。今年 2 月，微软正式宣布 Kubernetes 已经开始在其 Azure Container Service 中正式向用户开放。微软软件工程师，Kubernetes Maintainer 倪朋飞将在本议题中介绍 K8s 在 Azure 上的应用实践。主要内容包括：

  1. Kubernetes Cloud Provider 简介；

  2. Cloud Provider 的由来和演进；

  3. Azure 实践及挑战。


乔捷

《云游戏的架构设计和技术实现》


云游戏是以云计算为基础的游戏方式，用户通过各种终端设备（PC、机顶盒、平板、手机等）接入云游戏平台，让大型游戏运行在服务器端，实时接受终端用户的输入操作，并将渲染后的游戏声音和画面通过流媒体实时传送给终端用户，免去了专门的游戏设备和下载安装过程，让游戏像视频点播一样方便的玩。


云游戏的技术实现存在着很多方面的挑战，本议题将在云游戏工程化实际的基础上，对目前云游戏的架构和技术实现做一次全面的分享。


王晓波

《Serverless 架构：享受纯粹的编程乐趣》


同程旅游近些年中技术平台进行了微服务化改造，解决了原来许多架构上的问题（耦合、扩展、熔断、降级、调用治理、多语言等）。但这些问题被解决掉后我们进行了更进一步的升级，将 Serverless 想法引入架构中并开发上线了一个基于我们私有云的 Serverless 平台，现在整个同程的 Web 站点和部分变化快的轻型都这个 Serverless 之上开发。本主题将分享同程在 Serverless 架构实践过程中的一些坑与收获。


陈迪豪

《AI 算法实现与云平台应用》


机器学习的基础架构包含了机器学习算法、机器学习类库以及机器学习平台等多个层次的内容。根据业务的需求，我们可以选择特定的领域进行深入研究和二次开发。本议题会重点介绍主要会介绍机器学习底层原理和工程实现方面的内容，以及如何借助开源的 Caffe、TensorFlow 或者 AWS、Google CloudML 等可以实现诸多业务应用。
