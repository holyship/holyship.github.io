PhD这屁早放早清净

文丨飞总  图丨源自网络

本文有毒，PhD慎入

0

0

“飞总，听说你是PhD。请问我怎么样才能在工作中发挥PhD的优势呢？” 小A是一次聚会认识的。毕业入职场一年的PhD。书香门第加图灵奖获得者的学生的她，问这问题的时候有说不出来的疲倦感。我也一时眼前恍惚，仿佛一群大雁迁徙飞过，一会排成了S形，一会排成了B形。


“恩，多吃蔬菜，清清肠胃。PhD这屁，放了就好了。”不知道怎么的，话从我的嘴巴里出来。对面的小A一脸的懵逼。

1

时间依稀回到多年前。我坐在会客椅上，对面是个美国中年白人,眼睛炯炯有神。今天是我入职3个月，第一次和skip one:one.


"请教大领导，我作为一个PhD，工作中怎么样才能发挥自己的优势啊？" 我诚惶诚恐的问。


“PhD好啊。真羡慕你。我读书差，读完本科就出来混江湖了。从小公司一路做过来。到现在还是觉得自己像个文盲啊。Dr.XX,公司就需要你这样有知识有文化的PhD啊。” 大领导打开了水闸侃侃而谈。我心里一种自豪感油然而生。


若干年后，这个见面就叫我Dr.的大领导一巴掌拍过来，把在升职道路上的我拍得稀巴烂。

0

2

PhD，Permanent Head Damage的简称。通常在美国需要5年以上的时间，发表Paper若干才能获得。永久性脑残属二级残废，不影响正常吃饭睡觉，但是药不能停。


随着经济的发展，互联网事业的腾飞，越来越多的PhD混迹到IT互联网行业里面来，有研究猿，有数据科学家，还有程序猿，攻城狮。


经过永久性脑残训练的PhD出来大致分三种情况。第一种是顺利变异，化身成为叫兽或者是大公司的研究猿。此后要做的事情和此前要做的事情并无太多差别，发表更多的Paper是主业。此类PhD是异类，少数，不在讨论之列。


第二类和第三类都成为了程序猿，攻城狮，或者数据科学家。所不同的是第二类的PhD，自己读书期间的知识对从事的工作还有点价值。而第三类则从事完全无关的工作。如果按照这个标准来判断，本人属于2.5.

0

3

PhD从事这行职业，劣势非常的明显。和同龄人比，对方如果是本科毕业已经工作5年，如果是硕士毕业也已经工作3年。在IT行业里面，三五年的先发优势可以让对方比你更会写feature更知道industry的工作实际情况。而刚毕业的你，既不懂source control，也不会写unit test，更不知道怎么样把一个feature design好。有些公司为了照顾脑残患者的感受，对PhD进来还多加一级。


所有的这些加在一起的结果是，从manager的角度来讲，PhD们性价比不高。更为重要的，无数的PhD，比如我，觉得拿了PhD以后就有优势。在那个entry level的你，有个鬼优势。干活慢，自视甚高，工资贵。如果过上一两年，没有潜力发挥出来。那就只能做渣渣了。

0

4

当然PhD其实也是有优势的，这种优势的具体概括来说就是纸上谈兵的能力。大凡PhD都写过一些论文，英文叫paper，和厕纸的拼写差不多。只要PhD的论文不是导师写的，一般来说写作能力表达能力还在。


实际情况是，作为一个entry level的你，其实不需要这些技能，需要的是能够立刻出活的技能。如果底层码砖头做不好，总想着发挥眼高手低的纸上谈兵能力的话，没有人会给你这个机会。


PhD的另外一个专长是对领域的了解和文献的查找能力。这个要求PhD从事的工作岗位不是简单的码砖，最好还是和自己的研究方向有点关系。可遇不可求。


所以实际一点呢。进到industry来，就把PhD当个屁放掉。其实paper(论文)和paper(手纸)的差别就如同dog和god一样。从左边看还是从右边看的问题。在industry里面，基本的干活能力是敲门砖。如果想要获得更多的机会，先把基本的干活能力练习好。别去扯什么PhD发挥优势的问题。PhD唯一的优势就是药不能停。早点当个屁放掉最实际。

0

“请问是飞总吗?” 一声洪亮的声音传来。

“您是哪位？”

“我是小B啊。就是上次爬山认识的。听说您是PhD。我能约您出来吃个饭吗？”

“你也是PhD吧？刚工作半年？”

“是啊是啊，飞总就是聪明。我想向您请教一下怎么发挥PhD的优势。”

我眼前一黑，仿佛一群大雁飞过，一会排成了S形，一会排成了B形。
