修Bug的罪与罚（2）

文丨飞总  图丨源自网络

本故事纯属虚构，如有雷同，纯属巧合。

距离上次写小说有10多年了，文笔已经烂了。距离我看这本书，就更久远了。

后台很多人问故事到底想说啥，只能说一万个人心里有一万个哈姆雷特，我讲故事，就不评论了。

人这种卑鄙的东西，什么都会习惯的

          ----陀思妥耶夫斯基

0

1

时间过得特别快，才一瞬间的感觉，店里就要打烊了。小关已经付了账单早早的离开了。


老张开车回家。车库的门响起的时候，仿佛一切照旧。老婆忙着哄孩子睡觉。今天要晚点回来，和同事有饭局，这是早就打电话告诉老婆的。


看着家里的灯光，老张觉得有点恍惚，望着老婆，终究还是一句话都没说出来。这晚睡的时候，老张觉得仿佛经过了一个世纪。老张梦到了自己高中的时候，在读郁达夫的诗，生死中年两不堪，生非容易死非甘。当时还是小小张的自己完全不能理解这诗什么意思。问班主任的时候，班主任的眼神有一丝迟疑，之后就让小小张爱干什么干什么去了。


老张梦到了自己的大学，认识了从上海来的老婆。老张至今还记得是闸北区的，不是黄埔区的。有车有房有后院草坪，还有新鲜空气，这就是小张当年的美国梦。老张实现了，除了还有一直都没付完的贷款。



0

2

老张依稀的记得自己公司，哦不，前公司的CEO，那个人人讨厌的胖子在喊 good old days。是啊，old days，老张感叹了一句，醒了。这个梦里的记忆既清晰又糊涂，好像记得很多，又完全不记得了。


老张爬起床，老婆准备好的饭盒早就在冰箱里放着了。通常老张都会走得比较早一点。这样可以避开交通高峰，节省一些时间。老张还是小张的时候，就有一句的至理名言，时间就是生命。


老张习惯性的吃了早饭，开了车库的门就往公司开。这条路开了10多年，一切都很习惯。什么地方要左转，什么地方要右转。哪里警察会躲起来抓人，哪里可以放心大胆的超速。这些东西都成了老张的本能。


车停到地下停车库，关门拿饭盒，然后走到电梯口。刷卡准备上电梯。卡滴的一声没刷卡。老张顿时脸色刷的一下白了。才想起来，原来自己已经被解雇了。

0

3

老张恍恍惚惚的从地下车场的走到走上来，看看熟悉的办公室。这个办公室里二楼靠湖的那边，就是老张一坐好多年的地方。那个湖大有来历。老张还是小张的时候，每次重大产品发布，就会看到创始人跳湖庆祝。这样的事情随着2000年的泡沫破灭公司越来越大就渐渐的没有了。所以每次新人加入的时候，老张就忆苦思甜回忆当年看创始人一次次跳湖的壮举。而旁边的小年轻们从开始的张哥到后来的张叔，看着老张一脸的崇拜。


老张有点不知道自己要干什么了。早上来公司，晚上回家，周末陪娃，偶尔休假，几乎已经成为了他生活的全部。偶尔他也会想，觉得自己的生活单调了点。但是为了美利坚的新鲜空气嘛。他可能不知道，若干年以后，有个小留学生就仰慕美国的新鲜空气，而远渡重洋去马里兰大学求。又在毕业以后因为在毕业典礼上以新鲜空气为题做演讲，掀起轩然大波。


老张决定就在这里走走。看看熟悉的风景。以前经常听到别人说，人老了就会念旧。怀念这个怀念那个的。但是大家都叫他老张了，老张却从来没觉得有什么需要怀念的。今天看着这周围的楼，就突然想到了早上的梦，梦里自己高中时候读的诗：生死中年两不堪，生非容易死非甘。老张老了。

0

4

对面走过来一个老头，头发有一点白了，但是腰杆还是很坚挺。看到老张的时候一个劲的挥手。老张看出来了，这是公司的院士Jose。Jose院士在这个大组工作的时间比老张还长，是真正的元老。这些年来除了在台上能看到他以外，也就是早上很早的时候能看他围着公司的草坪散步。


Jose是老张到这个组里来的第一任领导。当时老张见到这个名字就读了觉得这个人名字有意思，居然是酒色的读音。酒色容易掏空身体啊。结果老张被周围人狠狠的鄙视了。因为Jose这个名字里面的J发H的音。


Jose是这个项目的早期开发人员，不知道是希腊来的还是丹麦来的，总之是欧洲某个国家过来的。老张加入的时候，作为一个领导，Jose已经小有名气了。在老张入职之前，Jose重新构架了这个产品的存储模式，在能存更多的数据的情况下，还用更少的磁盘空间。这在当时显得非常的牛逼。

0

5

这次构架的改动总体来说是成功的，除了一点点小小的问题。就像当年经典物理是成功的，除了小小的不和谐。学物理的都知道，这些小小的不和谐，彻底摧毁了经典力学的框架，从此我们的世界进入了不确定的时代。量子力学告诉我们，这个世界是掷骰子的。


Jose的构架当然没有经典物理学的牛顿或者麦克斯韦那么伟大，而一点小小的问题也无非是存了文档以后再打开，里面的图给弄丢了而已。只是no zuo no die，这个东西宣传的好，大量新老用户都纷纷购买。结果好端端的把别人辛辛苦苦画的图都弄丢了。这个问题还不容易重现，所以修了几次都没真正修好，一下子就掀起来不少波澜。


当领导的Jose因为这个伤透了脑子。而当时还是小张的老张颇有初生牛犊不怕虎的味道。辛苦钻研若干日以后，终于信心满满的确信Jose的构架上有致命的缺陷。Jose听完老张的话，半饷没发声，最后问一句，小张，你说怎么修呢？


望着领导信任的眼神，于是小张又通宵达旦的奋斗，终于找到了一个不错的解决方案，然后又和Jose一起加班加点的修好了bug。

0

6

由于应对及时，Jose因为这个架构获得了很多的赞誉，也当之无愧的成为了他在这个公司往上升的资本。当时的小张虽然没有参与最初的开发，但是因为在修补这些bug上的不懈努力，对系统也有了别人难以企及的了解。Jose为此拿了个公司的奖。


那年的review，Jose给了小张一个合格的评价。Jose对小张说，小张啊，你是我见过的年轻人里面最有前途的人之一。好好干，以后前途大大的。


小张一脸懵逼的问老板Jose，为什么我和你一起努力把构架改正确了，最后我只是一个合格呢？Jose对小张说，作为一个新人进到这样核心的一个组，第一年能够拿到合格就很不容易了。大部分新人第一年都只能拿到不合格的两档里面的一档。你的合格表现非常的优秀啊。好好干，这个评价非常的高。

0

7

Jose的设计能力怎么样，老张不好判断，但是Jose的忽悠能力是非常的强。Jose后续又做了几次设计，然后又搞出一些这样或者那样的bug来。而作为修bug的小张，不管墙怎么坑坑洼洼，都有把墙抹平的能力。Jose一路给小张合格的评价，到后来，终于给了超越合格的档次。并且在两年后给小张升了一级。小张的升级速度，并没有比其他人快多少。


Jose不像其他的院士一样装逼，每次见到熟悉的人都很热情的打招呼。见到老张的时候，都会忆苦思甜当年和小张一起修各种bug的经历。每每这个时候Jose就会大大的夸一下自己的架构是如此的精妙。顺便提一句，小张当年修bug的手脚也很利索。那个时候老张也就会陪个笑脸。


看着老张走过来，Jose上来握住老张的手，说Mr. Zhang，很高兴新的一天又见到你这么早来啊。今天出太阳了，西雅图这鬼地方难得的晴天啊。看样子他还不知道老张已经被开掉了。老张望着Jose，有种说不出来的感觉。清晨的太阳照射过来，照到老张的眼里。老张的眼睛开始流泪，看着对面的Jose不知道为什么，觉得这和蔼的笑容，说不出的不和谐。
