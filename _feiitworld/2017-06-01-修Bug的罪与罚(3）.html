修Bug的罪与罚（3）

文丨飞总  图丨源自网络

昨天稿子修改完结果发出来却是老版，不知道企鹅出了什么问题，只好删之重新发。


人这种卑鄙的东西，什么都会习惯的----陀思妥耶夫斯基。

0

1

老张很快的和Jose挥手告别。Jose给老张来了一个热情的拥抱。老张不清楚Jose是不是知道自己被裁员了。但是Jose没有提，他也没有说。


老张走进车库，把车开出来。阳光直射进车窗的感觉让老张不适应。老张心里知道趁着早上保安和同事都没有上班悄悄的走，对自己是最合适的。万一晚一点被公司保安纠缠上，又被同事，应该是前同事看到，就不好了。


公司第一次裁人，政策算厚道。裁员方案里每个人给了两周薪水，此外，每在公司多工作两年加一周薪水。加上还没有用掉的假期，老张一算下来，公司给了两个月钱白拿。只要老张在未来两个月内找到工作，那么生活就会一切步入正轨。


一时间老张也不知道自己应该往哪里去。老张突然想起来，自己的那台电脑已经被回收了。心里想着，要不去买一台差不多的吧，鬼使神差的，就开到了BestBuy。

0

2

BestBuy作为美国最大的电器公司，老张早年光顾的也不少。可惜经过这些风雨，其他大一点的店比如CompUsa以及CircuitCity都倒闭了。唯独某家卖书起家的越发的红火起来，生意越做花样越多，电脑也卖。作为这个历史悠久的公司的前员工，老张的内心还是充满了优越感，对于湖对岸的那家卖书的，颇有几分不屑一顾。


随着时间的推移，老张的脑子清醒起来一些。昨天那个漫长的梦多了几分记忆。老张记得自己还是小小张的时候，从高考的千军万马里杀出来，在小小的县城里拔了头筹，给本是农民出身的父母带来了很多荣耀。上了电视，又受到了市长的接见。Old good days啊。等小小张上了大学，进了美国训练中心，在那一人一口吃成胖子的城市待上五年，又随波逐流的出国，去了猪的数量比人还多的某美国州立大学村。然后就是工作，一晃又是十余年。


老张太太姓吴，大城市出身，是校友，小两届。出国前老张并不认识。两人的结识颇有戏剧性。校友在海外都很团结，每年校友都会来接新生。老张当年开着自己那辆2000块钱买的16万迈的假美丽，从机场接来了老张现在的老婆，还有她男朋友。一年后她男朋友又接来了师妹。


又是一年，在男朋友师妹的帮助下，当时还不是老张老婆小吴找到了东部一家和师妹家里有业务往来的公司的实习。又是三个月，等实习回来的时候，发现自己的男朋友已经爬到师妹床上去了。等小吴找到老张让帮忙评理的时候，老张就去把那个男的揍了一顿。为此也把警察招来了。好在没有人希望这事情闹得沸沸扬扬，老张也不知道自己是不是趁虚而入，总而言之，言而总之的，师妹成了老张的女朋友，然后迅速在美国就领了证。

0

3

走进BestBuy的时候，里面冷冷清清的。不知道是不是因为经济不好了，还是湖对岸的卖书的开的网店越来越声势浩大了。疯传的实体经济被虚拟经济取代，的确不是空穴来风。


作为早于互联网时代就开始写程序的老张，眼里的互联网还是那个雅虎作为互联网第一股登陆纳斯达克上市的互联网。老张很习惯于用terminal登陆bbs，打开浏览器只是为了通过雅虎看看新闻收个邮件。至于这些年里越来越流行的社交网络，那个东西在老张眼里不过就是一个bbs的HTML版，不成气候。老张一边逛着商场，一边又开始了回忆。


随着小张在组里工作的深入，Jose也越来越多的依赖小张去修那些一般人修不了的bug。小张修bug的能力，连他领导的领导都很知情，非常的远近闻名。


又是一年，因为工作的需要Jose组里已经扩容一倍了。Jose终于招来了第二个中国人汪女士。汪女士说话声音细细的，见到谁都很有礼貌。听说小张非常擅长修bug，汪女士就常粘着小张请教问题，整天张老师张老师的。小张回答的格外的卖力。

0

4

随着接触深入，小张渐渐知道了汪女士和她老婆是地道的老乡，来自同一大城市的另外一个区。家里是书香门第，又在普林斯顿大学读的博士，师从有计算机界诺贝尔奖的图灵奖获得者，研究的是量子计算。汪女士家里有一儿一女，先生姓罗，在那个被从本公司拆分出去的著名旅游公司做构架师。虽说年岁稍长，然而上天厚爱，小张竟无法从其身上找到岁月留下的痕迹。


老张很快的就看到了一台他想要的电脑，正是公司给他配的小黑。IBM这个古老而又有地位的品牌出的ThinkPad。只是如今这个牌子已经叫做Lenovo了。老张一直认为IBM抛弃自己精心设计的商务电脑的品牌，转手卖给Lenovo是一种背叛，一种精神上对广大程序员们的背叛。作为一个ThinkPad的忠实拥护者，每次摸着那个小红帽，再看着黝黑有质感的机器做工，老张就会油然升起对IBM的崇拜。IBM做的硬件，结合老张精心开发和维护的软件，在办公场所，是多么高端又完美的一种组合啊。对，那就叫职业化。

0

5

尽管老张知道，ThinkPad的品牌归Lenovo的那一天起，这个牌子总有一天会沦为平庸，他还是没有办法选择那个本来就平庸的组装机戴尔，或者那个笨重的惠普。日本鬼子里有些好牌子，但是老张是爱国的。虽然老张也不知道自己爱的是中国还是美国，但是不管怎么样，起码不能用小日本的牌子。至于那个装逼的苹果，华而不实，老张很快就略过了。只是老张不知道是装糊涂还是真的不知道，那个所谓的小黑IBM，其实也是IBM在日本的研究院设计的。


看着电脑发呆的时候，老张的回忆从汪女士身上跳回到了他的老婆。人不同命不同，老张想着自己，他老婆，汪女士，罗先生，还有Jose，形形色色的。果然是人不同命不同啊。老张记得自己还是学生的时候背着若干瓶Costco买的鱼油维生素去拜访老丈人一家。老丈人目光斜向远处看着他，过了许久之后从鼻孔里冒出很长的一声哼，让当时的小张顿时有平地一声雷的感觉。


之后在那场众多亲戚参加的聚餐宴上，作为准成员的小张，在角落里一声不吭的目睹了一场大戏。来自上等区的大姨在数落小张老婆的爹，数落着自己妹妹从上等区下嫁到下等区，对她老爹来说是多大的一种福分。小张偷偷问老婆什么是上等区什么是下等区，被老婆狠狠的捏了一下大腿。旁边小舅子望着老张说一句，外地人看什么看。小张后来偷偷去打听，才明白典故的由来。小张不由的明悟，人和人果然是有阶级的。


拿着那台碳素外壳的轻薄小黑笔记本去结账的时候，老张摸了又摸。这台电脑是公司最新给配的。虽然是Lenovo收购以后才出来的，但是做工精良，继承了ThinkPad X系列一贯的高质量高品质。配上老张这样的商业软件主力开发人员，资深工程师，真的是很配。老张心里又在做梦了。

0

6

随着认识的深入，汪女士热忱邀请小张夫妇聚餐。于是小张夫妇和汪姓女子夫妇见面吃了顿饭。老乡见老乡，两眼泪汪汪。更何况在海外这个地方。只是老婆望向汪姓女子的眼神里总有一点说不清楚的感觉。小张心里想，上等区和下等去，果然是两个世界。这话他没敢说出来。


1023.99，屏幕上显示出这个价格来。老张有的时候觉得美国人特别的傻逼。什么都喜欢放个.99.难道少1分钱就会吸引更多的人买吗？老张扒了扒自己的口袋。掏出一个信封来，是1000块，再一摸，还有24.收银员一敲键盘的时候，结账的屏幕上跳出来，9.5%的税。傻逼美国佬，老张又骂了一句。在美国，商品有税，吃饭还有消费。明明美国人数学那么差，美国的服务体系又强迫美国人每次购物算数字。每次老张看到这些的时候，都有种浑身不自在的感觉。


钱不够了，老张只好掏出自己的卡来。卡在手的时候，又犹豫了。老张心里有苦。当年小张找到这个工作的时候，老婆还在异地要读书差两年毕业。有一次小张一个人在这边觉得寂寞，就在狐朋狗友的怂恿下去了唐人街按摩放松什么的，然后掏了卡付了钱。


小张不知道的是，远在猪比人还多的州的老婆，每天都在认真查看小张的银行账户，信用卡消费明细。经过人肉数据挖掘，老婆确信小张多半干了什么见不得人的事情，本着坦白从宽抗拒从严的态度，在严格逼问下小张很快就坦白了。这件事情的后果是从此老张被老婆收走了财权。而老婆也因为当时小张的闹腾，草草的结束了学业拿了个硕士，迅速的来到当时小张的身边看着小张。这一看就是10多年，从小张看成老张。老张也从未得到老婆批准之前花超过50块的钱。


提着电脑出来的时候，老张的心还是在抖的。又一想，没有电脑怎么能够假装上班的同时准备找工作呢？老张的车开停在了一个星巴克咖啡店。咖啡店里人头攒动，又有免费的网络，是个写简历准备面试的好地方。老张打算以后就每天假装上班，来星巴克准备材料和面试。等到新工作一找到，一切都会重回轨道的。老张心里如是说。
