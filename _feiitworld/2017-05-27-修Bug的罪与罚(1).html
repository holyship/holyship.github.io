0

1

小关进来打招呼的时候，老张还是一脸的恍惚。仿佛是过了一瞬间，又像是过了一个漫长的世纪，老张脸上的肌肉，终于颤动了几下，挤出一丝笑。这顿临时凑上的饭吃起来，竟有种别样的滋味。


早上的那一幕还历历在目。供职于美国著名世界500强的软件巨无霸核心部门之一的老张，和很多其他人被叫到了一个办公室里。里面人头攒动，不少是部门的老人，认识和工作有10余年了。


最近的公司气氛有点紧张。美国经济不好，外界一直流传公司要裁员。但是老张对自己的公司有信心。2000年那么大的泡沫，公司都没有裁员一个人。如今的这点小风小浪算什么。


HR板着扑克脸严肃的宣讲。公司不容易要重组，你们这些都被裁了。下面是裁员以后的各种福利。然后你们可以签字拿着材料走人了。在办公室的各位的电脑登陆账号，email等等都已经被禁了。所以收拾收拾回家去吧。老张望着这一切一脸的懵逼。

0

2

恍惚间，老张回到了一年前，不记得到底是第几任领导在年度review的时候，握着老张的手说，感谢你为公司服务10年了。公司很快就会给你发个10年的纪念碑。另外祝贺你，根据你对公司一贯以来做出的杰出贡献，公司决定晋升你为Senior Engineer。


听到这个消息的时候老张老泪纵横，内心是喜悦的。辛辛苦苦奋斗这么多年，终于获得公司的认可，从此以后就是资深工程师了。升职加薪的年度考评的纸上写着老张从此以后能多拿多少钱。数字其实不重要了。重要的是，公司对自己这么多年来的认可。


人群里到处都是懵逼的状态，有美国人，有中东人，有印度人，也少不了中国人。HR很快宣讲完了，人群像鸭子一样的挪出去，老张也趋之若鹜。回到家的时候仿佛觉得自己做了一场梦，梦醒以后明天日子还是照常的过。接到小关约饭的电话，老张还是有些茫然。

0

3

小关其实也不小，进公司10年了。最开始的时候跟着老张干。后来组大了，做了几次重组以后，就去了隔壁的隔壁组。从行政上来说，和老张的交集仅限于在director往上走。


小关其实也不能叫小关了，人家已经是公司的首席工程师，做构架师，底下队伍浩浩荡荡的几十人。这几年来，小关是越发的有名起来。只是小关一直对老张很是尊敬。而老张则从心里看不起小关。这个小关做事情不牢靠啊。


老张还是小张的时候就比较性格内向一些。有中国人的一贯传统优势，做事情很靠谱。交给老张的功能模块，老张的程序写的不但精致而且测的异常仔细。老张不但对自己的程序很敬业，也非常乐于助人的帮助大家把bug找出来，修掉。所以只要是有老张在的team里，老张就是定海神针。从组里release出去的产品，从未出过大的幺蛾子。


小关刚进组的时候，老张是mentor，也就是所谓的导师了。小关年轻机灵特别好动，什么都敢去摸一下。什么地方的code都敢去加功能。但是按照老张的说法，做事毛糙而不精致，到处都是bug。小关很敬重老张。发自内心的觉得老张就像扫地僧一样，无所不知无所不能。所以这两位虽然渐行渐远，却也保持了某种奇怪的默契。

0

4

小关的一举成名来源于他离开老张的组以后。新组的人上线了一个新功能，里面就有小关的功劳。但是在公司CVP即将要在公开场合演示的前一个小时，东西坏了。现场的工作人员非常的着急却毫无办法。公司开发了那个功能的工程师正在休假。测试人员也不能解释为什么这个东西就不工作了。


小关临危自告奋勇的上去。以他从老张那边学到的对系统的了解和很快的动手能力，迅速的找出了问题。小关心里是有点虚的。这个问题表面上看，是新同事开发的功能的问题，但是实际上是他自己半年前做的某些东西有欠考虑，从而导致了多米诺骨牌效应，一下子都倒了。


但是小关不愧是有急智的人。他迅速的黑了一下代码，临时让事情跑起来。现场工作人员顿时非常的满意，CVP也很满意。事后，小关又认认真真的写了一篇事故总结，把他认为应该要突出的地方都突出了，认为应该轻描淡写的地方则一笔带过。索性在新的组里，小关是权威，其他人都没他懂。于是经过这件事情以后，小关很快坐上了火箭，半年一级的升到了senior。

0

5

老张对面的小关看着老张说：“老张啊，事情我都听说了。公司挺操蛋的。相信我，以你的能力，肯定能找到工作的。我有些朋友在各大IT公司。我会帮助你推荐，简历给我一份。”

老张看着小关，想哭却又哭不出来，一脸苦瓜脸的对小关说：“谢谢谢谢。患难见真情啊。我这被开了以后，只有你主动找我了。以前那些找我帮忙问我系统问题的人，一个都没理我。”


小关看着老张，突然神情严肃的说：“老张啊，我们认识也都快要有10年了。我是非常敬重你的业务水平的。但是有些东西，去新公司以后你得改改。不然的话，搞不定哪天你又被开了。”


小关继续说：“我知道你打心底里看不起我。我做事急也不牢靠，没有好好测系统。你做什么事情都精雕细琢。但是你有没有想过，为什么你这么艰难的升到senior。你做的这些东西，高质量的代码，我们懂技术的都知道。上面的director vp们都知道吗？”


老张的脸色有点不好看起来。小关接着说：“我知道这话让你不开心。但是你把事情都做得这样好了，修bug修的都没有bug可以修了，为什么还没升级呢？ 哥，好好想想吧。”

0

6

看着老张近乎奔溃的脸，小关没有继续说下去了。小关拍拍老张的肩膀说：“简历给我一份吧，我找朋友帮你内推。”


想了想，又掏出一个信封，里面是1000块钱，递给老张说:"这钱你先拿着，救个急。"


拍了拍老张的肩膀，小关走了。走没多久，想了想，又走回来，从口袋里掏出24块钱递给老张说：“还是凑个整吧。”


桌子前只留下瘫坐在椅子上的老张。一脸的懵逼。
