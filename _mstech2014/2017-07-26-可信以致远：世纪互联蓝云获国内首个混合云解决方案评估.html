2017年7月25日，在北京国际会议中心举行的第四届"2017可信云大会"上，由世纪互联运营的Microsoft Azure凭借Azure混合云解决方案通过了业界首次评出的"混合云解决方案评估"，同时还凭借优秀的混合云运维能力，成为唯一一个获得了大会颁发的"可信云技术创新奖——混合云奖"。


首次颁发混合云评估，走在行业前列


可信云大会是由工业和信息化部指导，中国信息通信研究院、中国通信标准化协会主办、数据中心联盟、云计算开源产业联盟和互联网医疗健康联盟承办的面向可信云服务评估的峰会。可信云大会对于推动我国国内可信云标准建设、云计算健康发展、计算技术提升及云计算产业走出国门都起到了极其重要的作用。此次世纪互联蓝云获得的Azure混合云解决方案评估，是可信云大会首次颁发的混合云评估，证明Azure在混合云服务能力方面走在了行业的前列。

大会给Azure混合云解决方案以高度评价：可以将公有云与私有云之间无缝对接，印证了Azure具有独立提供给客户混合云完整解决方案的技术实力。开发者还可以充分利用存储、备份和恢复功能选项，在提高效率的同时降低成本。

此外，在创新性方面，Microsoft Azure混合云解决方案也有独特的优势：

加速时间价值：利用简便安装、就地升级和自动化工作流时间快速实现价值。

高效操作：利用一键式特性和简单控制快速完成任务。

支持多系统：管理并监控Linux、Hyper-V 以及VMware 等多相开放系统。

服务器和桌面：部署和管理Windows Server 2016 和 Windows 10 —— 配置、安全和合规性。

云整合：利用完全一体化操作管理套件获得数据和应用程序的可见性和控制。 


可信以致远：只有合规才能谈安全


近年来，国内云计算产业发展迅猛，产业环境日益完善，产业规模保持高速增长。但在云计算产业"井喷"的背后也存在着诸如数据安全、服务质量、权益保障等多种问题。因此严格遵循国际和国内法律法规、获得权威可信评估，就成为国内云服务商发展的首要任务，只有"合规"、"可信"，才能谈"安全"。

由世纪互联运营的 Microsoft Azure 和 Office 365，作为首个落地中国市场的国际公有云服务，在采用业界领先的微软云计算技术为客户提供可信赖云服务的同时，严格遵循国际和国内法律法规和标准规定，获得多项权威评估，同时秉承安全性、隐私保护、合规性及透明度四项原则。

在由数据中心联盟发起的可信云服务评估评选中，由世纪互联运营的，共累计获得过10项可信云评估，包括：云主机、云存储、数据库、云引擎、全网负载均衡、企业级电子邮件、文件共享、联机会议、云备份，以及本次获得的混合云解决方案可信云评估，成为获得可信云服务评估数量最多和品质最优的云服务商。体现了世纪互联运营的云计算服务领先的技术和安全可靠的运维和服务。

世纪互联蓝云总裁柯文达表示，"混合云是当下云计算行业的大势所趋，各行业的用户对混合云的解决方案非常关注并有强烈的需求。此次世纪互联蓝云凭借Azure混合云解决方案获得混合云解决方案可信云评估，并成为唯一获得混合云技术创新奖项的企业，代表Microsoft Azure混合云服务完全符合中国法律法规的标准，并在安全性和可靠性方面走在了行业的前列。未来，世纪互联蓝云将继续发挥在混合云方面的优势，致力于为用户提供安全、可靠、高效的Azure混合云解决方案。"

微软亚太科技有限公司董事长、微软亚太研发集团首席运营官、微软中国云计算与企业事业部总经理申元庆表示："恭喜世纪互联蓝云获国内首个混合云解决方案评估！进入中国市场三年多以来，凭借覆盖全球的超大规模、安全可信的企业级服务，由世纪互联运营的Microsoft Azure将国际技术与本土运营完美融合，获得了客户与合作伙伴的广泛支持与认可。得益于微软与世纪互联长期稳定的战略合作，未来双方将继续引领混合云生态的发展，将微软的混合云部署能力，与世纪互联优秀的运维能力相结合，为国内客户提供更加一致、无缝衔接的混合云体验。"
