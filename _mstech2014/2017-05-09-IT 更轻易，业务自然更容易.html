科技飞速发展，让很多东西的保「先」期越来越短。

  • 买个最新款手机，可不到一个月，屏更大，配置更高，拍照效果更好的新型号又来了😤

  • 才添置了一台扫地机器人，可很快，能通过手机远程控制的新型号开始全面铺货😨

  • 你家新换了互联网电视，可隔壁老王家几周后换的电视不仅能看网络节目，还能语音控制🤒

其实生活中这些数码装置还是小意思，只要不差钱就行，反正不算太贵。但如果企业也想让 IT 系统追新，可不光是钱的问题，还有很多其他因素需要考虑。

再举个栗子吧。很多已成立多年的企业，想必在本地部署了各种 IT 系统。然而随着业务转型和发展，本地系统开始在容量、性能、可靠性，以及创新方面遇到挑战。这时该怎么做呢？上云？可毕竟不是从零开始新成立的公司，有很多「历史包袱」需要考虑。

  • 原有的本地系统是继续留在本地，还是迁移到云中

  • 如果留在本地，那么如何与云中部署的新系统相互连接

  • 如果迁移到云中，那么本地系统的相关投资是否全都浪费了？又该如何保障迁移过程平滑顺利

  • 更重要的是，云平台在安全、隐私等方面能否满足自己的需求

其实，你并不孤独，头疼这类问题的企业有很多

 “轻易贷”是轻易科技有限公司创建的网络信贷平台，该平台从成立之初便将业务应用部署在企业自行管理的私有云中，随着用户规模不断扩大，本地私有云环境的容量、规模已无法满足业务需求，公司在IT 管理方面也开始遇到瓶颈和挑战。

  • 成本：为应对不断增长的用户，需要不断对计算、存储、网络资源进行扩充，无论硬件服务器采购、带宽和IDC 托管机房的租赁，都会造成高昂的前期成本支出。

  • 运维管理：该公司一直采用开源的 OpenStack 方案构建私有云环境，部署前必须进行大量兼容性测试和性能调优。同时，采用单一节点的数据中心在稳定性和可靠性方面已无法满足新的要求。

  • 安全性：本地私有云环境的部署，要采购不同厂家的 WAF、IDS、IPS 设备，并进行复杂的配置。

该公司希望能通过一种新方式获得更加安全可靠、灵活，并且有弹性的 IT 资源支持。

Azure 智能云，消除「历史包袱」的终极武器

轻易科技选择了由微软提供技术，世纪互联运营的 Microsoft Azure 智能云平台。Azure智能云能提供完善的技术特性，覆盖从 IaaS到 PaaS 丰富的云服务。同时作为一个中立的平台，Azure 提供了全面的合规与隐私承诺，不将客户数据用于商业目的，从而满足轻易贷严苛的数据隐私保护要求。

数据迁移更平滑

通过配置 Azure 站点到站点 VPN 服务，将本地数据中心与 Azure 公有云实现连通，组建混合云环境，并执行迁移操作。借助 Azure 对开源软件的广泛支持，技术团队可轻松地将运行在 OpenStack 私有云环境的现有虚拟机无缝迁移至 Azure 公有云。

信息安全更有保障

借助 Azure 内置的 DDoS 防护，轻易科技无需额外的硬件设备，即可实现对所有入站 Internet 连接进行检测，保护平台免受大规模攻击。此外技术团队还利用 Azure 提供的 Web 应用防火墙和网络安全组规则对核心应用提供保护，保障云端核心业务系统获得完整的网络隔离。

部署周期大为缩短

轻易科技可以利用 Azure 开箱即用的丰富 PaaS 服务，例如 MySQL Database on Azure、Azure SQL Database 等快速构建数据库，并通过 HDInsight 这一完全托管的Hadoop 服务实现大数据分析，为中小企业客户提供业务洞察力，帮助投资者实现精准投资，取得最佳投资收益，开启智能理财新时代。

顺利打通本地和云端，业务发展驶上快车道

在 Azure 智能云帮助下，轻易科技顺利应对传统本地部署私有云环境的各种不足，降低企业 IT 管理难度和成本的同时，利用 Azure 丰富的 PaaS 服务为业务发展提供洞察力，更好地服务投资者与客户。

优化企业IT 架构和成本，专注主营业务发展

通过自动化扩展，轻易科技能轻松应对轻易贷用户不断增长对 IT 资源的需求，为用户提供不间断的可靠服务。迁移后无需自行维护 IT 基础架构，可直接利用 Azure 丰富的 IaaS 和 PaaS 服务快速部署服务，将高昂的 IT 固定资产和运维投入转化为日常运营成本。不仅实现了成本节约，也让企业实现了轻资产、重业务的目标。

全面提升在线金融平台的安全性与可靠性

得益于 Azure 丰富的安全特性和合规认证，轻易科技能以更具成本效益的方式提升平台安全性和可靠性。北京、上海两地最多共 6 个副本的数据地理冗余，确保数据的万无一失；安全管理方面，通过 Azure 门户即可配置所需安全功能，保障用户信息和资金安全。

引入新技术，为客户创造差异化的体验

轻易科技还能借助 Azure 丰富的认知服务将人工智能技术引入自身互联网金融服务中，为客户创造差异化的体验，在互联网金融服务日益同质化的今天保持独特竞争优势。
