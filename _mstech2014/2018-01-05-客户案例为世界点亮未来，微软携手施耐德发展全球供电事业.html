当今世界人口已有74亿，而作为读者的您也许无法相信，这其中12 亿人口仍生活在没有电力供应、无法获得光明的环境中。幸运的是，有那么一群人一些企业，依靠科技的进步和对世界的责任感，在为应对全球能源挑战不懈地努力着。施耐德电气与微软进行了紧密的合作，利用最先进的 Azure IoT和Cortana Intelligence，针对这一全球挑战设计了变革性的解决方案。给很多像尼日利亚一样的国家和地区带来了广泛可靠的电力供应，让更多地球居民的生活得到了改善，让更多孩子走向了更完备的教育之路，为数以百万的人点亮生活，点亮未来。


微软和施耐德电气公司携手，创造了人间“升级版”的普罗米修斯，使广泛可靠的电力供应不再是个遥远的梦想，为数以百万的人们点亮未来。

应对全球能源挑战

作为一家拥有 180 年历史、客户遍及 100 多个国家/地区的能源管理领域的领导者，施耐德电气公司一直在追求能源管理和自动化的突破。为此，施耐德公司甚至将其总收入 266 亿欧元（296亿美元）的 5% 用于科技研发。因为，施耐德公司相信，获得能源是一项基本人权，施耐德应该为人类的这一基本权利的获得作出努力。

面对当前能源与环境保护之间的冲突，施耐德电气也一直寻求在用电与可持续性之间实现平衡，希望找到一种更有效的方式来监控和管理远程设备。施耐德关注的领域包括可再生能源，如由太阳能、太阳能发电厂供电的隔离场址，以及居民用电与电网性能之间的相互影响。为了达成这一战略目标，公司需要实时掌握全球几乎任何位置的传感器、设备和系统的运行情况。施耐德电气公司数字服务和IoT 部门执行副总裁 Cyril Perducat 指出：“我们服务于大约 18 个不同的垂直市场，这涉及到大量的行业。能源管理是我们的企业战略中的统一主题。我们所面临的一个挑战就是需要建立一个全球解决方案，同时还要根据特定的地区性要求来做出调整。”

Microsoft Azure IoT，兼顾全球性与本地需求

施耐德公司需要一个能够全球扩展的技术平台，这个平台必须足够灵活，以便支持多个市场中的不同场景。Microsoft Azure 平台恰好完全符合施耐德公司的需求。利用 Azure，施耐德能够创建一个灵活的平台，此平台可扩展至全球大量的项目，其中很多项目地处偏远，资源十分有限，但施耐德利用微软Azure IoT 技术，就可以将施耐德的创新方案在另一种环境中立刻进行复制与应用。有效解决了资源紧缺问题。

在与微软的密切协作下，施耐德基于 Microsoft Azure IoT 套件和 Cortana Intelligence 套件创建了其自己的 EcoStruxure 平台。该解决方案将云服务连接到从传感器、恒温仪和数字接口到管理控制及数据获取（SCADA），并构建了自动化系统。基于该平台，施耐德电气实施了第一批产品，包括用于优化太阳能发电厂的Conext Advisor 监控系统，用于监控电池供电的远程光伏设备的 Conext Insight，以及用于交互式监控家用恒温器的Wiser Air。

予力多样化、变革性的项目

施耐德与微软的合作在全球都获得了良好的效果。

在尼日利亚，施耐德公司与微软及拉各斯国家电力委员会协作，通过向 200 多所学校和诊所提供可持续的太阳能电力，来帮助应对该国家的电力挑战。ConextInsight 利用 Azure IoT 套件将光伏系统连接到云，以监控、诊断和实时跟踪能源使用模式。利用 Cortana Intelligence 套件，工作人员可以识别趋势或性能下降的情况，并启动预测性维护。

在中国、印度和东南亚地区也启动了类似的项目。Perducat指出：“我们有多个项目，但所有项目都可归结为向没有电力资源的人们供电。我们的主要驱动因素之一就是教育。我们让孩子们可以去上学，让他们学习和发展，让他们的家园拥有更美好的未来。我们能做的只是让他们的家中有灯光，让孩子们能够做作业。”

电力供应还可以从其他方面改善生活质量。有了持续不断的能源供给，人们可以用泵抽水并对水进行过滤，并且更轻松、安全地烹制食物。

在世界其他地方，Conext Advisor可以远程管理偏远地区的多个太阳能农场。这样，资源有限的地区可以获得稳定的能源供给，同时更高效地运营并控制成本。远程监控中心可以通过分析阴影、天气和灰尘等变量对太阳能发电的影响来优化性能。

建立一个可持续性发展的未来

基于Azure的解决方案以多种方式造福本地经济，尤其是激励本地企业的创新。Perducat 指出：“我们与很多小型家族公司成为合作伙伴，合作伙伴可以是隔壁的电工，也可以是地区安装工人。EcoStruxure、Microsoft IoT 以及一些高级分析技术为他们予力赋能，从而丰富和发展他们自己的愿景与创想。”

施耐德正在帮助世界各地的社区建设更美好的未来。Perducat 表示：“通过物联网予力人们和企业是非常重要的。利用 Microsoft IoT 技术，助力全球经济发展，提高全球生活水平。“
