足球，似乎有这样的一种魔力，它能将不同地域、不同年龄的人凝结在一起，为同一支球队欢呼。它不仅是一种精神，一种寄托，更是一种文化，一种传承。

6月4日凌晨的欧冠总决赛，牵动着全世界皇马球迷的心。

而皇马力求可以和球迷进行更紧密的沟通和联系，给他们更好的体验，让他们拥有亲临体育场的感受。

“我们可以通过 Azure 解决方案与全世界的球迷建立一对一的关系，将这个巨大的群体联系起来，使得皇家马德里支持者获得更好的体验”。


José Ángel Sánchez，CEO Real Madrid C.F.


在体育界，没有比皇家马德里更大的俱乐部。世界第一的职业运动联盟，西班牙足球俱乐部在全球拥有超过 4.5 亿球迷，是球场上的霸主。自 1902 年成立以来，皇马已赢得 116 个官方奖杯 11 个欧洲杯和国际足联俱乐部世界杯，并被国际足联认定为 20 世纪最好的足球俱乐部。

作为世界上领先的职业运动联盟，皇家马德里有很多值得骄傲的地方，赢得过 100 多场冠军赛，拥有狂热的球迷基础。但是足球俱乐部无法直接同全球 4.5 亿球迷进行交流。基于 Azure 的全面技术解决方案的应用，让这种情况得以改变。

现在，皇家马德里能够与球迷更紧密地联系，给他们更好的体验，通过分析社交媒体数据和其他信息，定制营销活动并提高收入


实施数字化转型计划



皇家马德里与Microsoft服务团队合作开发了一个新的技术解决方案，为俱乐部的全球球迷提供创新的数字服务。同时皇马还希望使用该解决方案收集和分析球迷的数据，基于此微软服务团队与皇家马德里深入合作构建了一个基于海外Azure云服务和Office 365平台的综合平台，即服务解决方案。

解决方案包括以下部分：

  • 粉丝参与平台：作为该解决方案的营销后台引擎，该平台记录并存储与粉丝的每次互动，包括俱乐部体育场的移动端签到，俱乐部官方网站上的在线粉丝个人资料更新和在线商品购买。平台由基于Microsoft Dynamics 365客户服务，Dynamics 365市场营销和Power BI的Microsoft Office 365托管。

  • 扩展视频平台：平台由海外Azure媒体服务托管，包含新的视频内容和历史内容，以及皇家马德里以前的比赛。使用其应用服务将应用程序集成到媒体服务。同时解决方案还使用海外Azure媒体播放器将内容呈现给粉丝，Azure搜索作为搜索引擎，因此粉丝可以通过过滤搜索来查看特定标准的比赛。

  • 消费者手机应用程序：手机交互式应用程序专为iOS、Android和Windows平台而设计，使用.NET Framework与Microsoft Visual Studio 2013创建。粉丝通过手机、平板电脑或可穿戴设备即可访问应用程序，无论何时何地都能接触皇家马德里。

    对于用户身份验证和配置文件存储，解决方案使用海外AzureActive Directory企业对消费者（B2C）的预览版本提供服务。粉丝可以使用Facebook等社交帐户登录，在任何平台上跨移动应用程序无缝体验。

  • 遥测和数据分析：解决方案的各个服务，包括跨平台移动应用程序和支持Web服务，利用海外 Visual Studio Application Insights为应用程序提供先进的遥测技术，使粉丝能够跨移动设备和Azure感受整个基础设施的高端技术和操作洞察力。

客户收益



  • 与数百万粉丝进行一对一互动 提升球迷体验：以前，皇家马德里只有一种单向通信方法，用于向其球迷广播信息和新闻，但无法获得球迷的任何反馈。有了这种新的解决方案，皇家马德里如今可通过一种工具来获取数亿粉丝的详细信息，用于和粉丝进行深入互动。

    “我们可以通过境外 Azure 解决方案与全世界的球迷建立一对一的关系，将这个巨大的群体联系起来，使得皇家马德里支持者获得更好的体验。”Sánchez 说到，“我们的目标是获取所有支持者的资料，了解他们是谁，给他们想要从我们获得讯息，例如独家内容，使他们更接近俱乐部。

  • 开展近实时营销活动 大幅提升俱乐部收入

    皇家马德里还能够通过粉丝操作获取收集并存储在粉丝参与平台中的所有粉丝信息，以生成每个粉丝的详细用户简档。这样，皇家马德里营销人员可以提供更有针对性的定制营销举措，最终可以帮助俱乐部增加收入。例如，俱乐部的粉丝参与平台可以汇集通过皇家马德里网站或新的移动应用程序注册的每个粉丝的社交媒体帖子量。

    通过 Microsoft 社交软件分析这些社交媒体信号，皇家马德里可以自定义其营销和广告，或鼓励已下载该应用程序的粉丝使用该程序。或通过获取社交媒体信息，如粉丝的最喜欢的球员，皇家马德里可以向粉丝发送个性化的电子邮件，显示如何在应用程序内查找该球员的统计数据。“我们正在通过 Azure 来了解我们的球迷，他们是谁，身处何地，想要从我们这获得什么，”Sánchez 说到，“我们可以了解他们是否想要产品或内容，以及他们是否想要他们的决定产生影响。我们现在用 Microsoft 解决方案实现了一直以来的梦想。

  • 使用数据分析应用使用情况和粉丝行为 实时了解球员状态：随着解决方案的拓展，新功能的添加并以新的方式与粉丝互动，皇家马德里将继续彻底转变其业务。同时，皇家马德里也能够使用该解决方案来分析特定的用户行为。数字平台记录与用户行为相关的所有信息，例如在外部挑战中获胜、在特定地点签到或更新档案。数据收集和分析的过程基于 Azure Data Factory 用于提取、加载和转换信息，并将其存储在 HDInsight 中进行查询。然后将数据发布到 SQL 数据库，并由 Excel 和 Power BI 使用。此外，皇家马德里可以进行查询，从大型数据集中提取特定信息。

    如果用户点击应用程序弹出的广告，或使用集成在应用程序中的第三方应用程序访问特定区域，俱乐部可以从遥测信息中了解用户如何与应用程序互动，包括他们如何打开应用程序，使用 Visual Studio Application Insights，Real Madrid 可以查看哪些用户已执行特定操作。俱乐部可以使用此信息来更好地监控、操作和分析数据。通过收集和分析的所有数据，皇家马德里通过一系列程序将信息清理、分类并分组为报告，显示用户如何参与应用，与系统交互的用户数量以及在应用程序内的具体动作。

    展望未来，皇家马德里计划使用 Azure Machine Learning 直接与教练和球员联系，以了解球员的状态，并预测在赛季期间座位的价格。

* 此案例基于在全球其他地区由微软运营的 Microsoft Azure 云平台实施，仅供由世纪互联运营的 Microsoft Azure 客户参考。
