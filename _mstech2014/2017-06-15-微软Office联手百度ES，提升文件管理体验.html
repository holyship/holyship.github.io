6月13日，微软与百度达成合作，百度旗下安卓文件管理应用ES File Explorer将增加Office安卓手机端应用Word、Excel和PowerPoint的文件深层链接（deep link）， 用户可在ES File Explorer中直接打开Office文件，无需手动切换至Office应用。目前，新功能已经启用，微软也将在Office安卓端应用的“What’s New”中向用户推荐ES File Explorer。


微软公司全球Office产品市场部总经理Jared Spataro表示：“我们希望将微软Office应用带到每一个系统平台和每一台设备上。我们非常高兴能够与百度合作，将安卓手机端Office应用与ES File Explorer整合。”

百度ES File Explorer产品总经理罗锋表示，双方合作将给用户提供更简单、更流畅的使用体验。“ES File用户可以管理手机里的所有文件，包括Word、Excel和PowerPoint，就跟在电脑上操作一样。”




微软Office移动端应用支持图表、动画、Smart Art图形和形状等功能，为年轻人带来更多的办公选择和方式方法，在工作中更加突出自我专业性。

支持多种分享形式，不论微信、QQ，还是邮件形式，都可以即时分享，即使在旅途中，也可以修改项目汇报、提交或转发文件，随时随地提高工作效率。

无广告，让思路更连贯，注意力更集中，界面简洁，专注工作，移动端办公也能很高效。

现在即可前往各大安卓应用商店，下载体验微软Office移动端应用及百度ES File Explorer应用。
