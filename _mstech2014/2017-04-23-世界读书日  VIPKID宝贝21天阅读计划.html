世界读书日

每一个不曾起舞的日子，都是对生命的辜负。每一个不曾读书的日子，都是对自己的辜负。 阅读，是知识和智慧的源泉，也是孩子们打开认识世界的钥匙。“4.23世界读书日”，和你的宝贝一起开启阅读计划，让读书变成每日必行的自然而然的事情。

儿童英语绘本阅读

一直以来，绘本阅读都被认为是儿童启蒙教育中最为有效的方式之一。但绝大部分的中国家长都不知道如何去挑选适合孩子阅读读物，尤其是英语读物。对孩子来说，太难或者太容易的英文绘本，都会降低英语学习的积极性和效果。另外，中国孩子由于不具备天然的英文环境，阅读的英文读物也应该和纯母语国家的孩子区别开来。

这需要根据孩子的基础阅读需求，对绘本进行严谨、恰当的阅读分级，并在此基础上精心挑选适合孩子的趣味补充读物，既帮妈妈们免去挑选绘本的烦恼，也让孩子接触到更适合自己的内容，保持良好的阅读兴趣。

21天阅读节

在“4.23世界读书日”到来之际，微软好伙伴VIPKID推出“北美好外教，21天阅读节”的英语绘本线上阅读活动，通过微软提供线上绘本存储的云平台技术，提供阅读节活动所需的英文读物内容，帮助中国小朋友的英语学习。鼓励妈妈们带着孩子养成坚持阅读的好习惯，激发孩子英语学习的兴趣，帮助孩子在阅读中认识世界、了解世界。

VIPKID与微软Azure

在应用 Microsoft Azure 以前，用低成本的方式让美国的教师与学生实现跨太平洋的双向实时互动几乎是难以想象的。现在，通过将我们自主研发的在线教育平台部署到 Azure 上，利用 Azure 在网络方面的独特优势，让中国孩子实现“美国小学在家上”已不是梦想。

——张燕静，技术副总裁，VIPKID

在这个由互联网所引领的新经济时代，作为创业公司，当然要技术为先。在 VIPKID 技术副总裁张燕静的带领下，VIPKID 探索出一条基于网络的互动教学方式，即将业务直接部署到云上，通过网站、移动端 App 为学生提供随时随地的教学体验，而教师则可以身在美国以远程的方式与孩子进行1:1的授课与指导，这样既可以方便孩子的学习，又能够利用美国海量的教师资源，教师和学生都能够利用自己的自由时间，并且企业也能够降低教师的人力成本，从而为学生提供更具竞争能力的课程报价。

在实际的应用中，Microsoft Azure 的多种云服务为 VIPKID 的在线实时互动教学提供了全面的支持。不仅是教师与学生可以通过部署在 Azure 上的在线教育平台进行互动，学生家长也能够接入这一平台，及时了解学生的学习状况，每堂课程所使用的教学课件、完整教学录像将使用 Azure 存储服务被可靠的存储于位于北京与上海的数据中心当中，便于学生随时复习与查阅。

本次活动中，VIPKID在微软Azure云平台上，提供自主研发的phoenix绘本及拥有蓝思分级的美国进口经典绘本供孩子们在线阅读，帮助孩子从小形成热爱阅读的英语学习习惯。同时，VIPKID还通过丰富的H5互动形式，推出一系列的阅读奖励措施，激励家长和孩子们积极参与，在家长间形成一股“绘本社交”的亲子阅读学习氛围。

从即日起，家长只要登录VIPKID官网或官方新媒体平台，参加“北美好外教，21天阅读节”活动，都能获得价值100元的北美明星外教精品绘本公开课，以及价值288元的试听课大礼包；成功邀请好友一起参加打卡阅读计划，家长和好友能分别获得一本北美外教推荐的经典实体绘本；而完成连续21天打卡阅读的成就后，还能获得VIPKID提供的精致立体有声绘本一本以及学习礼盒大奖。
