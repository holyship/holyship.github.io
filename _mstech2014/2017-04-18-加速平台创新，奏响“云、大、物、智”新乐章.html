微软亚太科技有限公司 董事长、微软亚太研发集团 首席运营官、

微软中国云计算与企业事业部 总经理 申元庆

席卷全球的数字化转型为各行各业带来了巨大的发展机遇，而微软智能云作为推动技术创新的平台和动力源，其自身也实现了远超预期的快速发展： Azure云服务已覆盖全球38个区域；有超过90%的世界“财富500强”企业采用微软智能云服务；增长强劲的企业级云计算业务，已经成为引领整个微软成功实现业务转型、并取得健康发展的重要引擎。

在中国，由世纪互联运营的Microsoft Azure是第一个正式实现在华商用的国际公有云服务；三年来，我们陆续在中国推出了超过60项具有国际水准的云服务；Microsoft Azure在华解决方案合作伙伴超过1000家；有超过8万家来自不同行业的企业客户在使用Azure云服务—— 其中既有观致汽车、蒙牛集团这样三年来始终支持Azure的老客户，也有微鲸电视、棠棣科技、白山云这样以人工智能为契机与我们结缘的新朋友。

三年来，由世纪互联运营的Office 365也取得了不俗的市场表现：Office365服务于120多万付费企业用户和超过2000万教育行业用户，以23%的市场份额位居中国SaaS级云应用市场排名第一；Office365在华业务累计实现三位数的年度平均增长；跨平台Office 移动端的下载数量也已超过4500万次。

从“创新驱动发展”、“互联网+”到“中国制造2025”和“大众创业、万众创新”，充满创新活力的中国市场为推动数字化转型创造了最优越的政策环境。而微软智能云，希望能以我们的技术、平台、产品和服务，全力推动云计算、大数据、物联网、人工智能等新技术的广泛普及和实际应用，助力更多的中国企业与合作伙伴发现新机遇、创造新价值。

云计算：巩固先发优势，稳步扩大规模

覆盖全球的超大规模、安全可信的企业级服务、独特的混合云部署能力，以及微软与本土运营合作伙伴世纪互联长期稳定的战略合作，让微软智能云Azure将国际技术与本土运营完美融合，率先在中国市场生根发芽，实现了快速健康的发展，并得到了广大客户与合作伙伴的广泛支持与认可。

微软合作伙伴云角充分利用Azure全球部署的优势，帮助多家世界知名企业实现了广泛覆盖东南亚和北美地区的云业务部署；蒙牛集团旗下的23个品牌和产品的官方网站及所有市场活动页面运行在Azure上，并借助微软云上的数据平台实现了针对超过2000万用户画像的分类跟踪，通过数据洞察和精准营销促进了超过10亿元的在线销售额；复旦大学附属华山医院，不但采用基于Azure和Windows Server的混合云架构搭建起了医疗数据中心，还借助SQL Server上的R语言机器学习服务，打造出了智能化的个性化给药系统。

微软与世纪互联的战略合作，为微软智能云的健康发展奠定了坚实基础。世纪互联经微软授权在北京、上海两地运营着两个互为备份的Azure区域，能确保安全可靠的日常运营和快速的灾难恢复。目前世纪互联有超过400名经过微软培训和严格认证的全职技术人员，负责维护Azure和Office 365的日常运营工作。2016年，由世纪互联运营的微软智能云服务在中国的客户满意度为95%，高于Azure海外服务的表现。由世纪互联在华运营的MicrosoftAzure和Office 365累计获得9项国内权威的“可信云”认证，并分别通过了国家信息安全等级保护第三级评测，能够从数据安全、隐私保护、合规、透明等维度全方位确保微软智能云在华运行的安全可信。

中国的云计算市场正在进入快速成长期，数据显示“十二五”期间国内云计算市场实现了超过30%的连续年度增长。为了应对国内云业务持续增长的业务需求，为客户和合作伙伴提供更高水准的软硬件配置及服务，由世纪互联运营的微软智能云Azure，在过去三年间已经将云计算规模扩展了一倍。未来12个月，还将令云计算规模再提升一倍，将全球顶尖的公有云服务技术与功能同步引入中国市场，为中国客户带来更具创新价值的国际品质云服务。


大数据：以平台服务发掘数据洞察

3月初，在Gartner发布的“分析数据管理解决方案（DMSA）”魔力象限报告中，微软连续第六年被评选为这一领域的领导者。相比以往，Gartner今年的报告更看重面向云计算的数据存储和分析解决方案，而这恰恰与微软的云端大数据战略不谋而合 —— 微软智能云Azure的目标，就是通过打造一系列PaaS级数据服务，帮助用户简化大数据处理和分析的过程，全面加速从数据到业务洞察再到智能决策的过程，让数据真正高效地服务于业务创新。

受到Gartner充分肯定的Azure SQL数据仓库已在中国市场正式上线，它提供了一个完全托管、高度可缩放的云端数据仓库服务，能以千万亿字节的规模进行大规模的并行处理。Azure SQL 数据仓库的存储和计算服务可独立缩放，根据用户需要进行动态部署，在几分钟内就能完成计算能力的增加、收缩或者暂停操作，用户仅为查询性能付费，从而大大压缩了使用成本。此外，Azure SQL 数据仓库采用与SQL Server 和微软分析平台系统相同的数据库引擎，可以灵活选择数据存储位置，实现跨云端、本地和不同设备的无缝衔接。

基于Apache Hadoop架构的Azure HDInsight推出了基于Ubuntu 和 Hortonworks 数据平台（HDP）的Azure HDInsight on Linux。用户在部署工作负荷时选择Linux 群集，就能轻松地将现有 Hadoop 工作负荷转移到云端。这项Azure服务可自动更新、监控和运营 HDInsight 中托管的 Hadoop Linux 群集，从而让用户专注于查询和数据，而不必关注计算机管理或 Hadoop 修补。

在云端机器学习方面，我们推出了运行在Linux 虚拟机上的 R Server。作为目前部署最广泛的企业级分析平台，R Server提供了统计、预测建模、机器学习等高级大数据分析服务。而且，通过添加并行和块数据处理功能，运行在Azure上的R Server消除了开源 R的内存限制，让用户可以运行分析远大于主内存的数据。

微软PowerBI 被Gartner评为“商务智能与分析平台（BIAP）”魔力象限中的领导者。作为Azure平台的一项SaaS服务， Power BI Embedded 提供了REST API和SDK，让任何开发者都可以为不同平台的应用嵌入灵活多样的交互式分析报告。这项服务能以最生动、直观的形式在应用中嵌入数据洞察，所有数据均能同步实时更新，而且支持以云规模进行大范围的快速部署。

物联网：智能化升级，防患于未然

据预测，到2020年全球将有300亿到500亿台各类设备接入物联网，要管理如此规模的设备及其背后快速增殖的天文数字级别的数据，无疑将是一个巨大的技术挑战。为了以聪明、高效的智能管好物联网，微软推出了完整的云端物联网解决方案Azure IoT套件。

在去年9月上线的远程监控方案的基础上，Azure IoT 套件新增了预测性维护预配置解决方案。在利用远程监控服务实时采集遥测设备数据的基础上，这项方案加入了R Server预测模型，能通过机器学习对数据进行实时分析和研判，然后将模型预测的结果在监控仪表板上显示出来。所有有价值的数据洞察和预警信息一目了然地呈现在管理人员的眼前，让人们可以及时发现潜在的问题，实现防患于未然。

全球知名的电梯厂商蒂森 ∙ 克虏伯已经在利用Azure IoT套件的预防性维护服务，对部署在全球各地的110万台电梯设备进行检修和维护维护，显著减少了设备的停机时间和相关的成本投入；引擎制造商罗尔斯 ∙ 罗伊斯 则利用预防性维护来管理超过13000台民航客机的引擎，及时掌控这些喷气发动机的工作状态，不但有助于降低燃油消耗，还能将维护成本最低化，同时减少航班延误的潜在风险。

为了满足物联网设备的多样化技术需求，过去几个月，我们陆续升级了Azure IoT 网关 SDK 和Azure IoT中心设备管理的新功能，并为Azure IoT套件远程监控方案加入设备管理功能。其中，Azure IoT 网关 SDK可以帮助各种新老设备连接到云，提升物联网解决方案的安全性，并降低带宽成本；而Azure IoT 中心则提供了功能和可扩展模型，让设备和后端开发人员构建功能强大的设备管理解决方案，运用Azure超大规模云计算的能力，高效地组织、监控、查询和管理数以亿计的设备。

人工智能：把云端智能带给每个人

大约两年前，当微软CEO萨提亚 ∙ 纳德拉第一次提出“构建微软智能云”的远见时，很多人在问，微软Azure究竟智能在什么地方？如今，人工智能成了人们街谈巷议的“全民热词”，而我们在构建微软智能云上的战略规划也显得越发清晰起来—— 微软的目标是普及人工智能全民化（DemocratizingAI） —— 将先进的人工智能技术转化为每个人触手可及的工具、平台和服务，予力全球每一人、每一组织，成就不凡。

运行在微软智能云Azure上的微软认知服务，是微软在人工智能领域20多年研究成果的集大成者。我们将覆盖视觉、语言、语音、搜索、知识五个主要应用场景的24项人工智能服务，集成为24个API接口开放出来，让开发者只需要几行代码，就能借助微软的人工智能平台，开发出自己的跨平台人工智能应用。

人脸识别、情绪识别、计算机视觉三个API在去年9月率先在国内提供了预览服务，目前已经有多家国内客户、开发者和合作伙伴将这些API用于不同的应用场景，并且陆续开发出了成熟的产品。


两周前，微鲸科技发布的新型智能电视产品，就加入了基于微软认知服务的人脸识别应用。微鲸电视的“刷脸开锁”功能在识别出儿童使用电视时，会自动进入专属的“儿童电视”页面，同时对孩子观看电视的时间加以控制。获得首批“中国机器人认证001号证书”的“棠宝”智能服务机器人致力于为银行、商超、酒店、医疗机构提供智能服务，其开发商棠棣信息正在与微软展开合作，借助微软认知服务，为机器人打造更流畅的人脸识别、语音交互体验，构建低成本、高收益的人工智能解决方案。此外，国内知名的云内容加速分发服务商白山云，则利用微软认知服务的视频内容识别功能，开发出了一套为视频直播平台提供实时内容审核功能的智能化解决方案。

微软认知服务的更多功能和API将陆续落地中国市场，同时，我们也欢迎放眼国际市场的技术企业和开发者，对先期在海外推出的微软认知服务进行技术探索。例如，由深度神经网络技术驱动的Microsoft Translator翻译服务，能实现包汉语在内的9种不同语言之间的实时翻译，并提供了文字翻译和语音翻译两个API供第三方应用和服务使用。国内知名互动娱乐服务商游族网络已经在尝试利用这项翻译服务，为海外推广的国产游戏，提供实时翻译服务。

一周前，工信部发布了《云计算发展三年行动计划（2017-2019年）》，提出2019年中国云计算产业规模将达到4300亿元；并明确指出，目前云计算已经从概念导入进入到广泛普及、应用繁荣的新阶段，成为提升信息化发展水平，打造数字经济新动能的重要支撑。可以想见，被寄予重望的中国云计算市场即将迎来新一轮的加速发展期。

以“云、大、物、智”为切入点，微软将继续加深与中国云计算产业和云生态系统的紧密联系，立足本土运营，发挥技术优势，致力于将微软智能云打造成一个安全可信、成熟高效，并能持续创造价值的创新平台，帮助更多的中国客户、合作伙伴和开发者，赢得数字化转型带来的新机遇。
