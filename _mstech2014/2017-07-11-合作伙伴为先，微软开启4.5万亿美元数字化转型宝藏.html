本文作者：Judson Althoff，微软公司执行副总裁，全球商用业务部门


在技术剧变的风口浪尖，整个世界将迎来重大的机遇。值此之际，微软今天在华盛顿特区召开了Microsoft Inspire大会。由物联网、增强现实、人工智能和数据等云技术驱动的商业创新——数字化转型正为各行各业的企业用户带来巨大的商业价值，而微软的数字化转型给客户和合作伙伴带来了4.5万亿美元的市场。



微软Inspire大会现场


据《哈佛商业评论》统计，86%的企业认为数字化颠覆将带来新的机遇。来自各行各业的企业客户正在寻求通过数字化转型实现密切客户沟通、予力赋能员工、优化业务运营和转型业务模式的方法。与此同时，企业的领导者也意识到成功的转型必须依靠企业的员工及其文化，而微软今天推出的服务将是解决这一问题的核心要义。

 

推出Microsoft 365

在今天举行的年度合作伙伴大会上，我们宣布发布了全新的 Microsoft 365，一款全新的商用产品，包括 Office 365、Windows 10、企业移动性+安全性（Enterprise Mobility + Security）等。Microsoft 365是一个完整的、智能的、安全的解决方案，将帮助每一家公司、每一位员工释放出最大潜力，并充分体现“人”才是数字化转型的核心——以人的需求为出发点，Microsoft 365 将为员工提供充分的灵活性，让人们在自己喜欢的地方、以自己需要的方式工作，并能以创新的方式实现彼此协作——同时在整个过程中，确保企业数据的绝对安全。


在今天的大会上，微软带来了两个版本的Microsoft 365。Microsoft 365企业版是之前推出的Secure Productive Enterprise计划的升级，其包含有Office 365企业版、Windows 10企业版和企业移动性+安全性套件。它专为为大企业而设计，帮助大型企业赋能员工，使其能够更加有创造力、更好地协作，同时确保安全。


Microsoft 365 商业版将在8月2日开启公众预览，专为拥有不超过300名员工的中小企业设计。这一版本包含了Office 365商业高级版，还包含基于Windows  10和企业移动性+安全性定制的安全和管理功能，只要通过一个统一的控制台，就能对与设备、用户相关的部署、安全配置等所有功能进行有效的管理。


Microsoft 365代表了微软在设计、构建和走向市场方面的一个基本的转变，致力于满足客户的需求以让工作环境现代化。这是一个更加整合的策略，同时也反映出合作伙伴和客户身上正在发生的转变——今天，人们不再把生产力工具、网络安全和设备管理看作是相互独立的工作，而是在寻求用一种完善的解决方案来打造安全无忧的生产环境。对于合作伙伴，Microsoft 365将带来巨大的机遇，帮助他们提升业务规模、打造差异化的产品，并从托管业务中发现新的增长点。Forrester顾问公司总体经济效益影响调查（Forrester顾问公司执行的委托调查）显示，相比单独部署Office 365，Microsoft 365 企业版将为合作伙伴增加50%的营收机遇。

 

应用与基础设施

随着越来越多企业加入数字化转型的浪潮，我们的合作伙伴需要更大胆的想法和更快速的创造。为了进一步加速微软合作伙伴的成功，并为在2018年实现200亿美元云业务营收的目标增砖添瓦，微软宣布与首批硬件合作伙伴Dell EMC、HPE、联想共同推出微软Azure Stack混合云解决方案，并已开放预定。Azure Stack是微软智能云Azure的扩展，能够将云计算的灵活性和快节奏的技术创新带到本地数据中心，并由此创造出一系列全新的混合云应用场景。


Azure Stack提供了真正一致的混合云平台，既能充分满足客户对灵活性的需求，又无需重新构建业务应用。从Azure云端数据中心到云的边缘——无论是在地下几千米的矿井深处、在漂泊于大洋的航船上，还是需要不间断实时运转的工厂车间里——Azure Stack都能以现代化的云应用，来满足不同业务和管理的需求。真正一致的混合云架构，帮助客户更快速地执行云战略，并且可以最大限度地满足其业务场景需求。


Azure Stack进一步扩展了微软智能云Azure生态系统的市场机遇，借助混合云应用场景，合作伙伴将有机会在发展Azure业务的同时触及到一个更具前景的广阔市场。Rackspace、Tieto、Ressello，已经有一批微软合作伙伴开始使用Azure Stack为业务赋能。合作伙伴可以参考业务实践开发指南构建一个成功的云实践，也可以通过在线培训课程，完善自己驾驭Azure的技能。

 

业务应用

我们很高兴宣布微软的两项新举措，帮助合作伙伴更好地服务于客户，并将创新解决方案推向市场。


去年，我们尝试推出了一项全新的联合销售计划，为合作伙伴开发的Azure解决方案提供更全面的销售和市场推广支持。在最初6个月的市场实践中，这项计划成功帮助Azure合作伙伴签下了价值超过10亿美元的年合同量，为Azure合作伙伴输送了60亿美元的业务机会，总共为合作伙伴带来了超过4500笔生意。


微软智能云Azure是唯一能为合作伙伴带来如此惊人收益的公有云。在这种联合销售模式下，微软销售代表帮助合作伙伴售出符合要求的Azure云解决方案，就能获得最多相当于合作伙伴年合同金额10%的销售收入。如此诱人的激励措施，表明了微软全力支持自己的销售代表与Azure合作伙伴通力合作、携手并肩拓展新业务的决心；同时，合作伙伴也能充分利用微软旗下的销售大军，在全球范围内开拓更广阔的业务前景。


以如此强劲的发展势头为基础，微软将在未来12个月内继续增加投入，进一步促进Azure联合销售模式的发展。为此，微软还专门在内部新增了渠道经理的角色，其100% 地专注于帮助合作伙伴扩大市场影响力、确保解决方案到达新客户，并与微软自己的联合销售举措相互整合。微软将总共投入2.5亿美元，一方面用来增加服务合作伙伴的人力投入，另一方面用以继续加大对Azure联合销售的奖励力度。


为了帮助合作伙伴更轻松地对现有业务应用进行现代化升级，或者是从头打造全新应用，我们还带来了一项名为ISV Cloud Embed的新计划。通过这项计划，合作伙伴可以以最低50%的折扣价购买Dynamics 365、Power BI、Power Apps和 Microsoft Flow的核心功能，然后将其作为“搭建模块”嵌入到自己的应用中。类似于在Azure上构建云应用的方式，现在合作伙伴也可以利用微软的这些业务应用平台，轻松地在他们开发的应用中加入销售自动化、服务热线、运营后台服务等功能。通过引入微软的工程开发资源为我所用，该计划将大大降低合作伙伴的开发支出和成本。此外，合作伙伴还将得到不同等级的市场推广支持，有机会接触到Office365超过1亿的月活跃用户，或者通过Microsoft AppSource接触到Dynamics 365快速增长的用户群。


从Azure云开发到业界领先的联合销售支持，再到全新的云端嵌入业务应用服务，除了微软，没有其他任何一家云计算供应商能够提供如此全面的举措，来诚心实意地帮助合作伙伴加速业务的转型和发展。

 

合作伙伴为先

我们的使命是予力全球每一人，每一组织，成就不凡；每天，我们的合作伙伴都在服务客户的过程中，实践着这一使命。在全球范围内，微软合作伙伴创造了1700万个工作岗位，IDC于IDC微软足迹模型2017（IDC Microsoft Footprint Model 2017）预计，2017年微软及其合作伙伴生态系统将创造总计约1万亿美元的营收。


今天，微软正在努力帮助不同行业的企业更快速、更高效地发展。一方面，我们以云计算、物联网、人工智能、大数据驱动业务创新；同时，微软也在通过LinkedIn Sales Navigator与Dynamics 365的整合，以及今天宣布的混合现实合作伙伴计划等举措，创造更多的发展机会。在Microsoft Inspire大会上，来自毕马威、施耐德电气、丝芙兰、Adobe、Track’em及更多知名企业的高管，纷纷展示了他们如何在自己的行业中贯彻数字化转型。例如，毕马威宣布他们与微软合作创造了KPMG数字解决方案中心，毕马威旗下的智能审计平台Clara也运行在Azure上—— 这标志着毕马威成为第一个将审计平台放到公有云平台上的“四大”会计师事务所之一。


不过，对于微软来讲，由我们的合作伙伴和客户共同驱动的转型并不是一个新课题。在过去的40多年中，微软始终是一家以合作伙伴为先的企业，最直接的证明就是超过95%的微软业务都来自于其强大无比、并且不断发展壮大的合作伙伴生态系统。今天，微软拥有超过64000家云合作伙伴——比AWS、谷歌、Salesforce的总和还要多。合作伙伴告诉微软，他们之所以选择我们，因为微软不仅是一家技术提供商，更是长远发展的伙伴。值得一提的是，有30%的合作伙伴是在过去一年中加入到微软合作伙伴阵营中的，而且每个月还要新增6000个合作伙伴。携手同行，微软和合作伙伴将共同成就不凡。


为了帮助合作伙伴更高效地创新、更快速地走向市场，并且在正确的时间找到正确的客户，微软正在转变与合作伙伴沟通的方式，将所有与合作伙伴相关的角色都重新整合为一个新的部门。现在，微软内部所有面向合作伙伴的角色，已经统一为三项主要职能：共同开发、共同走向市场、共同销售（Build-with,Go-To-Market, Sell-with）。此举将有助于简化和优化沟通流程、增强互动协作，并且鼓励以合作伙伴为先的企业文化蓬勃发展。


作为微软最重要的年度合作伙伴盛会，Microsoft Inspire汇聚了超过17000名参会者，大会期间将举办14.5万次各类会谈。据统计，以往参加大会的参展商，其销售额呈现出30%的年增长。通过观看在线主题演讲，可以了解今年大会关注的更多热点。数字化转型势不可挡，只要抓住机遇，微软合作伙伴的发展前景不可限量。
