2017 年 5 月 2 日，纽约和雷德蒙德 —— 微软今天宣布针对教师和学生推出一系列教育类产品和服务，其中包括 Windows 的全新体验 —— Windows 10 S、为现代协作课堂打造的 Microsoft Teams 的全新体验、《我的世界》的新功能、激发创意的混合现实、一系列适用于 K-12 教室的 Windows 10 S PC 设备，以及为大学生设计的 Windows 10 S  设备 —— Surface Laptop。

微软推出 Windows 10 S

Windows 10 S 提供了为安全性和卓越性能而精简的Windows 全新体验。在 Windows 10 S 中，教师和学生下载的应用均来自 Windows 应用商店，经过 Windows 应用商店的安全验证，然后在本地的安全“容器”中运行这些应用， 从而确保 Windows 10 S 提供完整的应用体验， 如完整的 Office 365 应用体验，包括 Word、Excel、OneNote 和 PowerPoint  等。通过 Windows 10 S，教育工作者和学生可以访问他们最喜爱和最常用的涵盖 STEM 教育的应用程序、学习工具、安全评估工具、内容出版商和交互式电子白板等。

Windows 10 S 中的 Microsoft Edge 浏览器通过新的选项卡预览栏，能够让学生比以往更轻松地组织研究和管理学校的项目。学生还可以通过 Microsoft Edge 中的墨迹书写功能直接在网页上做笔记并与同学分享。Microsoft Edge 还与教育工作者和学生们经常使用的网站兼容。

我们的目标是通过 Windows 10 S，发展当下开放且充满活力的以合作伙伴为中心的生态系统。宏碁、华硕、戴尔、富士通、惠普、三星和东芝等合作伙伴，将在今天推出一系列 Windows 10 教育类 PC，起售价仅为 189 美金*。在未来几个月中， 这些合作伙伴还将带来一系列的 Windows 10 S 设备，其中不仅有功能强大且经济实惠的拥有墨迹书写功能的可触控 PC，还有外观精美的高端设备。

将 Microsoft Teams 带进课堂*

学生已经可以通过 Word、Excel 和 Powerpoint Online 实时共同编辑文档。今年晚些时候，用户就可以通过 Microsoft 应用商店教育版下载并体验 Microsoft Teams*，这种全新的课堂体验将团队协作带到一个全新的水平，而且 Teams 将成为课堂中教师和学生协作和学习的数字中枢。通过 OneNote 课堂笔记本创建新作业和小测验的功能，教师能够轻松的分发内容、批改作业、定制个性化学习内容，以及与学生、家长和同事进行沟通。

通过 Teams，教师还可以加入丰富和有趣的内容，这些内容来自像 busuu、Canvas、Flipgrid 及 Kahoot! 等教育合作伙伴的网站**、演讲嘉宾和解决方案。更重要的是，Teams 还可以帮助培养学生未来成功所需的沟通和协作技能。

《我的世界：教育版》Code Builder*：在课堂中教编程

超过 100 多个国家的教师正在通过 《我的世界：教育版》推动课堂中的创新和协作。《我的世界：教育版》能够帮助教育工作者在课堂传授数字公民意识、激发有创意的问题处理能力并使以学生为中心的探索贯穿整个课程，已经成为深受学生和教育工作者青睐的授课资源。

微软正通过《我的世界：教育版》的 Code Builder 拓宽游戏的吸引力。这一新功能旨在鼓励学生们将《我的世界》的开放世界与流行的编程学习平台（如 Tynker、ScratchX 和来自微软的全新开源平台 MakeCode**）相结合，来学习基础知识和编程能力。玩家可以通过编写代码在《我的世界》中实现自由移动、建造和创造。今日起《我的世界：教育版》和 Code Builder 扩展套件将为用户提供一年的免费试用时间*，全球使用 Windows 10 的学校可通过 Microsoft 应用商店教育版进行下载。

专为教师与学生们提供的免费优惠*

为了让教师与学生们能够更容易地在课堂上使用 Windows 10 系统、拥有 Microsoft Teams 的 Office 365 教育版和《我的世界：教育版》，微软将为他们提供：

  • 起售价为 189 美元的全新的 Windows 10 教育类 PC *

  • 为全新的 Windows 10 教育类 PC 提供《我的世界：教育版》一年的免费订阅*

  • 为所有使用 Windows 专业版 PC 的学校免费提供 Windows 10 S*

  • 免费的拥有 Microsoft Teams 的 Office 365 教育版*

  • 免费试用 Microsoft Intune 教育版**

Windows 10 S、拥有 Microsoft Teams 的 Office 365 教育版、《我的世界：教育版》和 Microsoft Intune 教育版将为教师和学生们提供完整的解决方案。

微软推出全新 Surface Laptop，搭载 Windows 10 S

微软扩增 Surface 产品系列，推出 Surface Laptop，给大学生们带来 Windows 10 S 的精简性能、易用及安全性。Surface Laptop 今日起在 20 个国家开始预售。Surface Laptop 将便携、性能和典雅的外观完美平衡，它搭载 Windows 10 S，配备第 7 代 Intel® 酷睿™ 处理器，电池续航高达 14.5 小时***，其设计旨在快速启动并高速运行。

Surface Laptop 的每个细节都经过精雕细琢，从而为经典的笔记本设计带来新的形态和功能，以吸引用户的关注。Surface Laptop 配备 13.5 英寸 PixelSense 显示屏，该显示屏为触控和书写功能进行了优化，从而为用户带来沉浸式的体验。显示屏采用光学粘合的金刚玻璃，使其耐用且拥有美丽的外观。其专业键盘盖采用奢华的 Alcantara 材质，键盘打字安静且反应灵敏，大尺寸精准的触控板和柔软的掌托为用户提供更舒适的输入体验。Surface Laptop 拥有四种颜色****：深酒红、亮铂金、灰钴蓝和石墨金。

Surface Laptop 将于 6 月 15 日正式上市，起售价为 999 美元。从今天起，美国、澳大利亚、新西兰、奥地利、比利时、丹麦、芬兰、法国、德国、爱尔兰、意大利、卢森堡、荷兰、挪威、波兰、葡萄牙、西班牙、瑞典、瑞士和英国的消费者可通过微软商店、 microsoftstore.com 和百思买预订 Surface Laptop。Surface Laptop 将于今年晚些时候在加拿大和大中华区市场上市。我们将会在接下来的几个月宣布其他市场的上市情况。

给课堂带来 3D 和 Windows Mixed Reality（Windows 混合现实）体验激发学生创意灵感

通过 3D 和 Windows Mixed Reality 技术，微软赋予教育工作者一整套全新的工具，帮助他们为学生和同事创造更多的课程。今天，微软宣布了今年秋季 Windows 10 将迎来一项新的功能“View Mixed Reality”。该功能可以实现通过普通的 RGB 摄像头，即可让您在屏幕上看到您的 3D 内容呈现在现实世界中，获得混合现实般的体验。

微软还宣布与培生（Pearson）建立新的战略合作伙伴关系，培生是全球领先的教育集团之一，它正在努力将 3D 和混合现实结合并运用到该公司的中学和大学课件当中。到 2018 学年，培生将在 Windows Mixed Reality 设备上提供健康、商业、历史和 STEM 的全部课件。

STEM 星期六讲习班将在微软商店上线*

微软通过一系列全年免费项目，让学生和教育工作者们直接获得技术和进行实践学习。除此之外，从 5 月 6 日星期六开始，美国、加拿大、波多黎各和澳大利亚的微软商店将在 5 月份举办 STEM 星期六讲习班。STEM 星期六讲习班将“微软教育”的快闪式教室带进微软商店，并为参与者提供一种全新的实践体验，让参与者构建一个用于测量手指的屈曲和伸展的传感器，从而了解人类手掌的解剖结构。包括教师、学生和家长在内的所有人，都可以在五月份的每个星期六去微软商店参加 STEM 星期六讲习班。
