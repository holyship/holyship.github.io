如今孩子的作业，已渐渐由单纯的学校老师批改，延伸至家庭的监督与检查，更有甚者成为了“家长作业”，这不禁让许多家长面露难色。家庭教育是孩子学习成长中必不可少的环节，但确实迫于时间与能力的双重压力，检查孩子家庭作业，家长如何突出重围？

关于家长如何检查孩子作业，甚至网上已出现各色“攻略贴”，但现实是残酷的：

  • 时间紧、任务重：在孩子小学阶段，很多家长正处于事业的顶峰时期，应顾不暇，回到家已身心俱疲，很难再静下心来为孩子批改作业；

  • 心有余而力不足：如今面对孩子的作业，离开学校十几二十年的家长，尚不论辅导技巧，可能连自己解题都还成问题。

此情此景之下，初创企业课课应运而生。


初创企业课课：解放家长，让孩子更轻松地完成作业


学习就像盖房子，小学学习就是这个地基，而通过作业踏踏实实地掌握这个阶段的知识，就如同一块一块地为自己的城堡添砖加瓦。家长也深知这重要性，仅几个月的时间，课课平台上已经有100万的家庭用户，而为了服务日均至少5万份的作业，课课上检查作业的师资团队达到2万人，其中有来自各大院校的年轻教师，也有毕业于全国前十院校的学霸。


初创企业课课： Azure 让课课无惧“早高峰”与“晚高峰”


“一天之中，早上五点至七点，下午四点至十点，都是作业上传的高峰期。课课现在把后台部署在 Azure 的 Web App 上，利用云的可伸缩性及高可用性，保证这期间的服务的质量与性能的稳定。”课课CEO班班说。

只需要使用智能手机，短短3秒，将作业拍照上传至课课 APP，这份作业就已分配至对应的批改的老师手中，他们也只需通过智能手机或Pad这样的设备，就能在5分钟内完成一份作业的检查与批复。当孩子有解答不出的题目时，甚至可以“召唤”老师，与老师进行实时语音探讨，打开解题思路。而在这全过程中，家长只需要帮助孩子进行作业上传，或支付解答的低额费用而已。

班班表示：“同时，在接下来的计划中，课课也将测试使用微软的认知服务，通过 OCR 的方式识别作业的类别、习题的内容等，进而完善功能，并提高整体效率。”


初创企业课课：创业助力，让我们更快成长！


谈起与微软BizSpark计划，即微软新创企业扶植计划的机缘，颇有一种误打误撞的感觉。班班回忆道：“当时，在北京参加微软的活动中，课课这个项目很荣幸拿到了名次，同时也获得了微软 3000美金Azure资源，技术专家及市场商务的支持。对于一个创业团队的初期，资金、资源和技术都是有限的，这样的支持无疑是一种极大的助力。其中技术专家让我尤为印象深刻，他们的工作效率、工作方式、技术规划等方面都让我们的团队备受启发。后续在创业的过程中，探讨项目、交流技术，彼此也建立起深厚的信任关系。”


课课应用的微软解决方案


课课的后台部署采用了 Azure 的 Web APP，基于云的伸缩性，在流量的高峰期极大满足了业务的可拓展性与高可用性。同时，后续也将采用微软认知服务，使用 OCR 对学生、家长上传的作业图片进行内容识别并建立题库，从而提升整体用户体验与运营效率。
