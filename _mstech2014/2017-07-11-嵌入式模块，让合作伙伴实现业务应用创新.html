本文作者：James Phillips，微软公司副总裁，业务应用、平台及智能


在今年的微软Inspire大会上，微软首席执行官和公司高层与超过17,000位合作伙伴、客户、商业领袖，共同分享了微软合作伙伴在支持各行各业客户转型业务方面，所发挥的重要作用。微软每年投资数十亿美元用于创新，就是为了给客户带来最好的产品，给合作伙伴带来一流的技术解决方案和服务，让他们以此为基础开发自己的业务。


今天，微软云合作伙伴生态系统拥有64,000多个合作伙伴，是世界上最大的云生态系统之一。合作伙伴们希望开发云应用、发展云业务，而我们为他们提供了最完善的云计算平台—— 从基础设施即服务（IaaS）、平台即服务（PaaS）、应用平台即服务（aPaaS）到业界领先的软件即服务（SaaS）。所有这些服务都能满足企业级应用的高需求，而且从一开始就是为了相互协同工作而设计的。


Microsoft Azure一直受到独立软件开发商（ISV）的青睐。事实上，我们有超过35,000个合作伙伴在利用Azure的IaaS和PaaS服务来开发自己的应用。而且，这些合作伙伴正越来越强烈地要求我们以嵌入式模块的形式提供更高级的云服务——他们尤其想要得到微软业务应用平台背后的服务。


Power BI是我们最早进行的尝试，从大约一年前开始，它就作为一个嵌入模块提供给合作伙伴了。在这个过程中，我们看到ISV充分利用Power BI Embedded为他们自己的应用增加了极其强大的业务分析能力。


现在，我们希望通过提供更多屡获殊荣的微软业务应用服务，让合作伙伴去创造更多的可能：

  • 用于销售、服务和运营的Dynamics 365 —— 被Forrester和Gartner等顶级行业分析机构誉为市场领导者。

  • PowerApps——6月，《PC》杂志授予PowerApp编辑选择奖，称其是“高级用户的梦想”，并把它排在Salesforce的快速应用开发产品之前。

 

ISV Cloud Embed计划

今天，我们推出了名为ISV Cloud Embed的新计划，让ISV能够更轻松地对现有业务应用进行现代化升级，或者是开发全新应用。今年晚些时候，我们还将加入更多高级服务作为嵌入式模块，其中包括：

  • Dynamics 365 Embedded —— 无需从头开发，就能为特定行业或区域快速定制业务应用

  • Microsoft Flow Embedded —— ISV将得以让高级用户和信息工作人员添加和自定义工作流，以便将其作为应用的一部分，自动处理业务流程

  • PowerApps Embedded——为应用提供了轻松扩展模型，能够在浏览器或移动应用中交付

  • Power BI Embedded——用户可以轻松嵌入完全交互式的报告和仪表板。成为ISVCloud Embed计划的一部分之后，ISV将能通过CSP交易该产品。

 

为ISV带来的收益

除了嵌入新功能，并在此基础上开发自己的应用，根据交易量的大小，合作伙伴还将在市场推广方面得到助力，以支持其业务的发展：

  • 合作伙伴可以聚焦于业务创新，而微软则负责提供平台，保持技术更新并持续添加新功能。

  • 最高可达50%的折扣让合作伙伴拥有更大的盈利空间。

  • Microsoft Azure合作伙伴可以更轻松地利用该计划—— 例如在Azure上托管其应用，然后再通过ISV Cloud Embed服务扩展其功能。

  • 通过Microsoft AppSource，有机会接触1亿Office 365活跃商业用户和不断增加的Dynamic365客户群—— Microsoft AppSource是企业用户寻找和部署合作伙伴商业SaaS应用的集散地。


就像所有微软合作伙伴计划一样，ISV Cloud Embed计划也会根据使用量和交易量的增加而提供更多的收益。对于所有层级，微软都会全力提供市场推广的支持。对于那些应用使用量特别显著的合作伙伴，微软还将动用我们全球最大的企业销售团队，为其提供联合销售支持。在Ron Huddleston今天发表的文章中，可以了解到更详细的情况。

 

立即加入

任何ISV只要能开发出符合要求的完整应用，就可以申请加入ISV Cloud Embed计划，根据开发商的具体情况，不同等级的应用将得到相应的收益。要进一步了解该计划，请访问ISV资源中心。只需在线填写一份表格，即可注册加入。


在Inspire大会上我们将分享关于该计划的更多信息，也期待未来能有机会展开合作。
